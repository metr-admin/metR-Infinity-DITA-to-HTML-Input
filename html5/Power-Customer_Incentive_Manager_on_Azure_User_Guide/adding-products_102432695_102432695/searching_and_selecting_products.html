Searching and Selecting Products

Searching and selecting option enables you to add products using the search and select option.

To add products using search and select option, please follow the below steps:

  1. Go to the Qualifications & Benefits page and click Add, as shown in the below image. Product Catalog page appears.
  2. Click Search Filters.
  3. Search Filters page appears.

    You can search for products by using the Basic and Advanced search filters.

    Search Filters

    Values

    Basic Search Filters

    In the Basic Search filter, you can search the products using the product Name, Product Code, and Product ID.

    Advanced Search Filters

    In the Advanced Search filter, you can search the products using the Field, Operator, and Value.

  4. Click Search.
  5. Select the products from the searched list, and then click Add.
  6. Select the required fields, as shown in the below image, for the added products, and then click Save. The selected products are added.
Note:

For each product, Customer Incentive Manager performs real-time validation for all the mandatory fields and product duplication, and displays a validation message if applicable.
