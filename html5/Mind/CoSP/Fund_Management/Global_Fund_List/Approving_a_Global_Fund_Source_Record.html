Approving a Global Fund Source Record

  • The role of the logged-in user should be Fund Manager.
After creating a global fund list, you can submit the Global Fund List record for approval.
  1. In the module menu, click Home.
    The Enterprise Dashboard is displayed.
  2. In the navigation pane, expand the Fund Management folder, and then click Global Fund List.
    The GLOBAL FUND LIST page is displayed.
  3. To approve the Global Fund List record, perform the following steps:
    Note:
    The following steps cover only the positive and forward workflow actions to reach the Approved status.
    Phase Role Workflow Steps
    1 Fund Manager

    1. In the list page, select the appropriate record that is in the Draft workflow status, and then click Select Actions.

    2. Click Submit, and in the Mind dialog box, click OK.

      The workflow status of the record is set to Submitted.
    2

    1. In the list page, select the appropriate record that is in the Submitted workflow status, and then click Select Actions.

    2. Click Approve, and in the Mind dialog box, click OK.

      The workflow status of the record is set to Approved.
