Global Fund Sources

The Global Fund List is a repository of all fund sources that are available to all projects in the enterprise. You can use the list of approved global fund sources for fund reporting in projects.

The fund source records are automatically created through data integration.

In a scenario where the transaction is done in EBS for the Fund Number that is not available in Mind, a new Global Fund should be created and approved by the user (Fund Manager).

The functional flow for global fund management (manual creation) is as follows:
  1. Global Fund List
    1. Create a global fund source
    2. Approving a Global Fund Source Record
    3. Perform workflow actions to implement the business process for fund sources
You can activate or deactivate a fund source at any point in time. By default, only the active records are displayed in the Global Fund List page.

  • To view and activate an inactive fund source, refer to Activating a Global Fund List Record.

  • To deactivate a fund source, refer to Deactivating a Global Fund List Record.
