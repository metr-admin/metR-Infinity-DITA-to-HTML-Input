Viewing the Revised Budget Estimate Record

The Budget Estimate Revision form tracks only the updates made to the budget estimate. The approved budget is revised to add, modify, or deleted.

The budget estimate revisions are received through integration with EBS.

  1. In the module menu, click Projects.
    The PROJECTS list page is displayed.
  2. In the list page, double-click the required project.
  3. In the navigation pane, expand the project folder, and expand Budget Management, and then click Budget Estimate Revision.
    The BUDGET ESTIMATE REVISION list page is displayed.
  4. In the list page, select the appropriate record, and then click View.
    The BUDGET ESTIMATE REVISION DETAILS page is displayed.
  5. Provide the required information in the fields, as described in the following table.
    Field Name Description
    Budget Estimate Rev ID An identification number for the budget estimate revision.
    Budget Estimate Rev Name The name of the revised budget estimate.
    Budget Estimate ID The code of the approved budget estimate for which the revision is created.
    Budget Estimate Name The name of the approved budget estimate for which the revision is created.
    PWA Project Number The identification number of the associated project.
    Project Name The name of the associated project.
    Budget Estimate Type The budget estimate type of the approved budget estimate for which the revision is created.
    Measurement System The measurement system used in the approved budget estimate.
    Standard Item List This field is blank.
    Description The description of the budget estimate revision.
    Estimate Total in $ A value is displayed based on the EBS added, deleted, or modified in the BUDGET ESTIMATE REVISION ITEMS tab.
  6. To view the details of EBS, click the BUDGET ESTIMATE REVISION ITEMS tab.
    The BUDGET ESTIMATE REVISION ITEMS page displays the following non-editable fields:
    Column Name Description
    Task Number The task number as received from EBS.
    Line Number The line number of the task as received from EBS.
    Task Name The name of the task as received from EBS.
    Unit The unit of measure assigned for the task.
    Quantity The quantity is defaulted to 1.
    Unit Price in $ The unit price for the task as received from EBS.
    Fund Rule This field is blank.
    Amount in $ It is auto-calculated as: Quantity * Unit Price.
