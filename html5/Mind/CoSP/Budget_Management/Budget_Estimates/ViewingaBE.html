Viewing the Budget Estimate

  • You must have View permission to view a record.

You can view the budget estimates records in the BUDGET ESTIMATES list page.

  1. In the module menu, click Projects.
    The PROJECTS list page is displayed.
    Alternatively, perform the following steps:
    1. In the module menu, click Planning.
      The PLANNING DASHBOARD is displayed.
    2. In the navigation pane, click Planned Projects.
      The PLANNED PROJECTS list page is displayed.
  2. In the list page, double-click the required project.
  3. In the navigation pane, expand the project folder, expand Budget Management, and then clickBudget Estimates.
    The BUDGET ESTIMATES list page is displayed.
  4. In the list page, select the appropriate budget estimate record, and then click View.
    The BUDGET ESTIMATE DETAILS page is displayed.
    The BUDGET ESTIMATE DETAILS page displays the following non-editable fields:
    The BUDGET ESTIMATE DETAILS displays the following non-editable fields:
    Field Name Description
    Budget Estimate ID An automatically generated alphanumeric code that uniquely identifies the budget estimate.
    PWA Project Number The project number for which the budget is created.
    Budget Estimate Name The name of the budget estimate as received from EBS.
    Project Name The name of the associated project for which the budget is created.
    Budget Estimate Type Displays Work Order.
    Created Date The date the budget estimate is created.
    Measurement System The measurement system for the estimate.
    Created By Displays Data Migration.
    Standard Item List This field will appear blank.
    Budget Template This field will appear blank.
    Description The description of the budget estimate.
    Estimate Total in $ Displays the estimate amount of the project.
    The BUDGET ESTIMATE ITEMS page displays the following non-editable fields:
    Column Name Description
    Task Number The number of the item received from EBS integration.
    Line Number The Mind assigned line number.
    Task Name The name of the item received from EBS integration.
    Unit The unit of measure for the project.
    Quantity The unit quantity of the defaulted as 1.
    Unit Price in $ The value is defaulted to 1.
    Fund Rule This field will appear blank.
    Amount in $ It is calculated as: Quantity * Unit Price.
