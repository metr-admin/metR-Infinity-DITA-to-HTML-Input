Creating a Portfolio

  • The role of the logged-in user should be one of the following:
    • Administrative Support
    • Construction Inspector
    • Construction Manager
    • Engineer of Record
    • Director
    • Field Supervisor
    • Fund Manager
    • Manager
    • Program Manager
    • Project Manager
    • Support Services
You can create a portfolio to group the projects, programs, and sub-portfolios. For ease of tracking, you can create portfolios that are in the purview of a user, and assign projects, programs, and sub-portfolios.
  1. In the module menu, clickHome.
    TheENTERPRISE DASHBOARDpage is displayed.
  2. In the navigation pane, clickPortfolio.
    ThePORTFOLIO list page is displayed.
  3. ClickNew.
    ThePORTFOLIO page displays the following non-editable fields:
    Field Name Description
    Created By Displays the name of the user creating the portfolio.
    Created Date The current date is displayed.
  4. In theDETAILS tab, provide the required information in the fields, as described in the following table.
    Field Name Description
    Portfolio Name Enter a unique name for the portfolio.
    Portfolio Manager To select the portfolio manager, perform the following steps:
    1. Click .

      The Portfolio Managers dialog box is displayed.

      The list comprises user names of all users in the enterprise.
    2. Select the required portfolio manager, and then click Select.
    Description Enter a detailed description of the portfolio.
    Start Date Select the start date of the portfolio.
    End Date Select the end date of the portfolio.
  5. To add projects to the portfolio, in the PROJECTS section, perform the following steps:
    1. ClickAdd.
      TheProjects dialog box is displayed.

      The list of all the projects in the enterprise is displayed.

    2. Select the required projects, and then clickSelect.
      The selected projects are added to the portfolio.
  6. To add programs to the portfolio, in the PROGRAMS section, perform the following steps:
    1. ClickAdd.
      ThePrograms dialog box is displayed.

      The list of all the programs in the enterprise is displayed.

    2. Select the required programs, and then clickSelect.
      The selected programs are added to the portfolio.
  7. To add portfolios as sub-portfolios, in the SUB PORTFOLIOS section, perform the following steps:
    1. ClickAdd.
      TheSub Portfoliosdialog box is displayed.

      The list of all the portfolios in the enterprise is displayed.

    2. Select the required portfolios, and then clickSelect.
      The selected portfolios are added as sub-portfolios to the portfolio.
  8. Click Save to save the record and return to the list page.
