Changing the Status of a User Account

You can Deactivate a User Account to prevent a user from accessing the application. On deactivating an account, the user cannot log into the application. However, the account is not deleted, and all details of the account are retained and available.

You can Active a User Account. On activating an account, the user can log in to resume working on the application.
