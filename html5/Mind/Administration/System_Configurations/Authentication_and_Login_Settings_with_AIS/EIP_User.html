External Identity Provider User

The External Identity Provider User page lists all the users registered through third party IDP such as ADFS.
  1. In the module menu, click Administration.
    The ADMINISTRATION page is displayed.
  2. In the navigation pane, expand User Management, and then click External Identity Provider User.
    The EXTERNAL IDENTITY PROVIDER USER list page is displayed.
  3. To view the details, select a record and clickView.
