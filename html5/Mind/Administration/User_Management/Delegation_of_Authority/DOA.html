Delegation of Authority

When a user is unavailable or unable to perform tasks in Mind, you can transfer all that user's tasks to other users. The users to whom the tasks are transferred to can then perform the associated tasks and workflow actions of the transferer.

Using theDELEGATION OF AUTHORITYform, you can delegate tasks of one user to another user with the same role in Mind for specific projects.
Note:
Only users with theAdministrator role can perform these activities.

You can perform the following tasks:

  • Delegate authorities
  • Viewdelegated records
  • Edit delegated records
  • Delete delegated records
  • Revoke a delegation
  • Delegating Authorities
  • Editing Delegation Details
  • Viewing Delegation Details
  • Revoking a Delegation
  • Deleting a Delegated Record
