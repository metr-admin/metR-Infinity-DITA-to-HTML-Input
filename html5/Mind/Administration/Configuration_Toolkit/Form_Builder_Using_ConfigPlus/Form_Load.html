Form Load

The Form Load event enables you to set values and attributes on a form after the form is generated. The logic is executed when you click New, Edit, or View the record, and the execution is completed when the form is generated.

Use Case# Set the value for a field based on user role

In the RFI form, if you want the company of the contractor to display automatically when a contractor is creating any record, then you can use the Form Load event to set the value for the appropriate field based on the user role.
