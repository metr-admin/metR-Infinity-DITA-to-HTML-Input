Actions Block

The blocks available in FORM BUSINESS LOGIC and LIST PAGE BUSINESS LOGIC are same except for the ACTIONS block. The ACTIONS block is the final output of the business logic that you must configure once you navigate to the list page. For more information on other blocks, refer to Blocks.

The following table provides information on the blocks available in the ACTIONS block:
Block Name Description
Set Menuitem Hidden Enables you to hide a toolbar option
Set Menuitem Disable Enables you to disable a toolbar option
