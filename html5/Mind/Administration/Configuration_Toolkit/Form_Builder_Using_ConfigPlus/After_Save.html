After Save

The After Save event is executed when you click Save or Save & Continue and the logic is executed after the record is saved.

Use Case# Update information from current form to another form

If you want to update the details from the Expenses form into associated PO form, then you can use the After Save event to define the attributes.

Use Case# Collect and consolidate the information from a different form, and update the information in the current form

In the Contract Change Order form, if you want to get the consolidated information from a field in the RFP form, and update a field in the CHANGE ORDER page, then you can use the After Save event to collect the appropriate information.

Use Case# Define auto-generated numbering logic for a record

To define auto-generated numbering logic that requires information to be retrieved from Mind database, you can use the After Save event. For example, in the BUDGET ESTIMATES list page, if the numbering logic is calculated as follows:

Project Code - Project Type - Auto sequence number, then you can use the After Save event to define the numbering logic.
