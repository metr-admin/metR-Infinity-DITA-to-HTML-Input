Events

Events act as activation points on which the logic is executed and are considered starting points. To create an event, you can click the appropriate event from the Select Event drop-down list in the left pane. Once the appropriate event is selected, it appears in the right pane design area where you can configure the logic.

The following events are available in the FORM BUSINESS LOGIC tab:
  • Modify Form
  • Form Load
  • Before Save
  • After Save
  • Workflow Action Complete
