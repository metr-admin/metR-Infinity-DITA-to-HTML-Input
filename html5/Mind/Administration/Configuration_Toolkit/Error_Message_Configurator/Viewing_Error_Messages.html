Viewing Error Messages

You can view error messages from the ERROR MESSAGE CONFIGURATOR list page.
  1. In the module menu, click Administration.
    The ADMINISTRATION page is displayed.
  2. In the navigation pane, expand the Configuration Toolkit folder, and then click Error Message Configurator.
    The ERROR MESSAGE CONFIGURATOR list page is displayed.
  3. To view the error message, in the ERROR MESSAGE CONFIGURATOR list page, select the appropriate error message.
  4. Click View.
    The ERROR MESSAGE CONFIGURATOR page is displayed with the following information:
    Field Name Description
    Key A unique identification code for each error message.
    Culture The geographic location of the error message.
    Form Id A unique identification code of the form name.
    FormName The name of the form from which the message is generated.
    FieldName The name of the fields from which the message is generated. The same message can generate in many fields, or the message can also generate from one or more fields.
    Description
    • Defines the error message meaning and when it is displayed.
    • Defines the solution to the error message for the problem statement and explains why or how this message is generated.
    Value The error message that is currently displayed in Mind.
    New Error Message The modified error message that replaces the message in the Value field.
  5. Click Cancel to return to the ERROR MESSAGE CONFIGURATOR list page.
