Error Message Configurator

In Mind, all the error messages that are displayed in the UI are compiled in one place so that you can view and modify them based on the requirements. The Error Message Configurator form enables you to view and modify error messages in the UI by changing the sentence structure without changing the code.

In the ERROR MESSAGE CONFIGURATOR list page, each error message displays the information to help you understand from which module these messages are generated and why these messages are generated.
