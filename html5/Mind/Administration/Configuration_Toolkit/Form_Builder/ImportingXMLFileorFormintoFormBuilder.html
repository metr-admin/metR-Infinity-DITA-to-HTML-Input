Importing XML File or Form into Form Builder

The Import XML feature allows you to either upload an XML file or import an existing XML-based form in Form Builder.
This feature allows you to:

  • Customize the existing XML-based forms in Form Builder

  • Import a customized XML-based form into Form Builder

  1. In the module menu, click Administration.
    The ADMINISTRATION page is displayed.
  2. In the navigation pane, expand the Configuration Toolkit folder, and then click Form Builder.
    The FORMS LIST page is displayed.
  3. Click Import XML.
    The IMPORT FORM page is displayed.
  4. To import from a file, perform the following steps:
    1. Select File Upload.
    2. In the Select File box, click Choose File.
      The Open dialog box is displayed.
    3. Select the required XML file to be uploaded, and then click Open.
    4. Click Import.
      A form is created and displayed in the FORMS LIST page.
    5. To modify and update the imported form, select the imported form, and click Edit.
  5. To import from an existing form, perform the following steps:
    1. Select Form Import.
    2. In the Select Form box, click and select the required form from the drop-down list.
      Available forms are the forms in Mind that are created using XML.
    3. Click Import.
      A form with the same Form ID and Form Header of the selected form is created and displayed in the FORMS LIST page.
    4. To modify and update the imported form, select the imported form, and click Edit.
