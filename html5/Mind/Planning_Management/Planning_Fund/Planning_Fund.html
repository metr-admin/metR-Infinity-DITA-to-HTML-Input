Creating a Planning Fund

  • The role of the logged-in user should be one of the following:
    • Fund Manager
    • Director
    • Manager
    • Program Manager
The Planning Fund is used to plan and estimate the fund requirements for a program. When a program is created, a planning fund source is associated. Planning funds are a subset of approved global fund sources.
  1. In the module menu, clickPlanning.
    ThePLANNING DASHBOARDpage is displayed.
  2. In the navigation pane,clickPlanning Fund.
    ThePLANNING FUND list page is displayed.
  3. ClickNew.
    TheDETAILS page is displayed.
  4. In theTitlefield, enter a name for the new planning fund source to be created.
  5. From the Default Fiscal Year drop-down list, select the fiscal year for the planning fund.
    Available options are the fiscal years defined in the Planning Management form of the Administration module. For information on fiscal years, refer to Configuring Planning Management.
  6. From theProgram Yeardrop-down list, select the year for which the planning fund is created.
    The selected year is the first year of the program duration as defined in theprogram cycle.
    For information on program cycle, refer to Configuring Planning Management.
  7. In theDescriptionfield, enter a brief description of the planning fund.
  8. To add fund sources,in the DISTRIBUTION DETAILSsection, perform the following steps:
    1. ClickAdd.
      TheGlobal Fund Listdialog box is displayed.

      Available options are global fund sources that are in theApproved workflow status.For information on global fund sources, referto Global Fund Sources.

    2. Select the appropriate fund sources.
    3. ClickSelect.
      The selected global fund sources are added to theDISTRIBUTION DETAILS section.
    4. To distribute funds for the program duration, enter the amount to be distributed corresponding to each fund for each year.
      Available years are based on theProgram Yearselected for the planning fund and the defined program cycle.For information on program cycle, refer toConfiguring Planning Management.
      Note:
      You can enter the distribution amount irrespective of the amount available in the selected fund sources.
  9. Click Save.
  10. To view and update the monthly forecast of the funds, perform the following steps:
    1. Select the appropriate planning fund, and then clickEdit.
    2. Click theFORECAST tab, and then click OK.
      TheITEM DETAILS page is displayed.
    3. Expand the planning fund source.
      The funds are distributed linearly on a monthly basis, starting from the first fiscal month of the program year until the end ofthe program duration.
    4. To view the forecasts of forthcoming months, clickNext Period.
    5. To update the forecast for a month, double-click the appropriate cell, and then enter the forecast amount.
    6. ClickSave.
      To perform workflow actions, refer to Planning Fund Workflow.
