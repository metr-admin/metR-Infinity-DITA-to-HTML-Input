Program

A program is a collection of proposed and authorized projects.

A program provides a comprehensive view of project requirements and available funds.

The Program form enables you to:
  • Add multiple planned projects to a program
  • Group projects under a program and assign funds
  • View funds that are available, remaining, andrequired
  • View details of each fund category and fund source
  • View the forecast of the program over the program period
  • Perform what-if analysis by creating multiple program scenarios and selecting the best scenario for approval
  • Performwhat-if analysisusing fund views
  • Create snapshots for comparison
  • Approve projects
  • Manage program revisions
  • Generate reports
You can perform the following tasks:
  • Creating a Program
  • Submitting and Approving a Program
  • Program Workflow
