Submitting and Approving a Program

  • Based on the workflow status of the record, you must be assigned one of the following roles:
    • Fund Manager
    • Program Manager

After creating a program, you can submit and approve a Program record.

  1. In the module menu, click Planning.
    The PLANNING DASHBOARD page is displayed.
  2. In the navigation pane, click Program.
    The PROGRAM list page is displayed.
  3. To submit and approve the Program record, perform the following steps:
    Note:
    The following steps cover only the positive and forward workflow actions to reach the Closed status.
    Phase Role Workflow Steps
    1
    • Fund Manager
    • Program Manager

    1. In the list page, select the appropriate record that is in the Draft workflow status, and then click Select Actions.

    2. Click Submit, and in the Mind dialog box, click OK.

      The workflow status of the record is set to Submitted.
    2 Fund Manager

    1. In the list page, select the appropriate record that is in the Submitted workflow status, and then click Select Actions.

    2. Click Approve, and in the Mind dialog box, click OK.

      The workflow status of the record is set to Approved.
    3

    1. In the list page, select the appropriate record that is in the Approved workflow status, and then click Select Actions.

    2. Click Close, and in the Mind dialog box, click OK.

      The workflow status of the record is set to Closed.
