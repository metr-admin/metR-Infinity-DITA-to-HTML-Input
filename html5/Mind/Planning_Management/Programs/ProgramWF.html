2 Submitted Fund Manager Approve Approved On approval of a program, the following changes occur:
  • The published budget estimates of the projects that are part of the program are approved. Also, thecurrent budget of each project is updated with the approved budget estimate items.
  • If what-if analysisis done, then the project start and end dates, forecast dates, and current budget datesare updated based on the forecast changes done in the program.
  • The workflow status of each project that is part of the program is set toPlannedif the start date of the project is not in the current yearand is set to Pre-Awardif the start date of the project is in the current year.
  • A project is moved from the Planned Projects list to the Projects module list page if the workflow status of the project is set to Pre-Award. The project remains in the Planned Projects list page if the workflow status of the project is set to Planned.
  • Forecast snapshot is generated automatically when the program is approved.
