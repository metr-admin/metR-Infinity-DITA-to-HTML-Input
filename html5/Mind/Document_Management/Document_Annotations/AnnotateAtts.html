Annotating Attachments

You can annotate files that are attachments to forms.
  1. In the navigation pane, click the appropriate form.
  2. Select the appropriate record, and then click Edit.
  3. To annotate an attachment, perform either of the following steps:

    • To annotate a document or image file, perform the following steps:

      1. In the ATTACHMENTS section, in the Document Name column, click the appropriate attachment.

        The file is opened in the viewer.

        Alternatively, select the appropriate attachment, and then click in the toolbar.

        The DOCUMENT ANNOTATION list page is displayed.

      2. In the list page, click New.

      3. If the annotations toolbar is unavailable, then in the document viewer, click Edit, and then click Enable Annotations.

        The various annotation options are displayed.

      4. Click the required annotation and add it to the document.

        You can annotate documents using the various annotation options that are available.

      5. Click Save.

        On saving the annotated document, a copy of the annotated document is saved with the timestamp of review and reviewer details.

        For information on accessing annotated copies of the attachment, refer to Accessing Annotated Attachment Documents.

    • To annotate a drawing file, perform the following steps:
      1. In the ATTACHMENTS section, select the appropriate drawing file.

      2. Click in the toolbar.

        Note:
        The button is not available if multiple drawing files are selected.

        The DOCUMENT ANNOTATION list page is displayed.

      3. Click New.

        The file is opened in the viewer.

      4. Click the required annotation and add it to the document.

        You can annotate documents using the various annotation options annotation options that are available.

      5. Click Save.

        On saving the annotated document, a copy of the annotated document is saved with the timestamp of review and reviewer details.

        For information on accessing annotated copies of the attachment, refer to Accessing Annotated Attachment Documents.
