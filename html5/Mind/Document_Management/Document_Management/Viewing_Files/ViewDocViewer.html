Viewing a Document in the Viewer

  1. In the navigation pane, expandDocuments, and then click the required document folder.
  2. In the documents list page, select the appropriate document, and then clickView.
    Alternatively, double-click theappropriate document in the list.
    The document is opened ontheVIEWERpage.
  3. If the document file type is mapped to a viewer that supports 2D and 3D drawings, then to switch the view between 2D and 3D, perform the following steps:
    1. In theVIEWERpage, clickon the toolbar.
      TheDocument Browserdialog box is displayed.
    2. In theTreetab, expandModeland then click the appropriate view.
      Note:
      If the document supports only 2D view, then the 3D view option is not available.You can view and examine a 2D or 3D drawing design in detail using the viewer tools.For information on using the 2D and 3D viewer tools, refer toUsing 2D and 3D Viewer Tools.
