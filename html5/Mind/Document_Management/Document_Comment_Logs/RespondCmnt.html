Responding to a Comment

You can respond to a comment posted by any user. The date the response is posted is also saved.
  1. To access a Documents folder at the project or contract level, perform the following steps:
    1. In the module menu, clickProjects.
      ThePROJECTS list page is displayed.
    2. Select the appropriate project, and then clickView.
      TheVIEW PROJECTpage is displayed.
    3. In the navigation pane, expand the appropriate project folder, expand the Documentsfolder,and then click the appropriate folder.
    4. Inthe navigation pane, expand the appropriate project folder, and then perform either of the following steps:
      • To access theDocuments folder at the project level, clickDocuments or expandDocuments, and then click the appropriate folder.

        The DOCUMENTS list page is displayed.

      • To access the Documents folder at the contract level, perform the following steps:
        1. Expand theContracts folder.
        2. Expand the appropriate contract folder, and then clickDocuments or expand Documents, and click the appropriate folder.

          The DOCUMENTS list page is displayed.

  2. Select a document, and then click Comment Log.
    The COMMENT LOG page is displayed.
  3. Select a comment, and then click Edit to respond to the comment.
    The Responded By column is updated with the user name of the logged-in user.

    The Responded On column displays the response received date.

  4. In the selected row, double-click in the Response column, and enter a response to the comment.
  5. Select the Complete check box if the comment has received an appropriate response.
  6. Click Save.
