Inviting Users to a Project

  • User accounts are created.
  • Roles are assigned to users.

    For information on creating user accounts and assigning roles to users, refer toUser Management.

You can invite users to a project and assign project specific roles as required.
  1. In the module menu, click Planning.
    The PLANNING DASHBOARD page is displayed.
  2. In the navigation pane, click Planned Projects.
    The PLANNED PROJECTS list page is displayed.
  3. In the list page, click the appropriate project,and then click View.
    The PROJECT DASHBOARD page is displayed.
  4. Click the PROJECT DETAILS tab.
  5. Click More, and then click Manage Users.
    The MANAGE USERS IN PROJECT page is displayed.
  6. Click Add User.
    The All Users dialog box is displayed.

    Available users are users belonging to the business unit as selected for the project. For information on Business Units, refer to Business Units. For information on associating users to business units, refer toUser Accounts.

  7. To add specific users, select the users to invite to the project, and then click Add Users.
    Alternatively, click Select All Recordsto add all users to the project.
  8. To assign project roles to invited users, perform the following steps:
    1. Double-click the cell in theProject Rolecolumn of the user.
      The list of roles assigned to the user at the enterprise level is displayed. For information on roles, refer to Security Roles.
    2. From the Project Roledrop-down list, select the roles to be assigned to the user in the project.
  9. Ensure the Is Primaryis selected for the user to receive task email notifications as set in workflows.
    Optionally, clear the check box to block the user from receiving email notifications as defined in workflows.
  10. Click Save.
