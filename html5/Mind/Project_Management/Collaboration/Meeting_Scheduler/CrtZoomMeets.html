Creating Zoom Meetings

The user should be authorized to create the meeting.
You can create Zoom meetings in Mind using the Meeting Scheduler feature.
  1. In the module menu, clickProjects.
    ThePROJECTSlist page is displayed.
  2. In thePROJECTSlist page, click the required project, and then clickEdit.
  3. In the navigation pane, expand the project folder.
  4. ExpandCollaboration, and then clickMeeting Scheduler.
    TheMEETING SCHEDULERlist page is displayed.
  5. ClickNew.
    TheMEETING SCHEDULERpage is displayed.
  6. Provide the required information in the fields, as described in the following table.
    Field NameDescription
    Title Enter the title of the meeting.
    Meeting Date Select the date of the meeting.
    Start Time Enter the start time of the meeting in accordance to the time zone of the application.
    End Time Enter the end time of the meeting in accordance to the time zone of the application. Ensure the end time is later than the start time.
    Meeting Type Select the meeting type.
    Available options are meeting types defined in theMeeting Typecatalog of the library.
    Note:
    When Minutes Of Meeting form exists, then both Minutes Of Meeting and Meeting scheduler forms share the same Meeting Type catalog of the library.
    Venue Select the venue of the meeting.
    Available options are venues defined in theMeeting Venuecatalog of the library.

    1. Alternatively, to specify the venue, perform the following steps.

    2. From theVenuedrop-down list, selectOthers.

      A text box is displayed.

    3. In the box, enter the venue details.
      Note:
      When Minutes Of Meeting form exists, then both Minutes Of Meeting and Meeting scheduler forms share the same Meeting Venue catalog of the library.
    Agenda Enter the agenda of the meeting.
    Notes Enter any notes for the meeting.
    TheMeeting Statusof the meeting record is set toDraftonly when the meeting created is saved but not published.
    Note:
    When the meeting is published, the Meeting Status changes to Published.
  7. To add meeting invitees in theMeeting Inviteessection, perform the following steps.
    1. To add invitees from the list of user accounts defined in enterprise.

      1. Click Add.

        TheSelect Usersdialog box displays the following details of all the application users and key contacts of all contracts within the project.

        • User Name

        • First Name

        • Last Name

        • Email

        • Department

        • User Type such as Project user, Non-project user, Contract, Key Contact

        • Contract name if the user is invited to a contract or if the user is a key contact of a contract

      2. Select the required users, and then clickSelect.

        The selected users are added as invitees to the meeting.

    2. To add external user.

      1. In theDepartmentcolumn, click and enter the department name of the invitee.

      2. In theEmailcolumn, click and enter the email address of the invitee.

      3. In theLast Namecolumn, click and enter the last name of the invitee.

      4. In theFirst Namecolumn, click and enter the first name of the invitee.

      5. ClickAdd External User.

        A row is added in theMeeting Inviteessection.

  8. ClickCreate Zoom Meetingto generate a Zoom meeting URL.
    The Zoom meeting URL is displayed in theMeeting Linkbox.
  9. To add references to any form within any record, in theLinked Objectsection, perform the following steps.
    1. ClickAdd.
      TheSelect Itemsdialog box is displayed.
    2. To select the required project form, in theSource Formbox, click and select the required form.
      Alternatively, to select a contract form, perform the following steps.

      1. In theSelect Project / Contractbox, corresponding to the project name, click .

      2. In theSelect Project / Contractbox, click and select the required contract.

        Alternatively, type the name of the contract, and then select the required contract.

        Available contracts are all the contracts the user is invited to in the project.

      3. To select the required contract form, in theSource Formbox, click and select the required form.

        Alternatively, type the name of the form, and then select the required form.

  10. Click Search.
    All the records of the selected form are displayed.
  11. Select the required records, and then clickSelect.
    The selected records are added to theLinked Objectssection.
    Note:
    To view the details of a linked record, in the Description column, click the required record.
  12. Optionally, in theAttachmentssection, attach or link related files.
    For information on attachments, refer to Attachments.
  13. Click Save & Exit to save the record and return to the list page. Optionally, click Save & Continue to save the record and continue on the same page.
