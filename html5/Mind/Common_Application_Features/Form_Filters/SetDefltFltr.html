Setting a Filter as the Default Filter

  • At least one saved filter exists. For information on saving a filter, refer to Saving a Filter.

In the list page of a form, you can set a filter as default. On opening the list page, the page displays only records that match the filter criteria.

The following procedure describes the steps todefine a defaultfilter.ThePROJECTSlist page is used as an example.

  1. In the module menu, click Projects.
    The list of projects you are invited to is displayed.
  2. In the tool bar, click .
    The FILTERS dialog box is displayed.
  3. In the Select Filter section, select the appropriate filter.
    Available options are filters defined for the page. For information on defining a filter, refer to Saving a Filter.
  4. Click to set the selected filter as default.
    On loading the page, the default filter is automatically applied.
