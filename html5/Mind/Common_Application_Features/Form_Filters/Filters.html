Filters

In a list page of a form in the application, you can search for specific records that match the criteria you provide. Records that match the criteria defined are displayed in the list page. You can also define and save search criteria as pre-defined filters.

If you apply multiplefilters successively, then the resultsare based on the collective search criteria. That is, if you apply a filter, then follow it up with another filter on a different column, then the second filter is applied on the results of the first filter.

You can search for records using either of the following ways:
  • Enter search criteria
  • Use a pre-defined filter
