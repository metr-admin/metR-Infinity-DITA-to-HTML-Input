Deleting Attached Files

  • You can delete a file only if you have attached it.
You can delete files that are attached to forms.
  1. To access the appropriate file to be deleted, perform the following steps:
    1. In the navigation pane, click the appropriate form.

      The form list page is displayed.

    2. In the tool bar, click More, and then clickAttachments.

      The attachments of all the forms are listed.

      Various document management features are available for attachments. For information on document management features, refer to Document Management.
