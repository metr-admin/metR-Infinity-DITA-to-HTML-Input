Standard Report Functions

Performing all report related activities is similar in procedure throughout the application.

All tasks that you can perform are based on the roles assigned to you in a project and the permissions granted to the roles. For information on roles, refer to Security Roles.

The standard report functions include the following:
  • Generating a report
  • Printing a report
  • Saving a report in various formats
  • Updating report to view latest information
  • Subscribing to a report
  • Unsubscribing to a Report
