Scoring Criteria

  • Categoriesare defined in the Category catalog in the library. For information, refer toCreating a Scoring Category.
  1. In the module menu, click Library.
    The LIBRARY MANAGEMENT page is displayed.
  2. In the navigation pane, expand Planning,and then click ScoringCriteria.
    The SCORING CRITERIA list page is displayed.
  3. Click New.
    The new SCORING CRITERIA page is displayed.
  4. In theName box, enter the name of the scoring criteria.
  5. In theDescription box, enter the description for the scoring criteria.
  6. From the Category drop-down list, select the category to associate the criteria with.
    Available options are categories defined in the Category catalog in the library and are marked active.For information, refer to Creating a Scoring Category.
  7. In the Max Value box, enter the required value to define the maximum score for the criteria.
  8. Optionally, to mark the scoring criteria as inactive, clear the Is Active check box.
    Inactive criteria are not available for selection during project scoring.
  9. Click Save & Exit to save the record and return to the list page. Optionally, click Save & Continue to save the record and continue on the same page.
