Deleting a User

You can delete a user account to prevent a user from accessing the application. On deleting an account, the user cannot log into the application.
  1. In the Home page, click Settings, and then select Users.
    The Users page is displayed.
  2. In the Users page, select the appropriate user, and click .
    A confirmation message is displayed.
  3. Click Yes.
    The user is successfully deleted from the system.
