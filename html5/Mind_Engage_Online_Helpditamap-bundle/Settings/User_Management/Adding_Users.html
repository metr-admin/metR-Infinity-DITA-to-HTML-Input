Adding Users

You can create an account for a user to log into and use the Mind Engage application.
  1. In the Home page, click Settings, and then select Users.
    The Users page is displayed.
  2. Click Add New User.
    The Add New User page is displayed.
  3. Provide the appropriate information in the fields, as described in the following table.
    Field Name Description
    FIRST NAME Enter the first name of the user.
    LAST NAME Enter the last name of the user.
    EMAIL Enter the email address of the user.
    PRIVILEGE Enter the security permission role of the user.
    DEPARTMENT Enter the department of the user.
    DESIGNATION Enter the designation of the user in the organization.
  4. Click Create.
    A new user is created.
    An email is sent inviting the new user to Mind Engage.
