Adding Layouts and Fonts

You can add or change the color of the public portal layout and font.
  1. In the Home page, click Settings, and then select Public Portal Template.
    The Public Portal Settings page is displayed.
  2. In the CHOOSE A LAYOUT COLOR: section, select the theme color for the portal.
  3. In the CHOOSE A FONT COLOR: section, select the font color for the portal.
  4. Click Preview Portal to preview the portal.
  5. Click Save Template.
