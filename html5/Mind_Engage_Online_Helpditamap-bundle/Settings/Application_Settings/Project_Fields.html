Project Fields

The Project Fields section is optional. This section enables you to add fields to display additional project details in the public portal.
  1. In the Home page, click Settings, and then select Application Settings.
    The Application Settings page is displayed.
  2. In the Project Fields section, click Add Field.
    The Add Project Field dialog box is displayed.
  3. In the FORM FIELD NAME field, enter the field name.
  4. Click Add.
    Note:
    The new form field is added to the Project Fields section.
