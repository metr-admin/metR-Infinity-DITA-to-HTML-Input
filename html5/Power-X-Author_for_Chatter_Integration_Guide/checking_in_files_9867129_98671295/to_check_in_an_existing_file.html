To check-in an existing file

There must already be a version of the file in Salesforce.

  1. Click Check-in from the X-Author Chatter ribbon.
  2. Enter comments in the What changed field and click OK.

    The filename cannot be changed, ensuring you are checking in a new revision of the existing file.

  3. Click OK, in the confirmation dialog, which indicates the version of the file

The new version of the file is in Salesforce.
