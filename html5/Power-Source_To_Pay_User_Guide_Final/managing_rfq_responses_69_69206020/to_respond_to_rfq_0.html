To respond to RFQ

  1. Click theSourcing Events tab.
  2. Select the appropriate sourcing event and click View.
  3. Click the RFQ tab.
  4. Click View RFQ.
  5. To download the RFQ, click Download.
  6. To respond to the RFQ, click Create Response.
  7. Enter the appropriate responses.
  8. Select the Group Name check box and click Refresh to view the Total.
  9. Select the price modifiers that are applicable. You can add comments in the Comment text box.
  10. To upload an attachment, click Upload.
    Note:

    For the Attachments in Apttus STP, you can upload any file. We allow files of the following extension types: (.PDF), (.RTF), (.RTX), (.TXT), (.JPG), (.JPEG), (.PNG), (.GIF), (.TIF), (.BMP), (.DOC), (.DOCX), (.DOCM), (.PPT), (.PPTX), (.PPTM), (.PPS), (.PPSX), (.XLS), (.XLSX), (.XLSM), (.ZIP), (.MP3), (.M4A), (.OGG), (.WAV), (.MP4), (.M4V), (.MOV), (.OGV), (.3GP), (.3G2). The application might become slow if you upload a file higher than 25 MB.

  11. Click Choose File and select the appropriate file.
  12. Enter the File Description.
  13. Select the appropriate permission from the Permissions picklist menu.

    Permission

    Description

    Public

    To make the attachment available to all the Suppliers.

    Private

    To make the attachment available to only the Buyer.

    Selective Public

    To make the attachment available to the selected Supplier(s).

    Note: At least one supplier must be selected for this option.
  14. Click Attach.
  15. Click Save.
  16. To edit an attachment, click Edit.
  17. To delete an attachment, click Delete.
  18. Click Submit Response.
  19. Click the Back to Sourcing Event link .
    Note:

    Once you submit your response to the buyer, View RFQ and View Response links are visible on the Sourcing Event Detail page. You can edit the responses before the RFQ deadline. Ensure that you click Submit Response after you have edited the response.
