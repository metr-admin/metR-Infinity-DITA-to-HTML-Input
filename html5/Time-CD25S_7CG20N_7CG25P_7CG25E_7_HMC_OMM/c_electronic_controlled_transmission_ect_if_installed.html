Electronic Controlled Transmission (ECT; if installed)

Electronic Controlled Transmission (ECT) is an electronic control system designed for the forklift trucks driven by diesel engines.

The main functions of the ECT are to improve the pressure of heavy inching pedal and reduce shock at forward/reverse gear shift. In addition, when Auto Shift Controller (ASC) is selected, the dame function is included.

ECT is suitable for the operations that use inching frequently by protecting the transmission from deformation and overheat.

The features of the ECT system include followings:

  • Reduce pedal pressure of inching pedal
  • Reduce shock at forward/reverse gear shift
  • Auto Shift Controller function (if provided)
  • Detect failure of transmission valve and system
  • CAN communication with the engine control system
