How does my MIL work?

The emissions control system utilizes a MIL to warn the operator or technician of a possible issue with the engine or emissions control system. The system will keep the MIL illuminated for the entire key cycle in which the trouble code was set. It will keep the MIL illuminated for three additional engine run cycles under the following two circumstances: (1) The fault caused the engine to shut down or (2) the fault is related to the exhaust gas oxygen (EGO) sensors. This function is called MIL persistence.
