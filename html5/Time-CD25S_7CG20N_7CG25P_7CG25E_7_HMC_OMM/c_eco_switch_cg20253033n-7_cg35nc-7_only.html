ECO Switch (CG20/25/30/33N-7, CG35NC-7 Only)

1. STD MODE – If Multi Torque switch is located in the Off position, Max. RPM performance will be shown.

2. ECO MODE – If Multi Torque switch is located in the Eco position, Max. RPM and Torque will be reduced.

Warning:

TURN IGNITION KEY SWITCH OFF BEFORE CHANGING AS STD MODE OR ECO MODE!
