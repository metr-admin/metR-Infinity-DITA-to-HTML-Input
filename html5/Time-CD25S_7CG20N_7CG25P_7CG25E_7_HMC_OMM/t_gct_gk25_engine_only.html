GCT GK25 Engine Only

  1. Start and warm up the engine using LPG.
  2. Close tank cock, and then consume LPG in fuel line.
  3. Remove drain plug, and then drain tar from LP regulator (vaporizer).
  4. Put the tray below so as not to make engine room dirty.
  5. Apply sealant to drain plug and assemble after finishing work.
    Warning:

    Fuel system supplier (GCT) recommends that Tar Drain would be progressed every 200hour.
