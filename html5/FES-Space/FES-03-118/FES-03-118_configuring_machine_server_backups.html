Configuring Machine Server Backups

Purpose

  1. This document describes how to configure a Machine Server to back up critical files. Ideally, one backup machine server will be operational on the network for every furnace that has machines requiring a machine server. This procedure assumes that SWSet 223 (SCS-0237-GR223) or higher is being used.
  2. This document supersedes instructions provided in the IDS Electronic Service , andamp; Reference Manual, Appendix A.

Application

This work instruction applies to all Space Plants that have FSC and Space IS Machines.

Safety

Follow all plant safety requirements.

Procedure

  1. On the Machine Server, log off the obop account (using the sequence Start-, andgt;Shutdown-, andgt;Log Off).
  2. Log onto the Machine Server using the obsetup account (password: SetupServer).
  3. Start the Backup Configuration Utility (Start -, andgt; Machine Server Applications -, andgt; Backup Configuration Utility), as shown in Figure 1.
  4. Use the drop-down box labeled Backup Server to confirm that the desired server has been added to the list of backup servers (see Figure 2). If it has, skip to step 6.
  5. Add the backup server. (Note, if desired, the Tab key can be used to toggle through the necessary fields)
    1. Enter a Server Name in the box provided.
    2. Enter the four IP Address Segments in the next box.
    3. Click the Add Server button to add the server.
  6. Select the desired backup server from the Backup Server drop-down.
  7. Click the Apply button to apply the changes.
  8. Click the Close button to close the backup configuration utility.
  9. Log off the obsetup account.
  10. Follow the directions in SOP-16-010 (Manual MDMS Backup) to manually trigger a server backup.
