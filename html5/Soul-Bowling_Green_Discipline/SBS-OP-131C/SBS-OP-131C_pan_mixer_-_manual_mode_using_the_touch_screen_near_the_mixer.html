Pan Mixer - Manual Mode Using the Touch Screen Near the Mixer

SBS-OP-131C

Revised: 2024-01-23

Purpose/Application

Defines the procedures for using manual mode for the pan mixer.

PPE Required

General

Bump cap/hard hat, safety glasses with foam liner, proper hearing protection, fire-resistant long sleeved shirt, safety toe boots.

Additional PPE When Needed

Respirator or PAPR in case of a toxic batch material spill.

Safety Hazards

Respirable Crystalline Silica

Crystalline silica has been classified as a human lung carcinogen. Breathing crystalline silica dust can cause silicosis, which in severe cases can be disabling, or even fatal.

Safety Controls

  • Exposure to heat can cause illness and death. The most serious heat illness is heat stroke. Other heat illnesses include heat exhaustion, heat cramps, and heat rash. Take precautions to avoid heat illness any time temperatures are high, and the job involves physical work.
  • The Hazard Communication Standard (HCS) requires chemicals to have Safety Data Sheets (SDS). Use the SDS for chemicals listed in this SOP for additional hazards, first aid measures, toxicological information, accidental release measures, and handling/storage of the chemical.
  • If a task causes exposure to hot molten glass, wear the appropriate shaded eyewear to decrease exposure to infrared radiation.
  • Review the Emergency Action Plan before performing work.
  • Follow donning and doffing procedure.
  • Some tasks in this document require additional PPE and tools/equipment to perform tasks, and may have specific safety hazards.

Equipment Needed

Access to the mixer control panel

Comments

Before starting work:

  • Read this document thoroughly.

  • Wear the required PPE.

Manual Mode Using the Touch Screen Near the Mixer

  1. To manually operate the mixer, use the touch screen on the front door of the mixer cabinet.
  2. On the Overview screen, log in and press the Manual Panel button.
    • When logged in, the operator can place the mixer into Manual mode and operate each piece of equipment individually.
    • After completing work, tap "remote" to tell the mixer to receive commands from the SCADA instead of manually from the HMI.
    Note:

    The Manual mode of operation is NOT interlocked! The operator can switch each drive on and off independently from another device. The operator is responsible for maintaining the correct order so that no equipment is damaged.

  3. Operate the listed devices by pressing the appropriate buttons.
    • Press the Start and Stop buttons, or Open and Close buttons, to operate a device.
    • Some devices have a tipping (jog) operation. Press and hold the Tipp button to turn the device on. Release the button to stop the device.
  4. Press the System button on the bottom of the screen to display the System screen.
    • DB Lock: This button only has functionality when the panel is connected to the PLC.
    • User Administration: Opens a screen that displays all active users, where users can change passwords, and new users are added.
    • Transfer: For Zippe-Service only, to transfer program changes to the panel.
    • Show Version: The current version of the touch screen is displayed.
    • System Diagnostic: Use this screen to browse all connected Profinet devices to check their actual working status.
    • Display Calibration: Used to improve touchscreen accuracy. Follow the instructions on the screen.
    • Display Cleaning: Locks the touchscreen for a few seconds to physically clean the display.
    • Exit Runtime: Click this button to shut down the HMI-Runtime. This is typically done to close the control screen, and access other parts of the touch panel operating system.
    • Control Panel: Opens the control panel of the underlying Windows System.
