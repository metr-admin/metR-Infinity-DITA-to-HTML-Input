Operating the AGV

SBS-OP-121

Revised: 2024-02-13

Purpose/Application

Describes the procedures for using the operating screens for the AGV.

PPE Required

General

Bump cap/hard hat, safety glasses with foam liner, proper hearing protection, fire-resistant long-sleeved shirt, safety toe boots.

Additional Equipment

Respirator or PAPR in case of a toxic batch material spill.

Safety Hazards

Respirable Crystalline Silica

Crystalline silica has been classified as a human lung carcinogen. Breathing crystalline silica dust can cause silicosis, which in severe cases can be disabling, or even fatal.

Safety Controls

  • Do not ride on the AGV or step onto it.
  • LO/TO the power before performing work. Follow plant procedures.
  • The Hazard Communication Standard (HCS) requires chemicals to have Safety Data Sheets (SDS). Use the SDS for chemicals listed in this SOP for additional hazards, first aid measures, toxicological information, accidental release measures, and handling/storage of the chemical.
  • Review the Emergency Action Plan before performing work.

Equipment Needed

  • Access to the SCADA SBS operating system.
  • Fire extinguisher in the cage.

Comments

Before starting work:

  • Read this document thoroughly.
  • Complete the required training.
  • Wear the required PPE.
Note:

This SOP uses Streator screens and images.

Safety Around the AGV

  1. Note:

    Electric Shock Hazard

    Hot Surfaces

    Battery and Materials Inside Fence Can Be Toxic

    Avoid injury during startup.


  2. Only authorized and trained personnel can enter the fenced area and work on the AGV.
  3. Personnel must wear the appropriate PPE around the AGV.
  4. The AGV must be stopped before personnel can enter the fenced area where the AGV operates.
  5. Observe all plant and equipment safety requirements.
  6. Turn the AGV off if a safety or operational issue occurs. Open the door to the fenced area to break the circuit.
    Note:

    The AGV has an E-Stop button on the left side, with an indicator light.

  7. Turn off power to the AGV before working on it. Then lockout-tagout-verify the power.
  8. When working on high-voltage assemblies:
    • Ground the supply cable after disconnecting the voltage.
    • Short-circuit the components, e.g., capacitors by means of a ground rod.
  9. Only install original or Hegla-approved spare parts.

Navigating the Operating Screens

  1. Note:

    The AGV has two travel modes


    • Manual mode is mainly used for setup and troubleshooting. In manual operation, the AGV is controlled via a wired remote control. The trained operator is responsible for the travel. Travel can take place with or without load.
    • In automatic operation, the AGV receives its travel orders from a master computer (travel to position X, fetch load Y, and bring it to position Z), which are processed along the specified route. In automatic mode, the AGV can travel both with and without a load.
    • There is also a semi-automatic mode that allows the operator to specify the AGV target position.
  2. The control screen and the main power switch are located on the AGV.
    Note:

    Use this power switch to turn the AGV on or off.

    • This switch must always be in the "O" position during maintenance work.
    • Follow plant lockout-tagout procedures.
    Note:

    If any E-Stop in the SBS system is pressed, the AGV and the rest of the system will stop.

    Note:

    The operator can use the SBS screens to stop operation, in which case the AGV completes its current order before stopping.

  3. The main screen is shown on the right. To display the main screen when another screen is showing, click the Homebutton in the lower left corner of the screen.

    The following information is displayed:

    • Current load
    • Current job
    • Current origin and target position
    • Status of battery, drives, temperature, network, WLAN
    • OnTrack, MasterLinkUp (connection to Kollmorgen PC)
  4. The top section of the main screen displays:
    • Name of the AGV (on the left)
    • Battery charge state, shown with a battery and percent charge. A lightning symbol appears if the AGV is currently charging.
    • Stop symbol, which appears only when the vehicle has been stopped by the controller. (The reason for the stop is listed on the detail screen.)
    • HEGLA logo
  5. Use the buttons on the bottom of the screen to navigate between the various operating screens. From left to right, the buttons are:
    • Back to Main Screen/Menu
    • Manual Operation
    • Carriage Details
    • Battery
    • Information Details
    • Maintenance
    • User
  6. Status Indicators are displayed using four colors in the top right box, behind the images:
    • Green = All OK
    • Yellow = Warning
    • Red = Error alarm
    • Purple = Serious error (prevents movement of the AGV)
  7. Messages are also provided in four colors to indicate the type of message. They are displayed on the alarm screen (3rd screen from left of the user log in button).
    • Black = Message
    • Yellow = Warning
    • Red = Error alarm
    • Purple = Serious error that prevents movement of the AGV
    Note:

    The last active fault messages are displayed with "Alarm‚Äù on the Information Details screen on the AGV, and usually on the top of the screen.

Description of the Operating Screens

  1. Click the Manual button on the bottom of the screen to display the Manual Operation menu.
    • The Safety Override bar turns green when the controller is in use.
    Note:

    AP means anti-parallel mode. Diff means differential (free movement).

  2. Click the Carriage button on the bottom of the screen to display the Carriage menu, which shows details about the carriage itself, but not the battery.
    • The System Status on the upper left shows the status of the CVC controller, SICK controller, CAN bus, and WLAN.
    • There are also details about the AGV position and whether the AGV is stopped.
    • In the upper center of the screen, the driver status for travel and lift are displayed.
  3. Click the Battery button on the bottom of the screen to display the main Battery menu, which shows the charge level and the status of the battery.
    • Click the upper right arrow on the right side of the screen to display the Battery Status menu, shown to the right.
    • Click the lower right arrow to display the Charger Status General menu.
    • Click the left arrow to return to the main Battery menu.
  4. There are three Charger Status screens. Navigate between these screens using the gray arrow next to the buttons.
    • The General screen displays general charging information.
    • The Error/Warnings screen displays the status of any warnings.
    • The Electronics screen provides additional data.
    • Click the left arrow to return to the main Battery menu.
  5. There are five Information Details screens. Navigate between these screens using the gray arrows next to the buttons.
    • The first three screens provide various operational data. See the three screens on the right.
  6. The fourth Details screen provides information about the four laser scanners/sensors.
  7. The fifth Details screen provides E-Stop information.
  8. Click the Maintenance button on the bottom of the screen to display the Main Maintenance screen.
    • Navigate between the two screens using the gray arrows next to the buttons.
    • The Drives Control screen provides the following information:

      1 = Sets the lift position to zero

      2 = Lifts and lowers the lift without limits

      3 = Sets the steer angle to zero

      4 = Graphical and numerical lift position

  9. An oil change must be performed on the lifting spindle drives after 500 operating hours.
    • If this value is exceeded, an oil symbol will appear in the top section of every menu/screen.
  10. Pop-up windows are displayed as an additional safety query to allow the lift, steering angles, or operating hours counter of the lift to be set to 0.
  11. Click the User button on the bottom of the screen to display the Main User screen, where users and administrators can log in and out, and set the language and units.
    Note:

    Parameters can only be changed with the highest user level. Settings in the system area of the control panel are reserved for HEGLA service technicians or specially trained and instructed specialist personnel.
