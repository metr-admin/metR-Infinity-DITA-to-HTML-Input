Navigating the SCADA Operating Screens

SBS-OP-102

Revised: 2024-02-13

Purpose/Application

Provides background information for navigating the SCADA SBS operating system, and using the operator screens.

PPE Required

General

Bump cap/hard hat, safety glasses with foam liner, proper hearing protection, fire-resistant long-sleeved shirt, safety toe boots.

Safety Hazards

Respirable Crystalline Silica

Crystalline silica has been classified as a human lung carcinogen. Breathing crystalline silica dust can cause silicosis, which in severe cases can be disabling, or even fatal.

Safety Controls

  • The Hazard Communication Standard (HCS) requires chemicals to have Safety Data Sheets (SDS). Use the SDS for chemicals listed in this SOP for additional hazards, first aid measures, toxicological information, accidental release measures, and handling/storage of the chemical.
  • Review the Emergency Action Plan before performing work.

Equipment Needed

  • Access to the SCADA SBS workstation

Comments

Before starting work:

  • Read this document thoroughly.
  • Complete the required training.
  • Wear the required PPE.

Basic Layout of Each Screen

The Zippe operating system has two SCADA workstations (based on Windows 10). One station is in the SBS control room, and the other is in the melter control room. The screens available through the workstations are used to control and monitor operations, and respond to alarms.

As shown below on the main menu, each screen is divided into three sections: Overview Area at the top, Working Area in the middle, and Button Bar at the bottom.

Overview Area

The top section of the screen provides the operator with an overview of the entire batch plant that for monitoring and detection of problems. The image below explains the purpose of each of these buttons.

Note:

The latest incoming message appears at the top of the screen. Recent messages can be viewed using the buttons in the upper left corner, which are identified on the following page. The messages and alarms are explained in detail in SBS-OP-003 Alarm Message System.

Four key operations have tabs in the Overview area at the top of the screen:

  • Silo Feeding
  • Batch System
  • Batch Cullet Weighing
  • Maintenance

To the right of each operation tab is a 5-segment group display that shows current messages for that operation. (Only three segments are currently used.) The messages include:

  • A=Alarm violation of the alarm limit (white on red)
  • W=Warning violation of the warning limit (black on yellow)
  • S = Process control message
  • F = fault, error

Button Bar

The bottom of the screen provides two sets of buttons for navigation and process control, and are the same for all four operation areas. Click the arrow button on the far left to toggle between the two button sets. Each button is identified with a letter and explained in the table below.

Working Area

When an Operation tab is clicked, the tab is highlighted and the working area of the screen displays the overview/main screen for that operation (Silo Feeding, Batch System, Batch Cullet Weighing, Maintenance).

Click the “down” arrow to the right of that operation tab to display a popup window used to navigate to the other screens for that operation area. The example below shows the main screen and popup window for the Silo Loading operation. (The main screens show smaller versions of several of their operation screens.)

The screens for each operation area are shown below.

Faceplates

If the cursor hovers over a piece of equipment on a screen and a “lightning bolt” appears, click on the item to open a faceplate (popup window] that provides information about that specific device. The faceplates vary depending on the type of equipment. For example, the faceplates for a screw drive or dosing feeder are shown below, and include the ability to change operating modes, change parameter values, and view alarms for that particular device.

Silo Feeding Screens

  • Click the Silo Feeding tab at the top of the screen to display the main Silo Feeding screen, shown below.
  • Click the down arrow button to display the popup window to navigate to the other two operation screens—for the majors silos 1-8, and for the minors silos 12-18. These screens are shown on the following page.

Batch System Screens

Click the Batch System tab at the top of the screen to display the main Batch System screen, shown below.

Click the down arrow button to display the popup window to navigate to the following operation screens, which are shown on the following pages:

  • Scales – monitors the two batch container scales, and the two minors scales.
  • Status – provides details about the operation of the scales, pan mixer, and other equipment.
  • Status AGV – monitors the AGV status and position. This screen is discussed in detail in document SBS-OP-023 Using the SCADA AGV Screen.
  • Preselection – selects the formula and scale to use for making the next batch.
  • Formula – Used to enter data for new batch formulas, and to modify existing batch formulas.
  • Timer Plant – sets the timing values for silo dosing, mixer operation, and pressure settings.
  • Names – used to assign names and numbers to each batch.
  • Silo Balance – Not Applicable
  • Consumption – shows the amount of material used from each silo over time.
  • Batch Protocol – provides details on each material in a batch including the weight added and its target value, and the start and stop dosing times.

Batch Cullet Weighing Screens

Click the Batch Cullet Weighing tab at the top of the screen to display the main Batch Cullet Weighing screen, shown below.

Click the down arrow button to display the popup window to navigate to the following operation screens, which are shown on the following pages:

  • Status – provides an overview regarding the current weighing cycle of the scales
  • Formula - clicking the button opens a Formula screen where the cullet ratio is entered
  • Consumption – the production data for today and the previous 14 days are displayed
  • Timer - used to adjust scale and plant timers for the Integration system equipment
  • Preselection – used to select the active formula
Note:

The Integration system screen also allows these operations, as well as others. For details, refer to SBS-OP-006 Operating the Integration System.

Maintenance Screens

Click the Maintenance tab at the top of the screen to display the main Maintenance screen, shown below. (You can also click the Plant Network button on the bottom of the screen.)

Click the down arrow button to display the popup window to navigate to the following operation screens, which are shown on the following pages:

  • Network – shows how the individual network components are connected, and whether they communicate with each other.
  • Maintenance – provides an overview of the connected PLC systems and their underlaying Profinet devices.
  • Scale Adjustment – displays the status of the minors scales, and is used to calibrate them.
