Login to the Web Interface

  1. Start a web browser from a PC on the same network as the gateway.

  2. Enter the IP address that is configured for the gateway in the address field.

  3. Enter the default login information:
    • User name: Administrator
    • Password: AB9001 AB9006
    Important: The user name and password must be changed after the first login.

  4. To change the login information navigate to Configuration > Authentication in the side menu.
