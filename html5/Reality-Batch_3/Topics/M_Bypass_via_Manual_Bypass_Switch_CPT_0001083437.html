Bypass via Manual Bypass Switch (Bypass Output)

CAUTION
REDUNDANCY LOST
If the manual bypass is in position 1 it is not possible for the system to automatically switch to the inverter or static switch if the supply from the bypass is lost.

The load is directly supplied from the bypass. The manual bypass switch is bypassing the static switch. The system will stay in bypass output mode until the position of the manual bypass switch manually changes to another. No automatic transfer is possible from this mode.

The manual bypass switch should only be used for service and only for a limited period of time.
