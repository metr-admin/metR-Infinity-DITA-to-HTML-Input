Optional Analog Instruments

As an option it is possible to select analog instruments for measuring certain values. There are available instruments for both AC and DC.

Note: Please refer to the Single Line Diagram of the system for the reference designator and exact position of the instrument.

Phase selector switch

For each position fitted with ammeters and/or voltmeters a phase selector switch can be used to select and measure the individual phases L1, L2 or L3
