How to Change to Forced Bypass Operation

Warning
SYSTEM OPERATION LOCKED
In forced bypass operation there is no automatic switch over to inverter output if the bypass mains becomes unavailable.
Important: This mode is a temporary mode and should not be used for an extended period of time.
  1. Press the Switch operation icon in the Status bar.
  2. Press the Force bypass operation button. Confirm the change.
    Note: The current active inverter operation mode is indicated by the green icon.
  3. Press the Home icon and check that the power flow indicates static switch operation and that the Unit state is Bypass operation.
    Note: The LED alarm for LEDs/N_P-LED-8, LEDs/N_P-LED-12 and LEDs/N_P-LED-17 will activate.
The system have switched to forced bypass operation and the load is supplied from the bypass. The inverter is turned off. The system is locked in this operation mode until it is manually changed.
