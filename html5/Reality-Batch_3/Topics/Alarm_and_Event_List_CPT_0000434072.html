Alarm and Event List

All alarms and events are always stored in the event log. Depending on the alarm it can also be stored in the alarm stack and affect the common alarm. There are three different types of alarms:
  • Event only: Saved only in the event log. Does not affect the common alarm.
  • Alarm only: Saved in the event log and the alarm stack. Does not affect the common alarm.
  • Alarm and common alarm: Saved in the event log and the alarm stack. Affects the common alarm.
Note: The alarm text is only displayed in English even if another system language is used.
