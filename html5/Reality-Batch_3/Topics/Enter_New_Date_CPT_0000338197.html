Enter New Date

To enter a new date:

  1. Press the Hash key to enter the User stack.

  2. Use the Arrow keys to navigate to Enter new date.

  3. Use the Numpad (Number 0-9 keys) to enter a new date, the format is YYMMDD.

  4. Press the Hash key to enter and confirm the new date.

  5. Press the Back key or wait 30 seconds to exit the User stack.

  6. After leaving the User stack press the Clock/number 1 key to view and verify the new date.
