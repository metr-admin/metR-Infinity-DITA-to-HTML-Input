Bypass Operation

Note: The bypass mains needs to be available and within tolerance. The static switch EN needs to be available. The switch over does not interrupt the load.
To switch between bypass operation and normal operation:

  1. Press the Hash key to enter the User stack.

  2. Use the arrow keys to navigate to Bypass operation.

  3. To turn on: Press the Clock/number 1 key to switch from normal operation to bypass operation. The load is now supplied by the bypass mains via the static switch EN.

    To turn off: Press the Star/number 0 key to switch from bypass operation to normal operation. The load is now supplied by the inverter.

  4. Press the Back key or wait 30 seconds to exit the User stack.
