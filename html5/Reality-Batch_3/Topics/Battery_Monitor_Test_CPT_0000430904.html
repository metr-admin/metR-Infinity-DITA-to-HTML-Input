Battery Monitor Test

Note: This menu option is only available if advanced battery monitor is installed.

The battery monitor test temporarily reduces the rectifier output to discharge the batteries. The test is completed when 25% of the total battery capacity is used and the rectifier output automatically returns to the defined level.

Note: The automatic switch over does not interrupt the output.
To start the battery monitor test:

  1. Press the Hash key to enter the User stack.

  2. Use the Arrow keys to navigate to Battery monitor test.

  3. Press the Clock/key number 1 to start the test.

    If you need to interrupt the test while it is running, press the Clock/key number 0 to stop the test.

  4. Press the Back key or wait 30 seconds to exit the User stack.

If the result from the battery monitoring test is within the tolerance range no alarm is shown in the Display. If the result is outside the tolerance range the Display shows one of the alarms:

  • Battery monitor warning: The battery capacity is weak. The batteries have to be replaced soon.

  • Battery monitor alarm: The battery capacity is gone. The batteries must be replaced immediately.
