Function Check of the ISBS Solution

Note
loss of output voltage
When the manual bypass switch is in Test or Bypass position, there is no backup from the battery it is not possible for the system to automatically switch back to inverter operation in case the bypass mains becomes unavailable.
Note: The bypass mains needs to be available and within tolerance. The static switch EN needs to be available. The switch over do not interrupt to the load.

How to Check the Function of the ISBS Controller

  1. Check that the system is in normal operation and that no alarms are active.

  2. Switch to bypass operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Clock/number 1 key to turn on bypass operation. The load is now supplied from the bypass mains via the static switch EN.

    4. Press the Back key or wait 30 seconds to exit the User stack.

  3. Only for 3-position manual bypass switches: Turn the manual bypass switch to the Test position.

  4. Only for 2-position manual bypass switches: Check that the EN input switch is closed. Turn the manual bypass switch to the Bypass position.

  5. Check the status of all indication LEDs on the ISBS controller (A703) is ok:

    • The STATUS_1 LED should be flashing slowly (0.5 Hz).

    • The STATUS_3 LED should be turned off.

    • The 15V FAN LED is not used.

    • All other LEDs should be turned on.

    For more information about the LEDs see the chapter Troubleshooting the ISBS in the Operating Manual.

  6. Monitor the system output voltage before the manual bypass switch.

  7. Press the Reset button on the controller (A070).

  8. Check that:

    • The ISBS controller (A703) switched to independent fire.

    • The STATUS_3 LED is turned on.

    • The STATUS_1 LED is flashing very fast (4Hz).

    • The output is supplied over the static switch EN.

    • The break in output voltage < 20ms.

    • The Display shows Standby.

    • It is not possible to turn the system on by pressing the On button on the front panel. The system should be locked in ISBS operation.

  9. Only for 3-position manual bypass switches: Turn the manual bypass switch to the Bypass position.

  10. Only for 2-position manual bypass switches: Open the EN input switch.

  11. Open the battery breaker to disconnect the battery bank.

  12. Open the mains breaker to turn off the supply to the rectifier.

  13. Open the inverter input switch to disconnect the DC power supply.

  14. Discharge the DC capacitor by keeping the charging/discharge switch in position 1 until the warning LED turns off.

Restart the System

After the function of the ISBS have been tested and the controller board have been reset the system can be started again. To restart the system:

  1. Check that the manual bypass switch is in the Bypass position.

  2. Only for 3-position manual bypass switches: Turn the manual bypass switch to the Test position.

  3. Only for 2-position manual bypass switches: Close the EN input switch.

  4. Close the rectifier mains breaker to turn on the power supply to the rectifier.

  5. Make sure the external source of the DC input power is available.

  6. Charge the DC capacitors:

    1. Keep the charging/discharging switch it in position 2 (charge) until the warning LED turns off.

    2. With the charging/discharging switch still in position 2 (charge), close the inverter input switch. If not possible, release the charging/discharging switch and quickly close the inverter input switch.

    3. Release the charging/discharging switch.

  7. After about 10 seconds the Display shows Standby.

    Note: If the unit is programmed for autostart and the input is within the tolerance range the unit will start automatically after 60 seconds.

  8. Press the Lamp test button and check that all operating status and alarm LEDâ€™s are illuminated.

  9. Press the ON button for one second on the front panel. After a few seconds the Display will show Normal operation load power ...%.

  10. Press the Number 8 key to check the DC voltage.

  11. If the DC voltage is within tolerance, close the battery breaker.

  12. Press the Number 9 key to check that the system output voltage is within the tolerance range.

  13. Switch to bypass operation:

    Note: The bypass mains needs to be available and within tolerance. The static switch EN needs to be available.

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Clock/number 1 key to turn on bypass operation.

    4. Check that the green static switch EN LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

    Important: Never turn the manual bypass switch from Test to Auto if the static switch EN is not active.

  14. Turn the manual bypass switch to the Auto position.

  15. Switch to normal operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Star/number 0 key to turn off bypass operation and switch to normal operation.

    4. If applicable, check that the green static switch EA LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

  16. Confirm any alarms by pressing the C key twice.

  17. Make sure the system operates correctly and no alarms are present.

The system is in normal operation and the load is supplied by the system.
