How to Reset the System After EPO

If the emergency power off (EPO) signal has tripped the output breaker, the signal needs to reset and the output breaker needs to be closed. If the emergency power off (EPO) signal has tripped the output breaker(s), the signal needs to reset and the output breaker(s) need to be closed.
  1. Make sure that the EPO signal is no longer active.
  2. Check that the load is ready to be energized.
  3. Reset the EPO signal from the Display:
    1. Make sure you are logged in as a user with at least Owner access.
      Note: If you are not logged in, a prompt to login will appear when trying to access the menu.
    2. Press the Main menu icon and navigate to Service > Remote control > Remote control.
    3. Press the Emergency Switch RESET on button.
      The EPO signal is now reset.
  4. Close the rectifier DC output breaker. Close the converter DC output breaker. Close the system AC output breaker. Close the system AC output breaker and if available, close the rectifier DC output breaker.
