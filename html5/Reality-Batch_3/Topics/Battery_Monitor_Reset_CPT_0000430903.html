Battery Monitor Reset

Note: This menu option is only available if advanced battery monitor is installed.

An activated battery monitor warning or alarm can be manually reset.

To reset the battery monitor alarms:

  1. Press the Hash key to enter the User stack.

  2. Use the Arrow keys to navigate to Battery monitor reset.

  3. Press the Clock/key number 1 to clear the battery monitor alarm from the alarm stack.

  4. Press the Back key or wait 30 seconds to exit the User stack.
