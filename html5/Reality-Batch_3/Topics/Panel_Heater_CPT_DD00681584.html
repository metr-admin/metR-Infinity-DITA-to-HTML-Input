Panel Heater

As an option, it is possible to add a panel heater inside the cabinet. The panel heater is fixed with screws.

For a set temperature, the heater is turned on if the temperature is below or turned off it the temperature is above.

The supply for the heater is connected to a push-in terminal. It needs to be connected to an external supply on the customer site.

Note: For details on the connection see the Terminal Diagrams.
