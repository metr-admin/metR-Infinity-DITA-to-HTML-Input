How to Replace the Manual Bypass Switch Module

Note: The manual bypass switch is not part of the replacement schedule.
DANGER
HAZARD OF ELECTRIC SHOCK, EXPLOSION, OR ARC FLASH
  • Always use appropriate personal protective equipment (PPE).
  • Follow safe electrical work practices. See NFPA 70E or CSA Z462.
  • Only qualified personnel are allowed to replace power modules.
  • All protective covers and blind covers must be installed when the module is replaced.
  • Do not install power modules that have been damaged in any way or that have an unknown operational state.
  • Keep a minimum distance of 200 mm (8 in) from the cabinet front while the system is energized.
  • Do not reach or use any tools inside the empty power module slot.
Warning
Falling module and sharp edges
  • Always wear protective gloves when handling modules.
  • If installing modules above shoulder height always use a hard hat.
  • If needed use an appropriate platform ladder to reach the rack.
  • Always follow your site's safety recommendations and any local laws and regulations.
CAUTION
Unintended operation
Make sure that the manual bypass switch module is inserted in the correct slot, second slot from the right in the bottom inverter rack.
  1. On the Display check that the power flow indicates static switch output.
  2. Open the door and verify that the installed manual bypass switch is in the static switch output position (Position 0).
    Important: If the manual bypass switch is in any other position, the supply to the load might be interrupted.
  3. Loosen the screw (A) of the module.
  4. Remove the module (B) completely with both hands.
  5. Correctly dispose of the module according to your local laws and regulations.
  6. Check that the new manual bypass switch is in static switch output position (Position 0) before inserting the module.
    Important: If the manual bypass switch is in any other position, the supply to the load might be interrupted.
  7. Carefully insert the new module (B) with both hands.
  8. Make sure that the module is fully inserted and that the module front is flush to the protective covers.
  9. Tighten the screw (A) at the bottom.
  10. Close the door and check that the Display indicates the correct power flow and that no alarm LED is active.
  11. On the display, press the Static Switch Icon to check that the is no warning messages.
The manual bypass switch is now replaced. It is recommended to test the function of the new manual bypass switch by switching between the positions. For more information see the Operation Manual.
