How to Start the Relay Tests

During the relay test the relay switches to its active state. Before starting make sure that it is approved to perform the test, especially if the active relay signal triggers a remote action.
  1. Make sure you are logged in as a user with Service access.
  2. Press the Main menu icon and navigate to Service > General > Relaytest.
  3. Press on the Start button for the relay you want to test.
    Note: For more information about the relays refer to the Drawings.
