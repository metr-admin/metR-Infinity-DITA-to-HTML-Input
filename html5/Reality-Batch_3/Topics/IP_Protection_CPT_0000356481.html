IP Protection

Notice
Only install the rain roofs, ventilation grids, filter mats or gland plates when the transport units are transported to the final installation location.
Each transport unit contains:

  • Front and back ventilation grids

  • Top and/or bottom mounted gland plate(s) (location and size depend on the system configuration)

  • Bottom ventilation grid (only for IP2x, IP3x, IP4x)

  • Filter mats (only for IP5x)

  • Rain roof (only for IPx1, IPx2)

Note: See the Rating Plate or the Technical Data in the manufacturing record book for the systems protection class.
