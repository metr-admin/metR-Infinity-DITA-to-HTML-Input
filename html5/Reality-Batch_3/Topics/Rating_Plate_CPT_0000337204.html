Rating Plate

The rating plate contains system specifications, operation parameters and installation requirements. The rating plate is mounted on each transport unit for identification.

  • (A) System type
  • (B) System type designation
  • (C) Mains input values
  • (D) Bypass input values (Not used)
  • (E) Battery values (Not used)
  • (F) Output values
  • (G) Protection degree and ambient temperature range
  • (H) Place for certification/conformity mark (e.g. CE, EAC, UL)
  • (I) Unique identifier and manufacturing information:

  • Reality project number

  • System serial number

  • Transport unit number

  • Year of manufacturing

  • Country of origin / Country of manufacturing

System Type Designation

The top section (B) of the rating plate contains the system type designation. The system type designation contains information about the system configuration.

(O)

Indicates if and what static switches the system has.
  • EA = Static switch EA (Inverter)
  • EN = Static switch EN (Bypass)
  • EAN = Both static switches EA and EN

Indicates the system type.
  • None = Single system
  • R = Redundant system
