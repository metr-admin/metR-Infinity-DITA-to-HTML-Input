Push In Terminal Type Size ≤ 16 mm2 (AWG 6)

Insert Cables

  1. Fully Insert the cable in (B).

    1. If the cable can’t be fully inserted manually, the tension spring needs to be opened by carefully pressing down (A) with a flat screwdriver.

      For terminal type size ≤ 4 mm2 (AWG 12): use a flat screwdriver of the size 0.6 x 3.5 mm (0.02 x 0.13 in).

      For terminal type size ≥ 6 mm2 (AWG 10): use a flat screwdriver of the size 1 x 5.5 mm (0.03 x 0.21 in).

  2. Make sure the cable is securely fastened.

Remove Cables

  1. Carefully press down the orange part (A) with a flat screwdriver of the size to open the tension spring.

    For terminal type size ≤ 4 mm2 (AWG 12): use a flat screwdriver of the size 0.6 x 3.5 mm (0.02 x 0.13 in).

    For terminal type size ≥ 6 mm2 (AWG 10): use a flat screwdriver of the size 1 x 5.5 mm (0.03 x 0.21 in).

  2. Remove the cable from (B).

  3. Remove the screwdriver from (A).
