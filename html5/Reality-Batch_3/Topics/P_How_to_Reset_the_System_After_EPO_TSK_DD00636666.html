How to Reset the System After EPO

If the EPO is triggered the system needs to be manually turned on when the signal is no longer active.
  1. Make sure that the EPO signal is no longer active.
  2. Check that the load is ready to be energized.
  3. Press the Switch operation icon in the Status bar.
  4. Press the Switch system on button. Confirm the change.
The system is now turned on and the system output is energized.
