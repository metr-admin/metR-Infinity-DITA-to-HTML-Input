SNMP Access to the Network Management Card

SNMP is an internet standard protocol for IP networks. The NMC supports SNMPv3, SNMPv2c over SNMPv1 and SNMPv1.

With SNMP a Network Management System (NMS) can access the NMC by configuring a user profile (SNMPv3) or a community (SNMPv1). To view the traps a trap receiver also needs to be configured for the NMS.

Super users, administrators and network only users can configure the settings for SNMPv3 and SNMPv1 from the CLI or the web interface.

Note: Reality system ignores any SET commands, only GET commands and traps can be used.
Note: Only for NMC AOS v6.8.0 and higher: Both SNMPv3 and SNMPv1 are disabled by default. There are no default settings available. The settings need to be defined before it is possible to enable the protocols.
