Dual System

A dual system consists of two independent units, that supply two independent loads. As an option, a normally open (NO) couple switch makes it possible for a dual system to function as a parallel system. For information about the couple switch see section Optional Output Couple Switch.

Important: If a dual system uses a couple switch, or the outputs are connected downstream the bypass mains must be connected to the same source. In these cases, both inverters must be synchronized to the same frequency. For more information about the synchronization in parallel systems see Output Synchronization.
Important: If a dual system uses a couple switch, or the outputs are connected downstream a blocking diode in both systems are necessary.
