Basic Functionality Without a Bypass of a Charger

In an uninterruptible power supply (UPS) a charger a rectifier converts alternating current (AC) from the rectifier mains to direct current (DC). The DC circuit supplies the load and charges the battery bank. The inverter converts back DC to AC that supplies the load. If the rectifier mains becomes unavailable, the load is supplied without interruption by the battery bank.

A system without a bypass can for example be used as a frequency converter. With this configuration no bypass operation or manual bypass is available.

In an inverter system an inverter converts direct current (DC) to alternating current (AC) that supplies the load.

A system without a bypass can for example be used together with a DC UPS system as a frequency converter. With this configuration there is no alternative supply for the load.
