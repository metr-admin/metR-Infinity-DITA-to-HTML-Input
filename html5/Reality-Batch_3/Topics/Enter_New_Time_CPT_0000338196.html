Enter New Time

To enter a new time:

  1. Press the Hash key to enter the User stack.

  2. Use the Arrow keys to navigate to Enter new time.

  3. Use the Numpad (Number 0â€“9 keys) to enter a new time, the format is HHMMSS.

  4. Press the Hash key to enter and confirm the new time.

  5. Press the Back key or wait 30 seconds to exit the User stack.

  6. After leaving the User stack press the Clock/number 1 key to view and verify the new time.
