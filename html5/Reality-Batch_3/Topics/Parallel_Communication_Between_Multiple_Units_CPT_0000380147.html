Communication Between Units (Parallel Communication)

Redundant systems always have parallel communication cables. The parallel communication cables are optional in dual systems but must be used if the system has a couple switch.

The parallel communication is redundant and sends the same signals in both cables. The signals are sent between the parallel operation board (A024) and the controller board (A070) in both units.

Redundant systems always have a parallel communication cable. The signal is sent between the external connection boards (A025) in both units.

The parallel communication controls:

  • Output Synchronization

  • Operation Mode Priority

  • Output Synchronization
  • Operation Mode Priority
