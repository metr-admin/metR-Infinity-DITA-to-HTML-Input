Auto Start

If auto start is enabled, the system starts automatically when the rectifier mains DC power supply returns in case the system lost all AC and DC power supplies. This can be useful if the system is located in a remote access location.

To enable/disable auto start for the system:

  1. Press the Hash key to enter the User stack.

  2. Use the Arrow keys to navigate to Autostart.

  3. To enable: Press the Clock/number 1 key to turn on auto start.

    To disable: Press the Star/number 0 key to turn off auto start.

  4. Press the Back key or wait 30 seconds to exit the User stack.
