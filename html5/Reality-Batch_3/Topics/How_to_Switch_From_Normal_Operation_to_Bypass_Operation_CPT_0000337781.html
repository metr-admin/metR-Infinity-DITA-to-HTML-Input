How to Switch from Normal Operation to Bypass Operation

Note: The bypass mains needs to be available and within tolerance. The static switch EN needs to be available. The switch over do not interrupt the load.

  1. Switch to bypass operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Clock/number 1 key to turn on bypass operation.

    4. Check that the green static switch EN LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

  2. Confirm any alarms by pressing the C key twice.

  3. Make sure the system operates correctly and no alarms are present.

The system is in bypass operation and load is supplied by the bypass mains via the static switch EN.
