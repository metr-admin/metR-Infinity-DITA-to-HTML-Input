Hot Standby (Redundant Systems)

Hot standby is an additional mode only for redundant systems.

If one unit tries to change to a lower priority operating mode, but the other unit is running in normal operation, the unit will instead switch to hot standby. In hot standby the unit is ready to operate when it can and if needed.

For example, if one unit is in hot standby (unable to run in normal operation but can run in battery bypass operation) and the other unit running in normal operation switches to battery bypass operation, the unit in hot standby will also switch to battery bypass operation. Whereas if the unit was in standby (instead of hot standby) the unit could not have switched over to battery bypass operation.

For more information about the behavior see Operation Mode Priority.
