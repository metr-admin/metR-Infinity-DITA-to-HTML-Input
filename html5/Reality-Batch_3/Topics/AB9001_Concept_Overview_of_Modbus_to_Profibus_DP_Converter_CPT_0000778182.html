Concept Overview of Modbus to Profibus DP Converter

To convert the alarm signals sent from the system to Profibus DP a AB9001 communication gateway is used. The AB9001 gateway converts the Modbus TCP/ IP signal sent from the network management card (NMC).

The NMC card is required for using the AB9001 communication gateway. The connection between the NMC and the AB9001 uses a straight Ethernet cable (RJ45 connection).

A connection can then be made to the AB9001 gateway with a straight Ethernet cable (RJ45 connection) to access the Profibus DP signals.
