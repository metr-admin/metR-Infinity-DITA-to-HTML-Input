Replace or Rotate Modules with Spare Modules

  1. If necessary, replace any inoperable modules with spare modules. For more information see:
    1. How to Replace or Rotate a Rectifier Module
    2. How to Replace or Rotate a DC/DC Converter Module
    3. How to Replace or Rotate an Inverter Module
    4. How to Replace the Static Switch Module
    5. How to Replace the Manual Bypass Switch Module
  2. Rotate any spare modules currently on stock with modules in the system. For more information see:
    1. How to Replace or Rotate a Rectifier Module
    2. How to Replace or Rotate an Inverter Module
    3. How to Replace or Rotate a DC/DC Converter Module
    Note: Extra modules should be swapped with installed modules during the annual maintenance. Capacitors inside the modules must be energized regularly, therefore extra modules should not be stored for more than 1 year in a row.
