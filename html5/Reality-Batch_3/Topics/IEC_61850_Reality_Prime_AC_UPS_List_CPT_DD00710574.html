IEC 61850 Reality Prime AC UPS List

The default logical device (LD) for all IEC tags for Reality Prime AC UPS is “i61PRIMEACUPS_LD1”.

The IEC tag path format is LN$FC$DO$DA.

  • Logical node (LN)

  • Functional constraint (FC)

  • Data object (DO)

  • Data attribute (DA)

Note: The Data Type for Status is Boolean and for Analog is UINT32 (2).
