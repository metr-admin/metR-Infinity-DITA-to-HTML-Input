Bypass via Static Switch

The load is supplied from the bypass via the static switch. In this state the static switch can automatically switch over to the inverter if necessary, as long as the inverter output is available and within tolerance range.

A system might have automatically switched to bypass via static switch because the DC input is unavailable or the inverter modules are inoperable.
