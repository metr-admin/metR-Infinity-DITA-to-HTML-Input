Change Passwords

CAUTION
Loss oF LOGIN ACCESS
There is no way to read out or retrieve a password.
Make sure new passwords are stored in a secure way, for example in a password manager.
If the password is lost a purchase of a new controller board may be necessary.
No responsibility or liability is assumed by Reality if a changed password is lost and on-site support is needed.

Password Criteria

All new passwords must meet the following criteria:

  • Each access level must have a unique password.

  • The password must consist of 8 to 16 characters.

  • The password must consist of at least 1 of the special characters, see the table below.

When entering a password, the numbers 0–9 are entered using the Number keys. Letters and special characters are entered by cycling through the list of available password letters and characters with the Arrow up key and the Arrow down key.

Note: To delete the last entered character press the C key.

Login to Master Mode and How to a Change Password

To access the change password feature a login to the master mode is needed.

  1. Press both the Clock/number 1 key and the Number 2 key at the same time. The Display will show Master Password: .

  2. Use the numpad keys to enter the password for Master Mode. Press the Hash key to confirm. The Display will show Enter Password:.

  3. Use the numpad keys to enter the password for Change Password. Press the Hash key to confirm. The Display will show Change Password:.

  4. Use the numpad keys to enter the existing password for the access level you want to change, for example if changing the default master mode password enter 111111. Press the Hash key to confirm. The Display will show Enter New PW:.

  5. Use the numpad numbers and arrow keys to enter a new password consisting of at least 8 characters, including at least 1 special character. Press the Hash key to confirm. The Display will show Confirm New PW:.

  6. Use the numpad keys to repeat the new password. Press the Hash key to confirm.

The password is changed.

Logout from Master Mode

To logout from master mode manually:

  1. Press both the Clock/number 1 key and the Number 2 key at the same time. The Display will show Enter Password:.

  2. Press the Number 6 key and then confirm with the Hash key.

You are now logged out from master mode.
Note: If no numpad key is pressed for 2 minutes you are automatically logged out from master mode.
