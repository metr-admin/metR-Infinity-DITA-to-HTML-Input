Static Switch Module

The static switch (SSW) module automatically switches between the inverter output and the bypass output, depending on what source is available. The module is fan cooled.

Settings for the static switch module is done through the system display.

The static switch module have one AC supply and one DC supply. There are two different static switch modules depending on the DC voltage used in the system.

DC voltage Output AC voltage
24, 48 or 60 V DC 110, 120, 127, 220, 230, or 240 V AC
110/125 or 220 V DC
