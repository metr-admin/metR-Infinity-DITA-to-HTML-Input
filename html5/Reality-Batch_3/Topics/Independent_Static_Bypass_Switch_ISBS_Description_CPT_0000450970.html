Independent Static Bypass Switch (ISBS)

The independent static bypass switch (ISBS) is an option with independent power supply for systems with a bypass and a manual bypass switch. It provides a backup for the automatic transfer to static bypass in case the main controller cannot turn on the static switch EN.

The ISBS monitors the system output and the status of the main controller to determine if a switch to static bypass if needed. When the switch over to static bypass is triggered by the ISBS it is called ISBS operation and is indicated in the display as ISBS fired independent. In ISBS operation the system is locked in static bypass.

As an option, in addition to the system alarms in the display, two relays can be added (K701) and (K702) to indicate:

  • ISBS status ok: The status of the ISBS.

  • ISBS operation active: The ISBS has fired independently and switched over the system to ISBS operation.

The same information can also be seen on the board. The status and activation of the ISBS is also indicated by the LEDs STATUS_2 and STATUS_3 respectively. For more detailed information of the LEDs on the board see Troubleshoot the ISBS.
