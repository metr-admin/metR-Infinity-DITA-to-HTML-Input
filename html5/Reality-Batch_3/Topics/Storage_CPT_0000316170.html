Storage

If the system needs to be stored a short time after delivery but before installation, the system should be stored in the transportation crate with the sealed inner packing intact.

Important: Batteries have different storage requirements than systems and depends on, for example, type, charge and manufacturer. If the batteries have to be stored after delivery but before installation the requirements of the original equipment manufacturer (OEM) have to be followed.
System storage environment:

  • Temperature:

    • Recommended: -10°C to +55°C (+14°F to +131°F)

    • Allowable min/max: -20°C to +70°C (-4°F to +158°F)1

  • Relative humidity: 0–95% Non-condensing

If the system needs to be stored for a longer time the system have to be stored according to the specifications and instructions in the chapter Storage of the System.
