Unique Identifier

The bottom section (I) on the rating plate contains the unique identifier. The unique identifier is built up of sections to make it possible to identify the cabinet.

Position Name Description
(R) Reality project number A unique number for each project.
(S) System serial number Indicates a specific system number in the project.
(T) Transport unit number The number indicates a transport unit. A system may consist of multiple transport units that can be separated for transportation.
