Boost Charge

If boost charge is enabled the system will charge the batteries with the boost charge voltage instead of the float charge voltage for a predefined time.

To enable/disable boost charge:

  1. Press the Hash key to enter the User stack.

  2. Use the Arrow keys to navigate to Boost charge.

  3. To enable: Press the Clock/number 1 key to turn on boost charge.

    To disable: Press the Star/number 0 key to turn off boost charge.

    Note: On the mimic diagram, the yellow rectifier LED will indicate that boost charge is active.

  4. The system will only apply boost charge for a set time, between 0 and 99 hours (default setting is 8 hours). To change this setting please contact your local Reality Service Center.

  5. Press the Back key or wait 30 seconds to exit the User stack.
