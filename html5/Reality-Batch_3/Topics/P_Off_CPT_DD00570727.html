Off

In the Unit state Off the rectifier, inverter and static switch are turned off.

Note: The Unit state is indicated in the bottom left corner of the Display.

Follow the procedure for how to turn off the system completely, the off button will only turn off the rectifier, inverter and static switch. The system is completely turned off when all the breakers are also open.

When the system is off there is no output voltage and the load is not supplied. When all breakers are open the display will also turn off.
