IEC 61850 PxW List

The default logical device (LD) for all IEC tags for PxW is V1_XXW_1_LD1.

The IEC tag path format is LN$FC$DO$DA.

  • Logical node (LN)

  • Functional constraint (FC)

  • Data object (DO)

  • Data attribute (DA)

Note: Depending on the system type and design, some values might not be available.
