Redundant System

A redundant system consists of two units working together to supply one load. Each unit supplies 50% of the total load. If one unit in a redundant system becomes inoperable, the total load is supplied by the other unit.

Important: The redundancy is only available if both output isolation switches are closed. If one output isolation switch is open the load is only supplied by one unit and the redundancy is lost. Never open the output isolation switches during normal operation.
Note: The units in redundant systems always have a blocking diode and a rectifier output switch.
Note: In redundant systems with individual batteries the rectifier output switches are placed after the battery connections.
