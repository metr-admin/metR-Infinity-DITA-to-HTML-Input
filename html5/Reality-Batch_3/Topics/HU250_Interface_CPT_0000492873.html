HU250 Interface

The HU250 web server provides both high-level overviews and detailed information about the status of the device, network and signals.

  • Home: The main dashboard with general information about the device.

  • Monitoring & control: Overviews of the setup or device. Detailed information about for data, signals and measurements.

  • Diagnostics: Logs files for events, system and cyber security. Troubleshooting information.

  • Maintenance: Device configuration for clock, network and physical ports.

  • Settings:

    Settings for external devices and optional modules.
Note: The menu options will depend on the user access settings.
