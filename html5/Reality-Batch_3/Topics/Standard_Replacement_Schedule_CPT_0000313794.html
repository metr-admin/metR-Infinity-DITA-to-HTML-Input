Standard Replacement Schedule

As part of the standard replacement schedule the following parts need to be replaced.

Part Reference Designator Normal Scope Extended Scope Replacement Cycle Notes
Fan sub-assembly E0.. X X Every 4-6 years Part of A060, A061
Capacitors CB02 (1CB2, 2CB2) CB02, CB03, CB04 X X Every 7-9 years
AC and DC power supply units (PSU) A201, A202 X X Every 8-10 years
Bypass stabilizer controller unit N1 X X Every 5 years If applicable, check the parts list for the system.
Interface board A071 X X Every 8-10 years Includes AC and DC power supply units (PSU).
Battery bank — X X See documentation from the battery manufacturer An optional battery monitor can be installed to notify when a battery needs to be replaced.
Air filter — X X When necessary Check the air filter at least every 6 months. If the air filter is dirty, replace it immediately. For more information see Check and replace the filter mats.
