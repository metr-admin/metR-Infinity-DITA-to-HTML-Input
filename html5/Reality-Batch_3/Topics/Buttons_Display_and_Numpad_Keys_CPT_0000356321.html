Buttons, Display and Numpad Keys

The three operational buttons are placed to the left of the display and numpad keys. All the numpad keys are used to enter the respective number in an input field, for example when setting a value.

Position

Name

Description

(A)

Lamp test button

Press to test all the LEDs on the front panel.

(B)

OFF button

Press together with the ON button for two seconds to turn off the system.

(C)

ON button

Press for one second to turn on the system.

(D)

Display

The display of the system interface. By default, the current operating mode is displayed.

(E)

Arrow up key

Used to navigate up between the selections in the system interface.

Press together with some numerical keys for additional options or measurements.

(F)

Arrow down key

Used to navigate down between the selections in the system interface.

(G)

Hash key

Press to enter the user stack.

In the user stack, press to confirm an option when using the C key to cycle between options.

(H)

Mute key

Press to mute the audible alarm (horn). Any new alarms will activate the audible alarm again.

(I)

C key

Press to confirm alarms, for more info see Confirm/Clear Alarms.

Press to cycle between options for a selection in the interface.

(J)

Star/number 0 key

Used to deactivate a mode/setting or decline a selection in the interface.

Press to enter the alarm stack.

Press together with the Arrow up key to enter the event log.

(K)

Back key

Used to exit a menu in the interface or to erase the latest entry if keying in a number.

(L)

Clock/number 1 key

Press to display the current date and time.

Used to activate a mode/setting or confirm a selection in the interface.

Press together with the Number 2 key to open the password input, for example to reset the event log.

(M)

Number 2 key

Press to display the inverter current.

If a battery sensor is installed, press to display the battery temperature.

Press together with the Clock/number 1 key to open the password input, for example to reset the event log.

(N)

Number 3 key

Press to display the output frequency.

—

(O)

Number 4 key

Press to display the mains current in the display.

—

(P)

Number 5 key

Press to display the battery current in the display.

Press together with the Arrow up key to display the total DC current.

If a battery sensor is installed, press together with the Number 8 key to display the battery temperature.

Press to display the DC current in the display.

(Q)

Number 6 key

Press to display the output current in the display.

Press together with the Arrow up key to display the output peak current.

(R)

Number 7 key

Press to display the mains voltage in the display.

Press together with the Arrow up key to display the bypass mains voltage (Mains 2).

Press to display the bypass mains voltage (Mains 2).

(S)

Number 8 key

Press to display the DC / battery voltage in the display.

If a battery sensor is installed, press together with the Number 5 key to display battery temperature.

Press to display the DC voltage in the display.

(T)

Number 9 key

Press to display the output voltage in the display.1

(U)

Common alarm

The common alarm LED indicates that there is at least one active alarm in the system. For more information see the chapter Alarm Stack and Event Log.

Note: There are a few alarms that does not affect the common alarm.
