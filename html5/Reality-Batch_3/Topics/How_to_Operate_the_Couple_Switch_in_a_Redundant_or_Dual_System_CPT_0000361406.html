How to Operate the Output Couple Switch

Note
loss of Redundancy
When the units in a redundant system are separated the redundancy of the output is lost.

Systems with multiple units can have an output couple switch to control the load sharing between the units.

Redundant systems use a normally closed (NC) output couple switch to share the load by default. Dual systems use a normally open (NO) output couple switch to supply individual loads by default.

NO and NC output couple switches are operated in the same way.

How to Open the Output Couple Switch

To open the output couple switch and separate the outputs:

  1. Check that:

    1. The system is running in normal operation float charge (with all units active).

    2. No alarms are pending on any unit.

  2. Open the output couple switch.

The system output is now separated and each unit is supplying its own load.

How to Close the Output Couple Switch

To close the output couple switch and connect the outputs:

  1. Check that:

    1. The units are all running in normal operation float charge (individually).

    2. The bypass mains and static switch EN are available for all units and within tolerance (check status on the mimic diagram).

    3. No alarms are pending.

  2. Switch over unit1 to static bypass operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Clock/number 1 key to turn on bypass operation.

    4. Check that the green static switch EN LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

  3. Follow the same procedure and switch over unit2 to bypass operation.

  4. Check that the Display on both units show Bypass operation.

  5. Check that the external load connection is closed.

  6. Close the output couple switch.

  7. Switch over unit1 to normal operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Star/number 0 key to turn off bypass operation and switch to normal operation.

    4. If applicable, check that the green static switch EA LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

    Note: The load will be supplied by the bypass until all units in the system have been switched back to normal operation.

  8. Follow the same procedure and switch over unit2 to normal operation.

  9. Check that the Display on both units show Normal operation load power ...%.

  10. Confirm any alarms by pressing the C key twice.

  11. Make sure the system operates correctly and no alarms are present.

The system is now in redundant/parallel operation with all units supplying an equal share of the load.
