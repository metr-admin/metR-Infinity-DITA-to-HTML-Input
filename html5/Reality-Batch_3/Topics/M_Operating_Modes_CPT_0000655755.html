The modules will automatically start up as soon as the rectifier mains is available. There is no standby mode.
The modules will automatically start up as soon as the DC input is available. There is no standby mode.

The systems operation mode depends on different settings and status of the system. The system will automatically detect the appropriate operation mode and automatically switch to it, depending on its configuration. It is also possible to manually switch to most operation modes.

Some modes are only valid for certain system types or configurations. For more information about the operation modes and system states see the sections:

  • Float Charge

  • Boost Charge (Charge)

  • Equalization Charge

  • Battery Operation (Batterytest)

Static Switch and Manual Bypass Modes

  • Bypass via Static Switch

  • Inverter via Manual Bypass Switch (Inverter Output)

  • Bypass via Manual Bypass Switch (Bypass Output)

System States
