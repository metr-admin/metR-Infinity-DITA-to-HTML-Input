How to Change Back to Static Switch Output from Bypass or Inverter Output

Manual bypass switch (MBS) positions:
  • 0 = Static switch (SSW) output
  • 1 = Bypass output
  • 2 = Inverter output
Note: If active, turn off the audible buzzer by pressing the Mute icon in the upper right corner of the Display. The Mute icon is only visible if the audible buzzer is active.
  1. On the Display check that no alarm LED is active. On the Display check that no alarm LED is active except the Manual bypass active and SBS Fault, if they are used.
    Note: There will be a warning triangle on the Static switch icon that indicates the SSW is in bypass or inverter output.
  2. On the Display check that the power flow indicates either bypass output or inverter output.
  3. Open the door and rotate the manual bypass switch to the static switch output position (Position 0).
  4. Close the door.
  5. On the Display check that the power flow indicates static switch output.
The system is in SSW output mode and will automatically switch between inverter or bypass as needed, depending on the available supply.
