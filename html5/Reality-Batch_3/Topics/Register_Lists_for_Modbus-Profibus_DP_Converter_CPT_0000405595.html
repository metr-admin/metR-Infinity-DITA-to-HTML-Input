The PEW distribution A/B solutions contains three gateway converter setups (the second and third setups are identical):

  • PEW Distribution A/B Setup — LUFP7 (A060)

  • PEW Distribution A/B Setup — LUFP7 (A062)

  • PEW Distribution A/B Setup — LUFP7 (A064)

The SDC solutions contains one gateway converter setup:

  • SDC Distribution A/B Setup — LUFP7 (A060)

The parameter has the format unsigned integer (byte). The byte order is the same for all parameters and follows the big-endian format:

  • Unit32: High byte, mid-high byte, mid-low byte, low byte.

  • Unit16: High byte, low byte.

Additional information about the parameters and descriptions:

  • Reality refers to the secondary Modbus devices (IM400/IFL12MC) and relate to the device itself.

  • System refers to the setup and relate to signals, communication or other inputs.

  • Each channel monitors the distribution line for the mentioned breaker. If no breaker is mentioned, the channel is reserved/not used.

  • Some alarms monitor multiple different alarms and can be seen as a common alarm, for example the Reality state (IM400/IFL12MC) or the insulation alarm (IM400).
