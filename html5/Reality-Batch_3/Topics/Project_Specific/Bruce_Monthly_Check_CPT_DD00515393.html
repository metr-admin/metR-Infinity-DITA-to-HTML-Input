Monthly Check

Make the following checks monthly:

  1. With an appropriate rated measuring instrument check:

    1. The battery voltage.

    2. The DC voltage.

    3. The system output voltage.

    4. The system output current.

  2. Record any present alarms or incorrect operating status indications.

  3. Reset any alarms by pressing the C key. Make sure that the system operates correctly and no alarms are indicated on the Alarm panel.

  4. Press and hold the Lamp test button to check that all LEDs on the front panel (alarm panel, mimic diagram and common alarm) illuminate.

  5. Perform required maintenance on the batteries, see the documentation from the original equipment manufacturer (OEM) of the batteries.
