A system check is performed during the commissioning. Pay special attention to specific steps that are only valid for some configurations and system types.

Only for single system: The system check is performed for the system. Afterwards it is in normal operation float charge, without any load connected.

Only for dual systems: The system check is performed for one unit at a time, the order of the units does not matter. Afterwards both units are in normal operation, without any load connected.

Only for redundant systems: The two first sections first section in the system check is performed for one unit at a time, starting with unit2. Afterwards unit2 is in standby charger only (standby) and unit1 is in normal operation float charge , without any load connected.

Battery Operation Check on One Unit

  1. Close the rectifier mains input breaker.

  2. After about 10 seconds the Display shows Standby.

    Note: If the unit is programmed for autostart and the input is within the tolerance range the unit will start automatically after 60 seconds.

  3. Press the Lamp test button and check that all LED’s on the front panel illuminates.

  4. Press the ON button for one second, if the unit did not start from the autostart. After a few seconds or if any key is pressed the Display shows Normal operation load power ...% Float charge load ...%.

    Important: Only for 250 V DC systems: The output terminal is energized.

  5. Only for redundant system with a common battery: Close the rectifier output switch.

  6. Press the Number 8 key to check the DC voltage.

  7. If the DC voltage is within tolerance, close the external battery breaker. The battery is now connected to the unit.

    Note: Only for redundant system with a common battery: If the system has a common battery protection breaker it will already be closed when performing the check on unit1.

  8. Press the Number 5 key to check the battery current.

  9. Press the Number 9 key to check that the inverter output voltage is within tolerance. Check that the white inverter LED on the mimic diagram is illuminated.

  10. Open the rectifier mains input breaker. Check that the unit changes to battery operation, and that the white battery LED on the mimic diagram is illuminated. The Display will show Batt. operation time ...min.

    Note: The remaining runtime will only be shown if advanced battery monitor is installed.

  11. Let the unit be in battery operation for 30 seconds to check that the Battery operation indication activates on the alarm panel.

  12. Close the rectifier mains input breaker. Check that the unit changes to normal operation float charge.

  13. Only for 48 V DC systems: Close the output breaker.

    Important: Only for 48 V DC systems: The output terminal is energized.

  14. Confirm the alarms by pressing the C key twice.

  15. Only for unit2 in redundant systems with individual batteries: Open the battery breaker.

  16. Only for unit2 in redundant systems: Press both the ON button and the OFF button at the same time for two seconds. The Display now shows Standby.

  17. Continue with the section:
    • For all systems with bypass: Bypass Operation Check on One Unit.
    • For single systems without bypass: Connecting the Output/Distribution.
    • For dual systems without bypass: Additional Steps for Dual Systems.
    • For redundant systems without bypass: Additional Steps for Redundant Systems.
    • For single systems: Connecting the Output/Distribution.
    • For redundant systems: Additional Steps for Redundant Systems.

Start Up Check on One Unit

Important: The DC capacitors needs to be charged before the inverter input switch is closed.

  1. Make sure the external source of the DC input power is available.

  2. Charge the DC capacitors:

    1. Keep the charging/discharging switch it in position 2 (charge) until the warning LED turns off.

    2. With the charging/discharging switch still in position 2 (charge), close the inverter input switch. If not possible, release the charging/discharging switch and quickly close the inverter input switch.

    3. Release the charging/discharging switch.

    Note: If the unit is programmed for autostart and the input is within the tolerance range the unit will start automatically after 60 seconds.

  3. Press the Lamp test button and check that all LED’s on the front panel illuminates.

  4. Press the ON button for one second, if the unit did not start from the autostart. After a few seconds or if any key is pressed the Display shows Normal operation load power ...%.

  5. Press the Number 8 key to check the DC voltage.

  6. Press the Number 5 key to check the DC current.

  7. Press the Number 9 key to check that the inverter output voltage is within tolerance. Check that the white inverter LED on the mimic diagram is illuminated.

  8. Confirm the alarms by pressing the C key twice.

  9. Continue with the section:
    • For all systems with bypass: Bypass Operation Check on One Unit.
    • For single systems without bypass: Connecting the Output/Distribution.
    • For dual systems without bypass: Additional Steps for Dual Systems.
    • For redundant systems without bypass: Additional Steps for Redundant Systems.

Bypass Operation Check on One Unit

DANGER
HAZARD OF ELECTRIC SHOCK, EXPLOSION, OR ARC FLASH
The open output terminal (or open output isolator switch, depending on configuration) is energized after the bypass breakers and switches are closed.
Make sure the load is kept volt free until the commissioning is complete. Only energize the load after approval by responsible personnel.
Note: For systems with a manual bypass switch: Check the position of the manual bypass switch. It should be in Bypass except if the check is performed on unit1 in a redundant system with a common manual bypass switch.

  1. Close the bypass mains input breaker.

  2. If applicable, close the bypass isolation switch.

  3. Only for 3-position manual bypass switch: Turn the manual bypass switch to the Test position.
    Note: Only for redundant systems with common manual bypass: The manual bypass switch will already be in the Test position when performing the check for unit1.

  4. Close the bypass isolation (EN input) switch.

  5. Check that the blue inverter LED on the mimic diagram is illuminated, indicating that the inverter is synchronized with the bypass mains.

  6. Check that the white LED before the static switch EN (upstream) on the mimic diagram is illuminated, indicating that the bypass mains is available and within tolerance.

  7. Switch to bypass operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Clock/number 1 key to turn on bypass operation.

    4. Check that the white static switch EN LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

  8. Check that the Display shows Bypass operation.

  9. Switch back to normal operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Star/number 0 key to turn off bypass operation and switch to normal operation.

    4. If applicable, check that the white static switch EA LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

  10. Check that the Display shows Normal operation load power ...%.

  11. Confirm any alarms by pressing the C key twice.

  12. Only for unit2 in redundant systems: Press both the ON button and the OFF button at the same time for two seconds. The Display now shows Standby Charger only (standby).

  13. Continue with the section:
    • For single or dual units with manual bypass switch(es): Set Manual Bypass Switch to Normal and Check Parallel Communication.
    • For single units without manual bypass switch(es): Connecting the Output/Distribution.
    • For dual units without manual bypass switch(es): Additional Steps for Dual Systems.
    • For redundant systems: Additional Steps for Redundant Systems.

Additional Steps for Redundant Systems

  1. Repeat the relevant sections for unit1:

    1. Battery Operation Check on One Unit.

    2. Start Up Check on One Unit

    3. Bypass Operation Check on One Unit.

  2. Repeat the Battery Operation Check on One Unit section for unit1.

  3. Check that the redundant communication cables are parallel communication cable is connected according to the specifications in the Connection Diagram.

  4. Only for unit2: Press the ON button for one second to restart unit2. Check that the blue inverter LED on the mimic diagrams is illuminated.

  5. Check that the Displays on both units show Normal operation load power ...% Float charge load ...%.

  6. Close both output isolation switches.

  7. Only for unit2 in redundant systems with individual batteries: Close the battery breaker.

  8. Confirm any alarms by pressing the C key twice.

  9. Check that alarms for the parallel communication are not active.

  10. Continue with the section Set Manual Bypass Switch to Normal and Check Parallel Communication.

  11. After both units are checked and in float charge, continue with the section Connecting the Output/Distribution.

Set Manual Bypass Switch to Normal and Check Parallel Communication

Only for redundant systems: This procedure is performed to verify that the parallel communication is working as intended. Only switch operation from unit1 and check that unit2 reacts in the same way.

  1. Switch the unit to bypass operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Clock/number 1 key to turn on bypass operation.

    4. Check that the white static switch EN LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

  2. Check that the Display shows Bypass operation.

  3. Only for redundant system: Check that unit2 also switched to bypass operation.

  4. Check that the white LED after the inverter (downstream) on the mimic diagram is illuminated, indicating that the inverter output is available and within tolerance.

  5. Only for systems with manual bypass switch(es): If the inverter output is available and within tolerance, switch the manual bypass switch to the position Normal.

  6. Switch the unit to normal operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Star/number 0 key to turn off bypass operation and switch to normal operation.

    4. If applicable, check that the white static switch EA LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

  7. Check that the Display shows Normal operation load power ...%.

  8. Only for redundant system: Check that unit2 also switched to normal operation.

  9. Confirm any alarms by pressing the C key twice.

  10. Continue with the section:
    • For single and redundant systems: Connecting the Output/Distribution.
    • For dual systems: Additional Steps for Dual Systems.

Additional Steps for Dual Systems

  1. Repeat the sections for the other unit in the system, until all applicable sections are performed and both units are checked:

    1. Battery Operation Check on One Unit.

    2. Start Up Check on One Unit

    3. Only for systems with bypass: Bypass Operation Check on One Unit.

    4. Only for systems with manual bypass switches: Set Manual Bypass Switch to Normal and Check Parallel Communication.

  2. After both units are checked and in normal operation, continue with the section Connecting the Output/Distribution.
