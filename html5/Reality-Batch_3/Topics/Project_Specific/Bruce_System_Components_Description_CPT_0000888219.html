System Components Description

The image shows a simplification of the system configuration intended as a reference for the description of the system components. See the Single Line Diagram for more details.

The system components are divided into three sections:
  • (A) The connection field. Located in the bypass or incoming.
  • (B) The system part. Located in the system cabinet behind the main protection cover.
  • (C) The low voltage section. Located in the system cabinet in front of the main protection cover.
The sections are divided to make maintenance and service easier. The connection field and system part are physically isolated from each other (different cabinets).
Note: The battery breaker is only monitored in 250 V DC systems, see the mimic diagram on the front panel of the system.

Components in the Connection Field

Designator

Component

Description

X090

Bypass mains terminals

Connection terminal for the bypass mains input. The bypass mains input requires an external protective device. A fuse or molded case circuit breaker (MCCB) according to the technical specification. L, N, PE for single phase systems. L1, L2, L3 and PE (and N if there is no isolation bypass transformer) for three phase systems. For specifications always refer to the Drawings for your system.

Q090

Bypass circuit breaker or Switch-disconnector

Breaker for the bypass mains input. The bypass mains input requires an external protective device. A fuse or MCCB according to the technical specification.

T090

Bypass transformer or stabilizer

Galvanic isolation between bypass mains input and load. Used if the required output is different from the available input. Either a transformer or stabilizer can be used.

Q050

2-position manual bypass switch

Makes it possible to isolate the system for maintenance. Positions are Normal and Bypass. For more information see Redundant System with 2p Manual Bypass Switch and both Static Switches.

Q050

3-position manual bypass switch

Makes it possible to isolate the system for maintenance and test the system without interruption. Positions are Normal, Test and Bypass. For more information see Redundant System with 2p Manual Bypass Switch and both Static Switches.

Q028

Switch-disconnector

Always used with 2-position manual bypass switches. Needed to isolates the static switch EN when the 2-position manual bypass switch is in Bypass position. This functionality is integrated in 3-position manual bypass switches.

Connection terminal for the rectifier mains DC power input. The rectifier mains DC power input requires an external protective device. A fuse or MCCB according to the technical specification. Always L1, L2, L3, and PE.

L002

Filters the AC output and removes distortions together with the CB03.

T002

Inverter power transformer

Galvanic isolation between the load and the DC. Transforms the AC output to the correct value for the load.

CB03

AC capacitor module

Filters the AC output and removes distortions together with the L002. Subdivided into several sections.

Fuse

Protects each section of CB03. If one section has a failure the other sections will be isolated.

F021

Inverter output fuse

Protection at the inverter output.

A035

Static switch EN
The static switch consists of antiparallel-switched power thyristors. It is used to automatically transfer the load to bypass without interruption. Can only be transferred if the bypass mains voltage is within tolerance. Transfer can occur when:

  • The system is heavily overload.

  • The inverter cannot keep the output voltage within tolerance.

  • The battery is discharged during battery operation.

  • DC input power is unavailable or out of tolerance.

  • The system has been switched to static bypass operation from the front panel.

F028

Fuse

Protects the static switch EN A035.

A036

Static switch EA

The static switch EA is an option at the inverter output. It can make the switch over between operating modes more reliable.

A201

Internal power supply

Supplied by the rectifier mains or the bypass mains. Part of the redundant internal power supply together with A202. One of the two supplies can supply the system alone. In case of one of the supplies stops suppling the system an alarm is generated on the front panel.

A202

Internal power supply

Supplied from the internal DC bus. Part of the redundant internal power supply together with A201. One of the two supplies can supply the system alone. In case of one of the supplies stops suppling the system an alarm is generated on the front panel.

A055

Fan system
Consist of:

  • Fan-transformer that transforms the voltage to the correct voltage to the fans.

  • Fans with internal hall sensor for monitoring the number of revolutions.

  • Fan monitor that monitors the number of revolutions of each fan. If the number of revolutions falls below the preset value an alarm is generated.

E00X

A070

Main controller board (Blackird)

Controls the main parts (rectifier, inverter, static switches) of the system and the alarm relays. Monitors the voltages, the currents and the alarms. Communicates with the front panel and the serial connections.

A071

Interface board

Contains control fuses and distributes power to auxiliary circuits and different modules. Converts AC and DC voltages to lower standard values for the controller. Secures galvanic isolation for the controller.

A073

Front panel

The human machine interface (HMI) of the system. Contains a display, operating status indications, alarm indications, operating buttons and numpad. For more information see Front Panel.

A703

ISBS Controller

The ISBS controller controls the static switch EN A035 based on the control signal from the controller board A070. It can independently switch the system to independent static bypass (ISBS) operation. The ISBS controller has an independent power supply and independent supervision of the output voltage. It is connected to the two relays K701 and K702.

A068

Adapter board

Connection interface to the controller board A070.

K701

1 relay

Indicates the status of the complete ISBS controller. If the relay is activated, the controller is fully operational and ready to react on the control signal from the controller board.

K702

1 relay

Indicates that the ISBS controller has switched the system to ISBS operation.
