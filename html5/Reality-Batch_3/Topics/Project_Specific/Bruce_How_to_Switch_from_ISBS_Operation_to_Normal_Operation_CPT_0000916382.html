How to Switch from ISS or ISBS Operation to Normal Operation

CAUTION
High current flow
Always charge the DC capacitors before closing the inverter input switch.
CAUTION
High current flow
Only connect the battery bank when the rectifier is in operation. The DC capacitors needs to be charged by the rectifier output before the battery is connected.

If the independent system surveillance (ISS) or independent static bypass switch (ISBS) has triggered the unit is locked in bypass operation. Before switching to normal operation the unit must first switch to manual bypass so that the system can restart.

Important: Always contact Reality Service Center if the ISS or ISBS was triggered during operation to analyze the cause.

To switch back to normal operation there are two steps:

  1. Turn off the system

  2. Restart the system

Switch to Manual Bypass and Turn Off the System

The unit needs to be turned off to allow the controller board to reset. To turn off the system:

  1. Only for 3-position manual bypass switches: Turn the manual bypass switch to the Test position.

  2. Only for 2-position manual bypass switches: Turn the manual bypass switch to the Bypass position.

  3. Open the battery breaker to disconnect the battery bank.

  4. Open the rectifier mains breaker to turn off the power supply to the rectifier.

  5. Open the inverter input switch to disconnect the DC power supply.

  6. Discharge the DC capacitor by keeping the charging/discharge switch in position 1 (discharge) until the warning LED turns off.

  7. Only for 3-position manual bypass switches: Turn the manual bypass switch to the Bypass position.

  8. Only for 2-position manual bypass switches: Open the bypass isolation (EN input) switch.

The system is turned off and the load is supplied by the bypass.

Restart the System

After the controller board have been reset the system can be started again, if the controller board is operable. To restart the system:

Important: The DC capacitors needs to be charged before the inverter input switch is closed.

  1. Check that the manual bypass switch is in the Bypass position.

  2. Only for 3-position manual bypass switches: Turn the manual bypass switch to the Test position.

  3. Only for 2-position manual bypass switches: Close the bypass isolation (EN input) switch.

  4. Close the rectifier mains breaker to turn on the power supply to the rectifier.

  5. Make sure the external source of the DC input power is available.

  6. Charge the DC capacitors:

    1. Keep the charging/discharging switch it in position 2 (charge) until the warning LED turns off.

    2. With the charging/discharging switch still in position 2 (charge), close the inverter input switch. If not possible, release the charging/discharging switch and quickly close the inverter input switch.

    3. Release the charging/discharging switch.

  7. After about 10 40 seconds the Display shows Standby Charger only (Standby).

    Note: If the unit is programmed for autostart and the input is within the tolerance range the unit will start automatically after 60 seconds.

  8. Press the Lamp test button and check that all operating status and alarm LEDâ€™s are illuminated.

  9. Press the ON button for one second on the front panel. After a few seconds the Display will show Normal operation load power ...%.

  10. Press the Number 8 key to check the DC voltage.

  11. If the DC voltage is within tolerance, close the battery breaker.

  12. Press the Number 9 key to check that the system output voltage is within the tolerance range.

  13. Switch to bypass operation:

    Note: The bypass mains needs to be available and within tolerance. The static switch EN needs to be available.

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Clock/number 1 key to turn on bypass operation.

    4. Check that the white static switch EN LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

    Important: Never turn the manual bypass switch to Normal if the static switch EN is not active.

  14. Turn the manual bypass switch to the Normal position.

  15. Switch to normal operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Star/number 0 key to turn off bypass operation and switch to normal operation.

    4. If applicable, check that the white static switch EA LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

  16. Confirm any alarms by pressing the C key twice.

  17. Make sure the system operates correctly and no alarms are present.

The system is in normal operation and the load is supplied by the system.
