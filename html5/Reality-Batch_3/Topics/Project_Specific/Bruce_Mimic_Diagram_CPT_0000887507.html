Mimic Diagram

The system specific mimic diagram is used to indicate the operating status of the system. From the LEDs it is possible to see the status of breakers, the rectifier, the inverter, the battery and the static switches. , the inverter and the static switches. , the rectifier and the battery.

Mimic Diagram Symbol

Symbol Name

LED Description / Symbol Description

Breaker

White: The breaker is closed

Amber: The breaker is open

Voltage Measuring

White: The voltage is available and within tolerance

Rectifier

White: The rectifier is switched on.

Blue: Boost charge (or initial charge) is enabled on the rectifier.

Battery

White: The battery supply is active.

Inverter

White: The inverter is switched on1.

Blue: The inverter is synchronized to the bypass voltage.

Blue: (Not used)

Static Switch

White: The static switch is active.

Current flow arrow

Indicates what direction the current can flow

Transformer / Stabilizer

Indicates if there is a bypass transformer or bypass stabilizer
