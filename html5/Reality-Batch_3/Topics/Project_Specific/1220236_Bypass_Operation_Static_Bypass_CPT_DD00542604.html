Bypass Operation (Static Bypass)

A system might be in bypass operation because:

  • The power supply to the inverter is unavailable.

  • The system was manually switched to bypass operation from the display panel without interruption. See chapter How to Switch from Normal Operation to Bypass Operation.

Note: Bypass operation is also called static bypass because the bypass is supplied over the static switch.

A switch to bypass operation without interruption is only possible if the bypass mains is available and within the tolerance range.

The system can be configured to switch over to bypass operation even if the inverter is not synchronized to the bypass mains. However, this will result in a 10–20 ms interruption to the output.

If the bypass mains becomes unavailable in bypass operation, the system will try to switch over to normal operation.

When the load is supplied over the static switch EN in bypass operation, the manual bypass switch is in UPS position.
