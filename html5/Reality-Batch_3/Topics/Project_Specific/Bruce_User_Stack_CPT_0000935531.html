User Stack

The user stack is used to make changes to some system settings, operating mode and general settings. User stack navigation:

  • To enter the User stack press the Hash key.

  • Navigate inside the User stack with the Arrow keys.

  • To turn on an option press the Clock/number 1 key.

  • To turn off an option press the Star/number 0 key.

  • It is also possible to use the C key to cycle between the options and the Hash key to confirm the selection. This is also needed if there are more than two options.

  • To exit the User stack press the Back key or wait 30 seconds.

In the user stack the following menu options are available:

  • System Language

  • Auto Start

  • Bypass Operation

  • Boost Charge

  • Charge Mode

  • Auto Boost Charge

  • Battery Capacity Test

  • Adaptive Slew Rate

  • Battery Monitor Test

  • Battery Monitor Reset

  • Enter New Date

  • Enter New Time
