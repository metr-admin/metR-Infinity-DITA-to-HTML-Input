Commissioning

Warning
Hazard of short circuits
Make sure the system is completely dry before commissioning. Moisture inside the cabinet can create hazardous short circuits. If the system is not completely dry, for example if there is any condensed water, let the system stand with cabinet doors open in a dry, warm and clean area for at least 24h.
CAUTION
UNINTENDED EQUIPMENT OPERATION
Before commissioning make sure that you:
  • Have read and understood the complete commissioning procedure.
  • Know the system and its configuration.
  • Have studied the Single Line Diagram.
  • Know the location of all switches, breakers and feeders described in the procedure.
Note
If a DC UPS system is used to supply the inverter system, make sure the DC UPS system is commissioned and operates correctly without any alarms before commissioning the inverter system.
