Function Check of the Unit

Note
loss of output voltage
If the system is turned off without the manual bypass in Test or Bypass the output voltage is lost.
When the manual bypass switch is in Test or Bypass position, there is no backup from the battery it is not possible for the system to automatically switch back to inverter operation in case the bypass mains becomes unavailable.
Always coordinate maintenance and turning off or on the system with the users of the consumers. Make sure that users of the connected consumers are informed before proceeding.
Notice
loss of output voltage
When the manual bypass switch is in Bypass position, there is no backup from the battery it is not possible for the system to automatically switch back to inverter operation in case the bypass mains becomes unavailable.

This function check describes how to perform a basic function check on a single system or one unit in a system with multiple units.

  1. Check that the system is in normal operation float charge and that no alarms are active.

  2. Open the rectifier mains input breaker.

  3. Check that the system switches to battery operation and that the Display shows Batt. operation time ...min.

    Note: The remaining runtime will only be shown if advanced battery monitor is installed.

  4. Open the battery breaker.

  5. Open the inverter input switch.

  6. Check that the system switches to bypass operation and that the Display shows Bypass operation.

  7. Close the rectifier mains input breaker.

  8. Keep the charging/discharging switch in position 2 to charge the DC capacitors (CB02). Keep the switch in position 2 until the warning LED turns off. With the switch still in position 2 close the inverter input switch.

  9. Check that the system switches to normal operation float charge and that the Display shows Normal operation load power ...% Float charge load ...%.

  10. Close the battery breaker.

  11. Switch to bypass operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Clock/number 1 key to turn on bypass operation.

    4. Check that the white static switch EN LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

  12. Check that the white static switch EN LED is active on the mimic diagram and that the Display shows Bypass operation.

  13. Switch to manual bypass and turn the system off and on:

    1. Turn the manual bypass switch to the Test position.

    2. Turn the manual bypass switch to the Bypass position.

    3. Check that the Manual bypass on LED is active on the alarm panel and that the white manual bypass LED is active on the mimic diagram.

    4. Turn off the system, press both the ON button and the OFF button for two seconds. The Display shows Standby Charger only (Standby).

    5. Measure the output voltage.

    6. Turn on the system, press the ON button for one second. After a few seconds the Display shows Normal operation load power ...%.

    7. Switch to bypass operation:

      1. Press the Hash key to enter the User stack.

      2. Use the Arrow keys to navigate to Bypass operation.

      3. Press the Clock/number 1 key to turn on bypass operation.

      4. Check that the white static switch EN LED on the mimic diagram is illuminated.

      5. Press the Back key or wait 30 seconds to exit the User stack.

    8. The Display shows Bypass operation.

    9. Turn the manual bypass switch to the Normal position.

  14. Switch to normal operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Star/number 0 key to turn off bypass operation and switch to normal operation.

    4. If applicable, check that the white static switch EA LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

  15. The Display shows Normal operation load power ...%.

  16. Reset any alarms by pressing the C key twice.

  17. Make sure the system operates correctly and no alarms are present.
