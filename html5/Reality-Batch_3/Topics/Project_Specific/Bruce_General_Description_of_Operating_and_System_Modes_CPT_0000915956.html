General Description of Operating and System Modes

Note: An example system is used to illustrate the operating modes. The function of the operating mode is similar in all applicable configurations, but the energized path will depend on the configuration.
The operating mode depends on the status of the system, the rectifier mains, the battery and the bypass mains. The operating mode depends on the status of the system and the DC input power. The operating mode depends on the status of the system, the DC input power and the bypass mains. The operating priority is:

  1. Normal Operation

  2. Battery Operation

  3. Bypass Operation (Static Bypass)

The operating mode of the system depends on the status of the rectifier mains and the charge mode of the system. The operation modes are:

  • Float Charge

  • Boost Charge

  • Battery Operation

Other modes that the system can be in during startup, charging, maintenance, troubleshooting and shutdown are:

  • Manual Bypass Switch in Bypass Position (Manual Bypass)

  • Manual Bypass Switch in Test Position

  • Standby Charger Only (Standby)

  • Charger Only

  • Initial Charge

  • Off

  • Independent Static Bypass Switch (ISBS) Operation

Additional modes in redundant systems:

  • Operation with One Unit Active (Redundant Systems)

  • Hot Standby (Redundant Systems)
