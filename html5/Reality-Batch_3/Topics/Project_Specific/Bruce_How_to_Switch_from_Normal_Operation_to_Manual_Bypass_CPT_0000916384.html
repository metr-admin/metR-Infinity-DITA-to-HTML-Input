How to Switch from Normal Operation to Manual Bypass

DANGER
HAZARD OF ELECTRIC SHOCK, EXPLOSION, OR ARC FLASH
  • Always wait 5 minutes after turning off the system and isolating all the power supplies (including the battery). The system contains DC capacitors with long discharge time.
  • Hazardous voltages are still present at some circuits, terminals and switches when the manual bypass switch is in bypass position.
CAUTION
UNINTENDED EQUIPMENT OPERATION
Never actuate the manual bypass switches when the system is in normal operation. Only switch to manual bypass from bypass operation.
Notice
loss of output voltage
When the manual bypass switch is in Bypass position, there is no backup from the battery it is not possible for the system to automatically switch back to inverter operation in case the bypass mains becomes unavailable.
Important: For redundant systems: Check that each step is perform on both units before proceeding to next step.
Note: For dual systems: Perform all the steps on one unit, then repeat the complete procedure for the other unit.
To switch from normal operation to manual bypass:

  1. Make sure that the system is in normal operation and that the Display shows Normal operation.

    Important: The bypass mains needs to be available and within the tolerance range. The static switch EN needs to be available.

  2. Switch to bypass operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Clock/number 1 key to turn on bypass operation.

    4. Check that the white static switch EN LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

  3. Make sure that the system is in bypass operation and that the Display shows bypass operation.

  4. Only for 3-position manual bypass switches: Turn the manual bypass switch to the Test position.

  5. Only for 2-position manual bypass switches: Turn the manual bypass switch to the Bypass position.

  6. Press both the ON button and the OFF button at the same time for two seconds. The Display now shows Standby Charger only (Standby).

  7. Only for 3-position manual bypass switches: Turn the manual bypass switch to the Bypass position to isolate the static switch EN from the bypass mains.

  8. Only for 2-position manual bypass switches: Open the bypass isolation (EN input) switch to isolate the static switch EN from the bypass mains.

The system is in standby charger only (standby) and the load is supplied by the bypass via the manual bypass switch.
