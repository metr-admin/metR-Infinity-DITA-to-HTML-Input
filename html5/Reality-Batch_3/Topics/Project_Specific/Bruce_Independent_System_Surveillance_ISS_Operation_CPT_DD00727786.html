Independent System Surveillance (ISS) Operation

The independent system surveillance (ISS) monitors the operation of the system to increase robustness. In case ISS is activated, the ISS will switch over the load to static bypass EN and the system will be locked in bypass operation, also called ISS operation.

Depending on what triggered the ISS to activate the display may or may not be operable. If the display is operable, the alarm “20: ISS active locked in bypass” will be visible to indicate that the system is in ISS operation.

From ISS operation it is not possible to switch to another operating mode because the correct function of the system cannot be guaranteed. To switch back to normal operation the system must be switched to manual bypass and then restarted. For more information see the chapter How to Switch from ISS or ISBS Operation to Normal Operation.
