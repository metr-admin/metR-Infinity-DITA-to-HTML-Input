Independent Static Bypass Switch (ISBS)

The independent static bypass switch (ISBS) is an option for systems with a bypass and a manual bypass switch. It provides a backup for the automatic transfer to static bypass in case the main controller cannot turn on the static switch EN.

The ISBS monitors the system output in case the main controller is inoperable and can switch to static bypass if needed. When the switch over to static bypass is triggered by the ISBS it is also called ISBS operation. In ISBS operation the system is locked in static bypass. The switch over is automatic without any interruption to the load.

To indicate the status of the ISBS two optional relays can be added (K701) and (K702). They are used to signal:

  • The overall status of the ISBS.

  • If the ISBS have triggered and switch over the system to ISBS operation.

The same information can also be seen on the board. The status and activation of the ISBS is also indicated by the LEDs STATUS_2 and STATUS_3 respectively. For more detailed information of the LEDs on the board see Troubleshoot the ISBS.
