Battery Breaker Black Start

With black start it is possible to start the system with only the batteries when the rectifier mains and bypass mains are unavailable.

How to Perform a Black Start

Important: The DC capacitors must be charged before connecting the load to the batteries.
Note: Read the complete procedure before performing the procedure.

  1. Charge the DC capacitors:

    1. Keep the charging/discharging switch it in position 2 (charge) until the warning LED turns off.

    2. With the charging/discharging switch still in position 2 (charge), close the battery breaker. If not possible, release the charging/discharging switch and quickly close the battery breaker.

    3. Release the charging/discharging switch.

  2. After about 10 seconds the Display shows Standby.

  3. Press the Lamp test button and check that all operating status and alarm LEDâ€™s are illuminated.

  4. Press the ON button for one second on the front panel to turn on the system. After a few seconds the Display will show Batt. operation time > ...min..

    Note: The time will only be shown if an advanced battery monitor is installed.

  5. Press the Number 9 key to check that the system output voltage is within the tolerance range.

  6. Check on the mimic diagram that the green inverter LED and the green static switch EN LED is illuminated.

  7. Close the output breaker.

  8. The system will now supply the load until the batteries are discharged.

  9. When the rectifier mains and/or bypass mains are available again close the respective mains breaker.

  10. Confirm any alarms by pressing the C key twice.

  11. Make sure the system operates correctly and no alarms are present.
