24V FAN(D1240)

20V supply for EN & fan ok

On: The 20V power supply used to power supply the static bypass switch EN and the fans is ok. Delivered from the bypass mains.

Off: Bypass mains not available or there is a hardware issue present.

Check bypass mains power supply and connection.

Contact Reality Service Center.

(E)

STATUS_1(D1242)

System firmware status indication

System firmware status indication. Shows what mode the system is in:On: System is in standby charger only (standby)

Off: Bypass mains not available or there is a firmware or wiring issue present.

Check bypass mains power supply and connection.

Contact Reality Service Center.

Slow flashing (0.5Hz): System is in normal operation.

Very fast flashing (4Hz): The system is in ISBS operation. There is an issue present with the main controller, or the main controller was manually reset during operation.

Transfer the system to manual bypass, for details see chapter How to Switch from ISS or ISBS Operation to Normal Operation.

Fast flashing (2Hz): System is in bypass operation (over the static switch), switched over by the main controller.

(F)

STATUS_2(D1243)

ISBS status ok

On: The ISBS system is ok and ready transfer to ISBS operation if needed.

Off: Bypass mains not available or there is a firmware or wiring issue present.

Check bypass mains power supply and connection.

Contact Reality Service Center.

(G)

STATUS_3(D1283)

ISBS operation active

Off: Indicated that the controller has not switched to ISBS.

On: The system is in ISBS operation. There is an issue present with the main controller, or the main controller was manually reset during operation.

Transfer the system to manual bypass, for details see chapter How to Switch from ISS or ISBS Operation to Normal Operation.
