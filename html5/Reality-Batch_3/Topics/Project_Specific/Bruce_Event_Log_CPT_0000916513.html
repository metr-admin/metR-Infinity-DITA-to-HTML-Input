Event Log

The event log (log stack) can store up to 1000 events and alarms. The latest entry is listed first.

Colon “:” indicates that the event or alarm was set, reset it is indicated with a bar “|”. Example:

  • SET: 67: Inverter volt out of tolerance

  • RESET: 67| Inverter volt out of tolerance

  • SET: 36: Battery current limit

  • RESET: 36| Battery current limit

How to Enter the Event Log

To enter the event log:

  1. Press both the Star/number 0 key and the Arrow up key at the same time to enter the event log.

  2. Use the Arrow keys to move up and down in the event log. The event log ends with the text No (further) alarm.

  3. Press the Back key or wait 30 seconds to exit the event log.

View Time Stamp for an Entry

In the event log it is possible to view the time stamp of an entry.To view the time stamp:

  1. Press both the Star/number 0 key and the Arrow up key at the same time to enter the event log.

  2. While in the event log navigate to an entry with the Arrow keys.

  3. Press the Clock/number 1 key. The Display will show the year, month, day / hours, minutes, seconds for that entry.

  4. Press the Back key or wait 30 seconds to exit the event log.

Reset the Event Log

Exit any menu.To reset the event log:

  1. Press both the Clock/number 1 key and the Number 2 key at the same time. The Display will show Master Password: .

  2. Use the numpad keys to enter the password for Master Mode. The Display will show Enter Password:.

  3. Use the numpad keys to enter the password for Event Log Reset. The Display will show Log stack cleared.

  4. Press the Back key or wait 30 seconds to exit the event log.
