Manual Bypass Switch in Bypass Position (Manual Bypass)

DANGER
HAZARD OF ELECTRIC SHOCK, EXPLOSION, OR ARC FLASH
  • Always wait 5 minutes after turning off the system and isolating all the power supplies (including the battery). The system contains DC capacitors with long discharge time.
  • Hazardous voltages are still present at some circuits, terminals and switches when the manual bypass switch is in bypass position.
Notice
loss of output voltage
When the manual bypass switch is in Bypass position, there is no backup from the battery it is not possible for the system to automatically switch back to inverter operation in case the bypass mains becomes unavailable.
Note
loss of output voltage
When the manual bypass switch is in Test or Bypass position, there is no backup from the battery it is not possible for the system to automatically switch back to inverter operation in case the bypass mains becomes unavailable.

With a manual bypass switch it is possible to manually switch and lock the system in Bypass position to isolate sections.

Note: This mode is also called manual bypass because the load is supplied over the manual bypass switch.
