Independent Static Bypass Switch (ISBS) Operation

The independent static bypass switch (ISBS) is an option, when triggered it is called ISBS operation. In ISBS operation the system is locked in static bypass by the ISBS solution. ISBS operation can be triggered if the main controller is inoperable and unable to switch over to bypass operation.

From ISBS operation it is not possible to switch to another operating mode because the correct function of the system cannot be guaranteed. To switch back to normal operation the system must be switched to manual bypass and then restarted. For more information see the chapter How to Switch from ISS or ISBS Operation to Normal Operation.
