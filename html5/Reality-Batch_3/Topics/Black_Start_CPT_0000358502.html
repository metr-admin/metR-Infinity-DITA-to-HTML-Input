Black Start

Black start is an optional feature. With black start it is possible to start the system with only the batteries when the rectifier mains and bypass mains are is unavailable.

There are two different types of black starts:

  • Black start installed on the inverter input switch

  • Black start installed on the battery breaker

Important: Check the status of the respective switch depending on the type.

How to Perform a Black Start

This procedure is for one unit. For a system with multiple units, repeat the procedure for the other units.

Important: The DC capacitors must be charged before connecting the load to the batteries.
Note: Read the complete procedure before performing the procedure.

  1. Charge the DC capacitors:

    1. Keep the charging/discharging switch in position 2 (charge) until the warning LED turns off.

    2. Only for black start on the inverter input switch: With the charging/discharging switch still in position 2 (charge), close the inverter input switch. If not possible, release the charging/discharging switch and quickly close the inverter input switch.

    3. Only for black start on the battery breaker: With the charging/discharging switch still in position 2 (charge), close the battery breaker. If not possible, release the charging/discharging switch and quickly close the battery breaker.

    4. Release the charging/discharging switch.

  2. After about 10 seconds the Display shows Standby.

  3. Press the Lamp test button and check that all operating status and alarm LEDâ€™s are illuminated.

  4. Press the ON button for one second on the front panel to turn on the system. After a few seconds the Display will show Batt. operation time > ...min..

    Note: The time will only be shown if an advanced battery monitor is installed.

  5. Press the Number 9 key to check that the system output voltage is within the tolerance range.

  6. Check on the mimic diagram that the green inverter LED and the green static switch EN LED is illuminated.

  7. Check on the mimic diagram that the green output voltage LED is illuminated.

  8. Close the output breaker.

  9. Close any outgoing feeders.

  10. The system will now supply the load until the batteries are discharged.

  11. When the rectifier mains and/or bypass mains are available again close the respective mains breaker.

  12. When the rectifier mains is available again close the rectifier mains breaker.

  13. Confirm any alarms by pressing the C key twice.

  14. Make sure the system operates correctly and no alarms are present.
