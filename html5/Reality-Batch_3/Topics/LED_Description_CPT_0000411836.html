LED Description

Part

Name

Status when working correctly

Description

(1)

Online

On (green)

Off: Profibus-DP bus gateway is offline.

Constant green: Profibus-DP bus gateway is online.

(2)

Offline

Off

Off: Profibus-DP bus gateway is online.

Constant red: Profibus-DP bus gateway is offline.

(3)

Not used

—

—

(4)

Fieldbus diag

Off

Off: Gateway initialization successful.

Flashing red slowly (1 Hz): Gateway configuration error. The input and/or output data length is invalid.

Flashing red (2 Hz): Gateway configuration error. User parameter data length and/or content is invalid.

Flashing red fast (4 Hz): Error detected with the ASIC in charge of Profibus-DP communications when resetting the gateway.

(5)

Modbus

On (green)

Off: No power.

Constant green: Modbus communications OK.

Flashing green: No Modbus communication.

Red: Loss of communication with at least one secondary Modbus device. Exception code coming from a command or a transaction.

(6)

Gateway

Flashing (green)

Off: No power.

Constant green: The gateway is currently being initialized and configured.

Flashing green: The gateway is in running order, configuration OK.

Flashing red and green: Configuration absent / not valid.

When the LED (5) is flashing red from communication loss, the LED will revert to a green state when communication is restored. When the LED (5) is flashing red because of incorrect values with the aperiodic read/write services, then the only way to clear the error is to reuse these aperiodic services with correct values.

If the LED (6) is flashing following a sequence beginning with one or more red flashes, note down the order of this sequence and save for troubleshooting. Powering off the gateway and turning it back on again will solve the problem in some cases.
