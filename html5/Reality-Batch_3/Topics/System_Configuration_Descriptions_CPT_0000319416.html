System Configuration Descriptions

A Reality PxW WxW SDC can be configured as a single, dual or redundant system.

  • Single systems: Consists of a single unit supplying the load.

  • Dual systems: Consists of two identical units working independently to supply separate loads.

  • Redundant systems: Consists of two units working together in parallel to supply a common load. Each unit can supply the full load if needed.
