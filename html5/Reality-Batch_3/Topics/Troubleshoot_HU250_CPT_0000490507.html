Troubleshooting the HU250 Gateway

Problem Check
LED(s) not illuminated
  • Check the status and connection of the power supply.
  • Test the LEDs by pressing the Indicator test button on the front panel to check if they illuminate.
  • Check status of the device, see Device Overview.
Canâ€™t connect with a PC to the HU250
  • Check that the correct cable (straight Ethernet cable (RJ45 connection)) is used.
  • Check that the cable is connected to the right LAN port (LAN right).
  • Check that the local mode is enabled on the HU250.
  • Check that the LAN right port is active, see Network Ports Settings.
No Modbus TCP/IP communication from the NMC No Modbus TCP/IP communication from the Modular controller
  • Check the status of the NMC card.
  • Check that the correct cable (straight Ethernet cable (RJ45 connection)) is used.
  • Check that the cable is connected to the right WAN port (ETH1).
  • Check that the ETH1 port is active, see Network Ports Settings.
  • Check the IP configuration for the WAN connection to the NMC, see WAN Settings.
No IEC communication No DNP communication
  • Check the status of the device, see Device Overview.
  • Check the status of the Modbus communication, see the customized LED Modbus status.
No Modbus RTU over RS485 communication from the Prime controller
  • Check that the correct cable (straight Ethernet cable (RJ45 connection)) is used.
  • Check that the ETH1 port is active, see Network Ports Settings.
