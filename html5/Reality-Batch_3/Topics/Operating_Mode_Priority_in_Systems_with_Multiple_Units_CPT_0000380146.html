Operation Mode Priority

In a system where the units work in parallel, the parallel communication shares the status and operation mode of each unit with the other units. The system will:

  • Try to operate both units in the highest priority operation mode.

  • Try to operate both units in the same operation mode. If this is not possible the unit that cannot operate in the higher operation mode will change to hot standby. For more information see the chapter Hot Standby (Redundant Systems).

The table shows different examples of how the operation mode priority functions in a redundant system.

Note: A dual system working in parallel with a closed couple switch will function differently since the redundancy is not ensured.
