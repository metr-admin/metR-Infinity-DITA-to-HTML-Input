How to Turn On the System

Prerequisites:

  • The system has been commissioned.

  • The batteries have been commissioned and are ready to be connected.

  • The rectifier mains is available and within tolerance.

  • The DC input is available and within tolerance.

  • The DC input is available and within tolerance.

  • The bypass mains is available and within tolerance.

Important: The modules will start to run as soon as the input supply is available. When the modules run the output(s) will be energized. If the system does not have any output breakers or output feeders, the output terminal will also be energized as soon as the modules run.
Note: Read the complete procedure before you turn on the system.
Note: If active, turn off the audible buzzer by pressing the Mute icon in the upper right corner of the Display. The Mute icon is only visible if the audible buzzer is active.
Note: If needed, reset any held alarms. Press the Double arrow icon and then press the Reset held alarms button.
  1. Check that all main breakers are open and that the load is disconnected.
  2. Check that all main breakers and module switches are open and that the load is disconnected.
  3. Check that the manual bypass switch is in static switch output (Position 0). If not, rotate the manual bypass switch to position 0.
  4. Close the bypass AC mains breaker.
  5. The Display starts and after a few moments and shows the Home screen.
    Note: If the Display has not started, check that the internal PSU switches are closed.
  6. Close the rectifier AC mains breaker(s).
  7. Close the converter DC input breaker(s).
  8. The Display starts and after a few moments and shows the Home screen.
    Note: If the Display has not started, check that the internal PSU switches are closed.
  9. If available, close all the rectifier module input switches.
  10. If available, close all the DC/DC converter module input switches.
  11. Check that all rectifier modules have started up. Check that the green LED is illuminated on all rectifier modules and that all red LEDs are off.
  12. Check that all converter modules have started up. Check that the green LED is illuminated on all converter modules and that all red LEDs are off.
  13. From the Home screen click on the Rectifier modules icon. Check that the rectifier output voltage is according to the float voltage on the Data sheet.
  14. From the Home screen click on the DC/DC converter modules icon. Check that the output voltage is according to the output settings in the Data sheet.
  15. Close the battery breaker(s).
  16. If available, close the rectifier DC output breaker. The DC output is energized.
  17. Close the rectifier DC output breaker. The output is energized.
  18. Close the converter DC output breaker. The output is energized.
  19. Close the inverter DC input breaker.
  20. If available, close the inverter DC input breaker.
  21. Check that all inverter modules have started up. Check that the green LED is illuminated on all inverter modules and that all red LEDs are off.
  22. Close the system AC output breaker. The AC output is energized.
  23. If available, close any output feeders.
  24. Press the Double arrow icon and check that no system alarms are active.
  25. Check that the system runs correctly and that no alarms are active.
The system is ready to supply the load(s). The system is ready to supply the load.
