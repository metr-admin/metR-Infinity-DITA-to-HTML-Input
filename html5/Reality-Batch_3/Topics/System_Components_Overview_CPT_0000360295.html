System Components Description

The image shows an example and a simplification of a system configuration with optional components intended as a reference for the description of the system components. See the Single Line Diagram for the configuration of your system.

Note: Optional parts are marked with a star.
The system components are divided into three sections:
  • (A) The connection field. Located in the bypass, distribution or incoming.
  • (B) The system part. Located in the system cabinet behind the main protection cover.
  • (C) The low voltage section. Located in the system cabinet in front of the main protection cover.
The sections are divided to make maintenance and service easier. The connection field and system part physically isolated1 from each other (different cabinets).

Components in the Connection Field

Designator Component Description
X090 Bypass mains terminals Connection terminal for the bypass mains input. The bypass mains input requires an external protective device. A fuse or molded case circuit breaker (MCCB) according to the technical specification. L, N, PE for single phase systems. L1, L2, L3 and PE (and N if there is no isolation bypass transformer) for three phase systems. For specifications always refer to the Drawings for your system.
Q090 OptionalBypass circuit breaker or Switch-disconnector Breaker for the bypass mains input. The bypass mains input requires an external protective device. A fuse or MCCB according to the technical specification.
T090 OptionalBypass transformer or stabilizer Galvanic isolation between bypass mains input and load. Used if the required output is different from the available input. Either a transformer or stabilizer can be used.
Q050 Optional3-position manual bypass switch Makes it possible to isolate the system for maintenance and test the system without interruption. Positions are Auto, Test and Bypass. For more information see Manual Bypass Switch.
Q050 Optional2-position manual bypass switch Makes it possible to isolate the system for maintenance. Positions are Auto and Bypass. For more information see Manual Bypass Switch.
Q028 OptionalSwitch-disconnector Always used with 2-position manual bypass switches. Needed to isolates the static switch EN when the 2-position manual bypass switch is in Bypass position. This functionality is integrated in 3-position manual bypass switches.
X001 Rectifier mains input terminals DC power input terminals Connection terminal for the rectifier mains DC power input. The rectifier mains DC power input requires an external protective device. A fuse or MCCB according to the technical specification. Always L1, L2, L3, and PE.
Q001 Q020 Rectifier input circuit breaker or switch-disconnector DC power input isolation Breaker for the rectifier mains input. The rectifier mains input requires an external protective device. A fuse or MCCB according to the technical specification. Provides isolation for the DC power input. No trip function.
X002 Battery terminals Connection terminal for the battery connection. The battery connection requires an external protective device. A fuse or MCCB according to the technical specification.
Q004 OptionalBattery circuit breaker Breaker for the battery connection. Can be a battery protection device. If Q004 is not selected, Q005 has to be selected. It is possible to have both options.The battery connection, protection and breakers can also be part of the customers scope.
Q005 OptionalBattery protection device External battery protection device for the battery connection, must be installed according to the technical specification. If Q005 is not selected, Q004 has to be selected and Q004 must be a battery protection device. It is possible to have both options.
X004 Load terminals Connection terminal for the load. L, N, PE for single phase systems. L1, L2, L3, N and PE for three phase systems.

Components in the System Part

Designator Component Description
T001 Rectifier input isolation transformer Galvanic isolation between mains input and rectifier. Transforms the mains input to the correct value for the rectifier.
A030 6-pulse thyristor rectifier-bridge Converts the AC voltage from the mains to DC voltage.
A031 OptionalAdditional 6-pulse thyristor rectifier-bridge To create a 12-pulse thyristor rectifier-bridge an additional 6-pulse thyristor rectifier-bridge (with a 30 degree phase shift) can be used. This reduces the repercussion of the rectifier back to the mains.
F026 OptionalFuse Power fuse at the output of the thyristor rectifier-bridge.
L001 Smoothing choke Filters the DC current together with CB02.
L005 Battery choke Reduces the AC ripple to the battery generated by the single phase inverter.
CB02 DC capacitor module Filters the DC current together with L001.
A032 Pulse width modulated (PWM) inverter Power module with insulated-gate bipolar transistor (IGBT). Converts the DC voltage from the rectifier back DC power supply to AC voltage.
L002 Smoothing choke Filters the AC output and removes distortions together with the CB03.
T002 Inverter power transformer Galvanic isolation between the load and the DC. Transforms the AC output to the correct value for the load.
CB03 AC capacitor module Filters the AC output and removes distortions together with the L002. Subdivided into several sections.
OptionalFuse Protects each section of CB03. If one section has a failure that section will be isolated.
F021 Inverter output fuse Protection at the inverter output.
A035 Static switch EN The static switch consists of antiparallel-switched power thyristors. It is used to automatically transfer the load to bypass without interruption. Can only be transferred if the bypass mains voltage is within tolerance. Transfer can occur when:
  • The system is heavily overload.
  • The inverter cannot keep the output voltage within tolerance.
  • The battery is discharged during battery operation.
  • DC input power is unavailable or out of tolerance.
  • The system has been switched to static bypass operation from the front panel.
F028 Fuse Protects the static switch EN A035.
A036 OptionalStatic switch EA The static switch EA is an option at the inverter output. It can make the switch over between operating modes more reliable. Especially in redundant systems it can make the switch back to normal operation easier for one system. With EA one unit can keep the inverter running in no-load operation, instead of switching it off. This additional mode is called either Standby+ or Hot standby depending on the redundant configuration. For more information see chapter Hot Standby (Redundant Systems).
A201 Internal power supply Supplied by the rectifier mains or the bypass mains. Part of the redundant internal power supply together with A202. One of the two supplies can supply the system alone. In case of one of the supplies stops suppling the system an alarm is generated on the front panel.
A202 Internal power supply Supplied from the internal DC bus. Part of the redundant internal power supply together with A201. One of the two supplies can supply the system alone. In case of one of the supplies stops suppling the system an alarm is generated on the front panel.
A055 Fan system Consist of:
  • Fan-transformer that transforms the voltage to the correct voltage to the fans.
  • Fans with internal hall sensor for monitoring the number of revolutions.
  • Fan monitor that monitors the number of revolutions of each fan. If the number of revolutions falls below the preset value an alarm is generated.
E00X
A070 Main controller board Controls the main parts (rectifier, inverter, static switch(es)) of the system and the alarm relays. Monitors the voltages, the currents and the alarms. Communicates with the front panel, the serial connections and between multiple units in redundant systems.
A071 Interface board Contains control fuses and distributes power to auxiliary circuits and different modules. Converts AC and DC voltages to lower standard values for the controller. Secures galvanic isolation for the controller.
A073 Front panel The human machine interface (HMI) of the system. Contains a display, operating status indications, alarm indications, operating buttons and numpad. For more information see Front Panel.
A703 OptionalISBS Controller The ISBS controller controls the static switch EN A035 based on the control signal from the controller board A070. It can independently switch the system to independent static bypass (ISBS) operation. The ISBS controller has an independent power supply and independent supervision of the output voltage. It is connected to the two relays K701 and K702.
A024 OptionalParallel operation board Used for communication between two units in a redundant system. The board is located in the system parts and the connections are located in the low-voltage section.
A068 Adapter board Connection interface to the controller board A070.
K701 Optional1 relay Indicates the status of the complete ISBS controller. If the relay is activated, the controller is fully operational and ready to react on the control signal from the controller board.
K702 Optional1 relay Indicates that the ISBS controller has switched the system to ISBS operation.

Components in the Low-Voltage Section

Designator Component Description
A025 External connection board An interface board with status relays and input terminals. For more information see the chapter External Connection Board.
Outputs:
  • Two common alarm relays with a programmable delay between 0-327 seconds (default value is 5 seconds)
  • A battery operation relay with a 30 seconds delay2
  • A bypass operation relay (static switch EN active)
Inputs:
  • Battery temperature sensor
  • Remote start/stop
  • Emergency power off (EPO)
  • Boost charge blocked
  • Battery breaker status
A076 OptionalCommunication interface Used for communication to the smart slot translator and then to the NMC. Contains one serial output (RS-232). For more information see the Network Monitoring System user guide.
