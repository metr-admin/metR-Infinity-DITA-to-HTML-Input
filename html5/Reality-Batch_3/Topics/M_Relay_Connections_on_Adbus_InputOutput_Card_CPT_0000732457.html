Relay Connections on ADBUS Input/Output Card

By default, 3 relays are available on the controller. If more relays are needed additional cards can be used.

Each input/output card have 4 potential-free change-over (CO) relays. Multiple cards can be used if needed.

Note: If multiple input/output cards are used, each card must have a unique address. The address is set using jumpers.
