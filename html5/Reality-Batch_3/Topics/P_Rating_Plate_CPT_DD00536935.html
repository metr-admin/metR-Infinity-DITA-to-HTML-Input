Rating Plate

  • (A) System type
  • (B) System type designation
  • (C) Mains input values
  • (D) Bypass input values
  • (E) Battery values
  • (F) Output values
  • (G) Protection degree, ambient temperature range and other environmental values.
  • (H) Place for certification/conformity mark (e.g. CE, EAC, UL) and any customer specific text.
  • (I) Unique identifier and manufacturing information:

  • Reality project number

  • System serial number

  • Transport unit number

  • Year of manufacturing

  • Country of origin / Country of manufacturing

System Type Designation

The top section (B) of the rating plate contains the system type designation. The system type designation contains information about the system configuration.

Position Description Options
(J) System type
  • PM – UPS with IGBT
(K) Manufacturing process
  • C – CTO, Configure to order
(L) Output phases
  • 1 – Single phase output
  • 3 – Three phase output
(M) Output rating (kVA)
  • 40 kVA
(N) Input voltage (V)
  • 400 V – Always phase to phase.
(O) DC voltage (V)
  • 220 V
(P) Output voltage (V)
  • 230 V – Phase to neutral for one phase, phase to phase for three phase.
(Q) Isolator at inverter output
  • C – Contactor
(R) Bypass configuration
  • W – Bypass static switch without bypass transformer
(S) Redundancy
  • S1 – Single system

Unique Identifier

The bottom section (I) on the rating plate contains the unique identifier. The unique identifier is built up of sections to make it possible to identify the cabinet.

Position Name Description
(T) Reality project number A unique number for each project.
(U) System serial number Indicates a specific system number in the project.
(V) Transport unit number The number indicates a transport unit. A system may consist of multiple transport units that can be separated for transportation.
