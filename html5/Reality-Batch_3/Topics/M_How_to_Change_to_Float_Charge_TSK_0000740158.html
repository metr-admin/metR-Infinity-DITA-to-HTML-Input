How to Change to Float Charge

  1. Make sure you are logged in as a user with at least Owner access.
    Note: If you are not logged in, a prompt to login will appear when you attempt to change mode.
  2. Press the Switch operation icon in the Status bar.
  3. Press the Float charge button.
    Note: The current active operation mode is indicated by the green icon.
  4. Confirm the change.
The rectifier modules have changed to float charge.
