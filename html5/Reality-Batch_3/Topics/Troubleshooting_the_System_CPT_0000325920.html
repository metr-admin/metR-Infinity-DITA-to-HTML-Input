There are multiple LED’s on the display that can be used for optional alarms. Here are information on how to troubleshoot some of the standard options:

Note: Not all alarms are possible for all system configurations.
Alarm Indicator Alarm Number and Text Possible Source
Rectifier output switch Q003 open [5 seconds]
  • 25: Mains breaker open
  • 140–145: Option x
The rectifier output switch Q003 is open. The battery and the load are disconnected from the rectifier. The load can only be supplied by the bypass mains or from the battery. The rectifier is turned off.
Inverter input switch Q020 open [5 seconds]
  • 65: Inverter breaker open
  • 140–145: Option x
The inverter input switch Q020 has automatically or manually been opened. The inverter is disconnected from the rectifier and battery. The load can only be supplied by the bypass mains. The alarm Inv. fuse blown will be activated. The inverter input switch Q020 has automatically or manually been opened. The inverter is disconnected from the DC input power. The load can only be supplied by the bypass mains. The alarm Inv. fuse blown will be activated.
Inverter output switch Q030 open[5 seconds] 140–145: Option x The inverter output switch Q030 is open. The load can only be supplied via the static switch EN or from the bypass mains via the manual bypass switch Q050 at position Bypass.
Distribution input Q100 open [5 seconds]
  • 71: Output breaker open
  • 140–145: Option x
  • The load output switch Q100 is open.
  • The system output is disconnected from the inverter.
  • The load is no longer supplied.
Distribution MCB tripped [5 seconds] 140–145: Option x
  • One or more of the outgoing feeders in the AC distribution has tripped.
  • Overload or short-circuit on one of the outgoing feeders.
Distribution MCB open [5 seconds] 140–145: Option x
  • One or more of the outgoing feeders in the AC distribution are open.
  • Overload or short-circuit on one of the outgoing feeders.
Distribution AC earth fault [5 seconds] 140–145: Option x
  • Earth fault on the AC output.
  • The insulation resistance to earth is below set point.
