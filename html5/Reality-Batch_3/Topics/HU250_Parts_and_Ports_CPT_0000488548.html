HU250 Parts and Ports

Description of the available parts and ports on the HU250.

Note: Not all parts and ports are available or used in the Reality solution.
Part Name Description
(A) Communication ports Slots for modem boxes.
(B) Ethernet ports
  • ETH1: Can be used for WAN communication. 2
  • ETH2: Connected to the NMC Reality Modular system.
(C) USB port Port no longer existing.
(D) Front panel For details see chapter HU250 Front Panel.
(E) Mini-USB port Port no longer existing.
(F) Dual Ethernet LAN (daisy-chain) port
  • LAN left: Dedicated to internal communication between T300 modules or for connection to a PC.3
  • LAN right: Used to connect a PC for local access to the HU250.
(G) Serial RS485 Modbus port Used for Easergy communication power supply or for third-party Modbus IEDs.4
(H) Temperature sensor input Input for a PT100 3 temperature sensor.5
(I) Digital output 2 dry relay contacts for multi-purpose control.6
(J) Digital inputs
  • 6 wet digital inputs compliant with IEC 611131-2 type 3 for multi-purpose monitoring 7
  • 2 digitals inputs dedicated for external Local / Remote button (internal or external L/R configurable) 8
(K) External indicator output 2 dedicated digital outputs allow connecting external Easergy light indicator for signaling the fault current outside the substation (5V-100 mA).9
(L) Power supply Daisy chain power supply connectors. Reality solution use 24Vdc ± 10%.10
