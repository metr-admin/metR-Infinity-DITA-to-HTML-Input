Annual Check

DANGER
HAZARD OF ELECTRIC SHOCK, EXPLOSION, OR ARC FLASH
Before performing any maintenance inside the system:
  • Read the safety information
  • Use appropriate personal protective equipment (PPE)
  • Isolate all the power supplies (including the battery) from the system and use lockout-tagout (LOTO)
  • Wait 5 minutes after isolating the system to let the DC capacitors discharge
  • Check that any removable (NC) earth link is closed
  • Check that the parts, capacitors, terminal and protective earth are volt free with a properly rated voltage sensing device
  • Check that protective covers are installed on any energized parts
Note
loss of output voltage
If the system is turned off without the manual bypass in Test or Bypass the output voltage is lost.
When the manual bypass switch is in Test or Bypass position, there is no backup from the battery it is not possible for the system to automatically switch back to inverter operation in case the bypass mains becomes unavailable.
Always coordinate maintenance and turning off or on the system with the users of the consumers. Make sure that users of the connected consumers are informed before proceeding.

Make the following checks annually:

  1. Perform both the Monthly Check and Semi-Annual Check.

  2. Isolate the system/unit with the manual bypass switch or turn it off and turn off the system/unit, see the Operating Manual for more information.

  3. If the system is dusty, clean the system from dust with a vacuum cleaner.
    Note: Do not use any liquid to clean!

  4. Perform the recommended maintenance for batteries supplied from Reality, see chapter Annual Battery Check. For batteries not supplied by Reality follow the maintenance recommendations by the original equipment manufacturer (OEM).

  5. Only for systems with bypass mains voltage stabilizer T090: Check if the bypass stabilizer needs to be cleaned, lubricated or if any parts are worn. For more information see the section Check the bypass stabilizer T090.

  6. If applicable, check and follow any annual maintenance recommendations by other original equipment manufacturer (OEM).

  7. Re-install any removed protection covers.

  8. Turn on the system or unit, see the Operating Manual for more information.

  9. Perform a functional check of the system to verify that the system switches correctly between operating modes (according to operation mode priority). See chapter Function Check of a Unit.

  10. Only for systems with an ISBS solution: Perform a function check of the ISBS solution to check that the ISBS takes over if the controller board is inoperable. See chapter Function Check of the ISBS Solution.
