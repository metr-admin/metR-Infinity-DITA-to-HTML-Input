Logs

From the web interface it is possible to view and download logs. The following logs are defined in the Reality solution: Event logs (Event and Measurements), System log (Device Name) and IEC 61850 log.

From the web interface it is possible to view and download logs. The following logs are defined in the Reality solution: Event logs (Event and Measurements), System log (Device Name).

Events log, Event

To access the event log Event navigate to Diagnostics > Events log, then select the Event tab. This log record changes in the monitored parameters. The IEC tag is recorded together with a time stamp and information about the change.

To access the event log Event navigate to Diagnostics > Events log, then select the Event tab. This log record changes in the monitored parameters. The DNP3 data format is recorded together with a time stamp and information about the change.

Events log, Measurements

To access the event log Measurements navigate to Diagnostics > Events log then select the Measurements tab. This log records the duration and status of when the monitoring of the parameters starts and stops.

System log

To access the system log navigate to Diagnostics > System log and select the tab for with the device name (default is HU01). This log contains additional system data for operational and maintenance purposes regarding the status of the device There are three categories for the entries:

  • INFO: Normal information on actions executed by the device

  • WARNING: Possible device anomalies that require user attention

  • ERROR: Potential issue detected, could impact operation of the device

IEC 61850 log

To access the IEC 61850 log navigate to Diagnostics > IEC 61850 log. This log contains information about the IEC 61850 communication for troubleshooting.

How to Download Logs

To download a log first navigate to the log. Then click on Export CSV located on the right side just above the table. This will download the log in the browser window.
