HU250 Gateway and Solution Concept

The HU250 communication gateway is a part of the modular platform Easergy T300.

Features of the flexible HU250 gateway:

  • Different local communication options

  • Multiple connection options to control centers or other IT applications

    • Standard and security-focused protocols: IEC 101/104, DNP3, IEC 61850 (Edition 2), Modbus

    • Open peer-to-peer communication to support self-healing application

  • Built-in webserver for commissioning and maintenance with local and remote access, compatible with PC, tablet and smartphone devices
