System Language

Reality systems have two languages installed. To change a language:

  1. Press the Hash key to enter the User stack.

  2. Use the Arrow keys to navigate to the current language.

  3. For the second language: Press the Clock/number 1 key to activate the second language.

    For the first language: Press the Star/number 0 key to activate the first language.

  4. Press the Back key or wait 30 seconds to exit the User stack.
