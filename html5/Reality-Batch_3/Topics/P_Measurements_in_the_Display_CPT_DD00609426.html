Measurements in the Display

There are multiple different measurements that can be viewed in the display, and some can be accessed in multiple ways.

Note: The screenshots are only samples.

  • Site measured values: Quick access to an overview of general measurements.

  • Rectifier (Unit state): Detailed measurements about the rectifier.

  • Battery (Unit state): Detailed measurements about the battery.

  • Inverter (Unit state): Detailed measurements about the inverter.

  • Bypass (Unit state): Detailed measurements about the bypass (from the static switch).

  • Power supply system (Unit state): Overview of measurements from the system.

  • Protocol (Measured values): Configured list of measurements from the system.

Site Measured Values

To view the Site measured values click on Measurement icon in the Status bar at the bottom of the screen.

Rectifier (Unit State)

To view the Measured values for the rectifier it is possible to either:

  • Click on the Rectifier icon on the home screen.

  • Navigate to Unit state > Rectifier.

Battery (Unit State)

To view the Measured values for the battery it is possible to either:

  • Click on the Battery icon on the home screen.

  • Navigate to Unit state > Battery.

Inverter (Unit State)

To view the Measured values for the inverter it is possible to either:

  • Click on the Inverter icon on the home screen.

  • Navigate to Unit state > Inverter.

Bypass (Unit State)

To view the Measured values for the bypass, measured from the static switch, it is possible to either:

  • Click on the Static switch icon on the home screen.

  • Navigate to Unit state > Bypass.

Power Supply System (Unit State)

To view the Measured values for the unit power supply status, navigate to Unit state > Power supply system.

Protocol (Measured Values)

To view the configured list of Measured values, navigate to Protocol > Measured values.
