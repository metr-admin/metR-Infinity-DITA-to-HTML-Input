WAN Settings

  1. In the HU250 interface navigate to Maintenance > Advanced > IP configuration.

  2. Select Device > WAN.

  3. Here you can view and change the IP settings for the WAN ports (ETH1 and ETH2).

  4. Click the Save button to apply any changes.

The default settings for WAN is fixed address with:
  • IP: 159.215.153.100 10.10.10.11
  • Subnet Mask: 255.255.255.0
Important: The default NMC Reality Modular IP address is 159.215.153.93 10.10.10.10. If the default IP is not used, please check the IP settings and specify a suitable WAN address accordingly.
