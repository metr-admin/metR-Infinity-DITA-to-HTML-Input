Solution Description

This user guide provides information about the Modbus RTU to Profibus-DP converter solutions for PEW and SDC distributions A/B in the project 1190128. The solutions are identical for the A and B distributions.

In the PEW and SDC solutions the LUFP7 gateway allows a primary device on the Profibus-DP bus to communicate with secondary devices on a Modbus RTU network.

An Ethernet patch panel is used between the gateway and the secondary devices, making it possible for the gateway to connect to the secondary devices.

The secondary Modbus devices consists of insulation monitoring devices (IM400) and the insulation fault locators (IFL12MC). The secondary devices send alarms, logs and monitored values to the Modbus RTU (RS485) network. A set of pre-configured signals are converted in the LUFP7 gateway to the primary device on the Profibus-DP bus.

The PEW and SDC solutions are project specific and optimized for the two systems.
