Advanced Battery Monitor (ABM)

The advanced battery monitor (ABM) is an option that uses battery monitor data from the battery solution used by the system. The battery monitor data includes parameters by the battery original equipment manufacturer (OEM).

The ABM uses the battery monitor data to:

  • Calculate an estimated value of the remaining backup time in battery operation.

  • Monitor the condition of the battery during battery tests and battery operation1.

With a battery temperature sensor installed, the ABM calculation will compensate for the current battery temperature.

Note: The advanced battery monitor (ABM) cannot be used in systems with a common battery.
