How to Isolate and Reconnect One Unit (Redundant System)

DANGER
HAZARD OF ELECTRIC SHOCK, EXPLOSION, OR ARC FLASH
  • Always wait 5 minutes after turning off the system and isolating all the power supplies (including the battery). The system contains DC capacitors with long discharge time.
  • Hazardous voltages are still present at some circuits, terminals and switches even when the system is turned off.
  • Only one unit is isolated. The other unit and any bypass common battery or distribution cabinet are still energized.
CAUTION
High current flow
Always charge the DC capacitors before closing the inverter input switch.
CAUTION
High current flow
Only connect the battery bank when the rectifier is in operation. The DC capacitors needs to be charged by the rectifier output before the battery is connected.
Note
loss of Redundancy
When the units in a redundant system are separated the redundancy of the output is lost.
Always coordinate maintenance and turning off or on the system with the users of the consumers. Make sure that users of the connected consumers are informed before proceeding.
Note: The steps are only performed on the unit that should be isolated/restarted.
Important: This only isolates the unit cabinet. The other unit and any bypass or distribution cabinet are still energized!

Isolate One Unit

To isolate one unit in a redundant system:

  1. Check that the complete system operates correctly in normal operation float charge and that no alarm is active.

  2. Only for redundant systems with individual batteries: Open the battery breaker to disconnect the battery bank.

  3. Press both the ON button and the OFF button at the same time for two seconds. The Display shows Standby.

  4. Open the battery breaker to disconnect the battery bank.

  5. Open the rectifier mains breaker to turn off the power supply to the rectifier.

  6. Open the inverter input switch to disconnect the DC power supply.

  7. Discharge the DC capacitor by keeping the charging/discharge switch in position 1 (discharge) until the warning LED turns off.

  8. Open the bypass isolation switch to isolate the unit from the bypass power supply.

  9. Open the output isolation switch rectifier output switch to isolate the unit from the system output.

The unit is isolated from the power supplies and the other unit.

Reconnect One Isolated Unit

To reconnect one isolated unit in a redundant system:
Important: The DC capacitors needs to be charged before the inverter input switch is closed.

  1. Close the bypass isolation switch to reconnect the unit to the bypass power supply.

  2. Close the rectifier mains breaker to turn on the power supply to the rectifier.

  3. After about 10 seconds the Display shows Standby.

    Note: If the unit is programmed for autostart and the input is within the tolerance range the unit will start automatically after 60 seconds.

  4. Make sure the external source of the DC input power is available.

  5. Charge the DC capacitors:

    1. Keep the charging/discharging switch it in position 2 (charge) until the warning LED turns off.

    2. With the charging/discharging switch still in position 2 (charge), close the inverter input switch. Alternatively, release the charging/discharging switch and quickly close the inverter input switch.

    3. Release the charging/discharging switch.

    Note: If the unit is programmed for autostart and the input is within the tolerance range the unit will start automatically after 60 seconds.

  6. Press the ON button for one second on the front panel. After a few seconds the Display will show Normal operation load power ...% Float charge load ...%.

  7. Press the Number 8 key to check the DC voltage.

  8. Only for redundant systems with individual batteries: If the DC voltage is within tolerance, close the battery breaker.

  9. Check that:

    1. The units are all running in normal operation (individually).

    2. The bypass mains and static switch EN are available for all units and within tolerance (check status on the mimic diagram).

    3. No alarms are pending.

  10. Switch over unit1 to static bypass operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Clock/number 1 key to turn on bypass operation.

    4. Check that the green static switch EN LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

  11. Follow the same procedure and switch over unit2 to bypass operation.

  12. Check that the Display on both units show Bypass operation.

  13. Check that the external load connection is closed.

  14. Close the output isolation switch to reconnect the unit to the system output.

  15. Switch over unit1 to normal operation:

    1. Press the Hash key to enter the User stack.

    2. Use the Arrow keys to navigate to Bypass operation.

    3. Press the Star/number 0 key to turn off bypass operation and switch to normal operation.

    4. If applicable, check that the green static switch EA LED on the mimic diagram is illuminated.

    5. Press the Back key or wait 30 seconds to exit the User stack.

    Note: The load will be supplied by the bypass until all units in the system have been switched back to normal operation.

  16. Follow the same procedure and switch over unit2 to normal operation.

  17. Close the rectifier output switch to reconnect the unit to the system output.

  18. Check that the Display on both units show Normal operation load power ...% Float charge load ...%.

  19. Confirm any alarms by pressing the C key twice.

  20. Make sure the system operates correctly and no alarms are present.

The unit is reconnected to the other unit. The system is working in parallel in normal operation float charge with redundancy at the output.
