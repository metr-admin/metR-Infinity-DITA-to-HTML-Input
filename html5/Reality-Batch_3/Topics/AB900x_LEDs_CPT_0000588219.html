AB9001 LEDs AB9006 LEDs

Position Name Description
(A) Gateway status (GW)
  • Off: Power off
  • Alternating red/green: Missing configuration
  • Flashing green: Standby
  • Green: Operating
  • Flashing red: Invalid configuration
  • Red: Inoperable
(B) Not used –
(C) PROFIBUS DP-V1 Network Status (OP) EtherNet/IP Network status (NS)
  • Off: Power off
  • Flashing green: Online, data transfer
  • Green: Online
  • Flashing red 1 time: Parametrization error
  • Flashing red 2 times: Configuration error
  • Off: No IP address
  • Flashing green: Online, no connection
  • Green: Online, connection established
  • Flashing red: Time-out
  • Red: Interrupted connection or inoperable
(D) Module Status (ST) Module status (MS)
  • Off: Power off
  • Green: Operating
  • Red: Inoperable
  • Off: Power off
  • Flashing green: Not configured, or scanner in standby
  • Green: Scanner operating
  • Flashing red: Interrupted
  • Red: Inoperable
(E) SD card status (SD)
  • Green: Accessing SD card
  • Flashing red: Inoperable
(F) Modbus-TCP status (MTCP)
  • Off: No Modbus-TCP network communication
  • Green: Communicating with Modbus-TCP network
  • Flashing red: Interrupted connection or time-out
  • Red: Inoperable
(G) Ethernet link 1 (LA1)
  • Off: No link
  • Flashing green: Receiving/transmitting Ethernet packets at 10/100 Mbit
  • Off: No link
  • Flashing green: Receiving/transmitting Ethernet packets at 100 Mbit
  • Flashing yellow: Receiving/transmitting Ethernet packets at 10 Mbit
(H) Ethernet link 2 (LA2)
