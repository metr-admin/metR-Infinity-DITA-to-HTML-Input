Preparations with Power Off

Before starting the commissioning process the following must be checked:

Important: Make sure that you are familiar with the actual system, have studied the Single Line Diagram and know the position of all the switches, breakers and feeders.

  1. Check with a properly rated measuring device that all sections are de-energized.

  2. Check that all protective earth connections and interconnections (between transport units) are secured and undamaged.

  3. Check that all connectors are firmly fixed.

  4. Check that all printed circuit boards (PCBs) are correctly mounted and that all plugs are firmly plugged in.

  5. Check that all interconnections and junctions are according to specifications, see the Connection Diagram.

  6. Check that all external fuses have the correct size, see the Technical Data.

  7. Remove all bags with moisture absorbing material left in the cabinets.

  8. Check that the interior of the system is completely dry.

  9. Check that all switches and breakers have the correct status according to the following table.
