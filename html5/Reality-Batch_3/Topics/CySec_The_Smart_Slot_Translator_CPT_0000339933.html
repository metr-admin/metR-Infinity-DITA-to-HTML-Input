The Smart Slot Translator

The smart slot translator converts the RS-232 signals from the Silcon protocol to the Smart Link protocol. The connection from the communication interface is to a RS-232 port with a DE-9 connector (male), using the Silcon protocol. The two-way connections to the NMC and the options slots are from RS-232 ports with 10 pin IDC-connectors, using the Smart Link protocol.

The smart slot translator has a serial monitoring port, a RS-232 port with a DE-9 connector (female). With a serial cable it is possible to access the Command Line Interface (CLI) from this port.
