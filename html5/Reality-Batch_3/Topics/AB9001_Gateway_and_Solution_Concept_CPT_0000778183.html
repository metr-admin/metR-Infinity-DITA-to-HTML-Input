AB9001 Gateway and Solution Concept

The AB9001 is a link between two industrial networks. On the Modbus-TCP network, it function as client and as server on the Profibus DP network.

  • Fast copying of cyclic I/O data between a Modbus TCP and an Profibus DP network (5 ms)

  • Max I/O data capacity of 256 bytes in each direction

  • Robust stand-alone housing for use in harsh industrial environments
