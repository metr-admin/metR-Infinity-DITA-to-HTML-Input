PEW Solutions Overview

The PEW solutions needed a larger number of secondary devices than the SDC. To connect all devices three gateways had to be used, so the solutions contains three setups, one for each LUFP7 gateway (A060, A062 and A064).

The first PEW gateway setup (A060) contains one IM400 insulated monitoring device and three IFL12MC insulation fault locators.

The second and third PEW gateway setups (A062, A064) are identical and only contains three IFL12MC insulation fault locators (no IM400).
