Change Address on a Secondary Modbus Device

The procedure for changing the Modbus address is the same for IM400 and IFL12MC.
  1. Navigate to Settings > Modbus > Address in the Display (A).
    When the Address menu is entered the Contextual menu buttons (B) will change to:

    • Plus: Press to modify the selected digit.

    • Arrow: Change the selected digit.

    • Check: Confirm the new value of the digit.

  2. Use the Contextual menu buttons (B) to change the Modbus address to the correct default value according to the table in the chapter Default Modbus RTU Addresses
