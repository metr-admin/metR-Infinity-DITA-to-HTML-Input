Equalization Charge

Notice
High output voltage AND REDUCED CURRENT
Never enable equalization charge mode with a load connected to the DC output.
Never enable equalization charge mode with the inverter modules in operation.

Equalization charge mode uses a high output voltage to overcharge the battery. Equalization charge is mainly used to condition the battery during its commissioning.

This mode is a temporary mode and is only used for a limited period.

Important: Never have the inverter modules in operation during equalization charge. Make sure the manual bypass switch is turned to the bypass output position, see How to Change to Bypass Output from Static Switch Output.
Note: Equalization charge can only be activated manually.
Note: It is possible to inhibit equalization charge. If equalization charge is inhibited, the rectifier modules cannot manually change to equalization charge. Equalization charge can be inhibited permanently (not available for the system) or by an external signal.
