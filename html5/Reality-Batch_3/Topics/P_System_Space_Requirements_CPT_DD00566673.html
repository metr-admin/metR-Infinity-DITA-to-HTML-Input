System Space Requirements

The system has minimum clearances requirements from the top (A) for airflow and to be able to transport the system.

It is possible to place systems side by side and with the back to the wall. It is recommended to not have any direct contact.

Important: Always make sure that there is enough space for the warm air to dissipate.
Position Minimum Clearances
(A)1 300 mm (12 in)
1 If the system has a rain roof, the minimum clearances is measured from the rain roof and not the cabinet roof.
