Static Bypass Operation and Forced Bypass Operation

In static bypass operation or forced bypass operation the Unit state is Bypass operation.

Note: The Unit state is indicated in the bottom left corner of the Display.

In bypass operation the output is supplied by bypass mains over the static switch.

If the system is automatically switching to bypass operation it is called static bypass operation. The system can automatically switch over to static bypass, for example during start up or in case the inverter output is outside of tolerance.

It is also possible to manual enable bypass operation from the display, then it is called forced bypass operation.

Note: If the rectifier is in operation, the batteries can still be charged.
