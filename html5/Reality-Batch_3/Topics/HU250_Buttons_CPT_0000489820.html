Front Panel Buttons

On the HU250 front panel there are buttons that can be manually pressed to perform actions.

Note: Not all buttons are used in the Reality solution. These buttons will not cause any action, since the function or module is not available.
Position Name Description
(A) Remote/local switch Press to switch between local and remote access mode.
(B) Indicator test Press to test that all LED indications illuminates.
(C) Automatic control change-of-state button (Not relevant for Reality solution).
(D) Change-of-state enabling (Not relevant for Reality solution).
(E) Reset (Not relevant for Reality solution).
