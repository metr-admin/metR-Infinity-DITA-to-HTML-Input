LAN Settings

  1. In the HU250 interface navigate to Maintenance > Advanced > IP configuration.

  2. Select Device > LAN.

  3. Here you can view and change the IP settings for the LAN ports (LAN left and LAN right).

  4. Click the Save button to apply any changes.

The default settings for LAN is DHCP server with:
  • IP: 192.168.0.254
  • Subnet Mask: 255.255.255.0
  • DHCP Start IP: 192.168.0.40
  • DHCP End IP: 192.168.0.60
Note: If you change the default IP address, remember to use this IP when connecting locally following the description in Local Ethernet Connection to the HU250.
