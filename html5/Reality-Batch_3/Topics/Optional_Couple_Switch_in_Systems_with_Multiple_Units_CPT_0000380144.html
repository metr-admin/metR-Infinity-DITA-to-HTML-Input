Optional Output Couple Switch

Note: This option is only possible in redundant systems with individual batteries.
A redundant system has two output isolation switches rectifier output switches. This makes it possible for one unit to supply the full load or to isolate one unit. With an optional couple switch between it is possible to separate the units to function as single systems and supply individual loads.
Note: The system output is not redundant if one or more of the switches (couple or output isolation) are open.
In a dual system each unit has an output breaker. With an optional couple switch between the output breakers it is possible switch over both loads to a single unit.
Note: In a dual system the units can work in parallel, but the redundancy of the load is not ensured.

For operation instructions see chapter How to Operate the Output Couple Switch.
