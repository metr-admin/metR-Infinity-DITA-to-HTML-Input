Starting the System

CAUTION
High current flow
Always charge the DC capacitors before closing the inverter input switch.
CAUTION
High current flow
Only connect the battery bank when the rectifier is in operation. The DC capacitors needs to be charged by the rectifier output before the battery is connected.
Important: For redundant systems: Check that each step is perform on both units before proceeding to next step.
Note: For dual systems: Perform all the steps on one unit, then repeat the complete procedure for the other unit.
Important: The DC capacitors needs to be charged before the inverter input switch is closed.

  1. Check that the system is turned off.

  2. Only for systems with a manual bypass switch: Make sure the manual bypass switch is in Auto position. If not turn the manual bypass switch(es) to the Auto position.

  3. Close the rectifier mains breaker to turn on the power supply to the rectifier.

  4. Make sure the external source of the DC input power is available.

  5. Charge the DC capacitors:

    1. Keep the charging/discharging switch it in position 2 (charge) until the warning LED turns off.

    2. With the charging/discharging switch still in position 2 (charge), close the inverter input switch. If not possible, release the charging/discharging switch and quickly close the inverter input switch.

    3. Release the charging/discharging switch.

  6. After about 10 seconds the Display shows Standby.

    Note: If the unit is programmed for autostart and the input is within the tolerance range the unit will start automatically after 60 seconds.

  7. Press the ON button for one second, if the unit did not start from the autostart. After a few seconds or if any key is pressed the Display shows Normal operation load power ...% Float charge load ...%.

  8. Press the Number 8 key to check the DC voltage.

  9. If the DC voltage is within tolerance, close the battery breaker and battery protection breaker if applicable.

  10. Press the Number 9 key to check that the inverter output voltage is within the tolerance range.

  11. Only for systems with a bypass: Close the bypass mains breaker to connect the bypass.

  12. Confirm any alarms by pressing the C key twice.

  13. Make sure the system operates correctly and no alarms are present.
