Analog Outputs

As an option, analog outputs can be programmed. When selected the function comes pre-programmed, with the selected mode and a defined range.

There are two analog outputs available on the monitoring controller. The outputs have three different programmable modes:

  • 0 – 10 mA

  • 0 – 20 mA

  • 4 – 20 mA

Connection Connection Name Pinout Description
XG141 Analog output 1 1 = +
2 = –
XG142 Analog output 2 1 = +
2 = –
