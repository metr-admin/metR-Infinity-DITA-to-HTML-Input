AB9006 Gateway and Solution Concept

The AB9006 is a link between two industrial networks. On the Modbus-TCP network, it function as client and as server on the EtherNet/IP network.

  • Fast copying of cyclic I/O data between a Modbus TCP and an EtherNet/IP network (5 ms)

  • Max I/O data capacity of 256 bytes in each direction

  • Robust stand-alone housing for use in harsh industrial environments
