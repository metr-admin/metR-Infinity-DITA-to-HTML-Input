Semi-Annual System Check

Every 6 months it is recommended to do a visual check of the system environment, check readings from the display and condition inside the door (without removing any protective covers), do a LED test and perform battery maintenance.
Important: If the battery is installed in the system the protective covers in front of the battery cells need to be removed for maintenance.
