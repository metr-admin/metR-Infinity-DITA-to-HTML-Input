Additional Functionality in Systems That Work in Parallel

Units in a redundant system or a dual system that work in parallel to supply one load must communicate with each other. The behavior of a redundant system is complex and will depend on multiple factors.

As an option in redundant or dual systems, an output couple switch can be used to temporarily separate or share the load between the units.

As an option in redundant systems with individual batteries, a battery couple switch can be used to temporarily connect both individual batteries to one unit.
