Connections on the Communication Interface External Connection Board

The communication interface makes it possible to connect the controller board to the smart slot translator. The communication interface has two RS-232 ports with DB-25 connector J1 and J2 and one RS-232 port with a 34 pin IDC-connector J5 using the Silcon protocol.

The Silcon protocol uses a Block Check Character (BCC) to acknowledge requests and to make sure the status messages does not contain any errors. Any other communication is ignored by the controller board without leading to any reaction from the system.

The connection between the communication interface and the smart slot translator is an extension of the one-way communication from the controller board. No conversion of the signal is done on the communication interface.

Pinout of J1 and J2:

  • Pin 2: RS-232 Tx

  • Pin 3: RS-232 Rx

  • Pin 9: 10 V (DC)

  • Pin 7: GND

Other pins are not connected.

The external connection board makes it possible to connect the controller board to the smart slot translator. On the external connection board two ports are used, one RS-232 port with DB-25 connector X2 and one RS-232 port with a 34 pin IDC-connector X5, and uses the Silcon2 protocol.

The Silcon protocol uses a Block Check Character (BCC) to acknowledge requests and to make sure the status messages does not contain any errors. Any other communication is ignored by the controller board without leading to any reaction from the system.

The connection between the communication interface and the smart slot translator is an extension of the one-way communication from the controller board. No conversion of the signal is done on the communication interface.

The communication interface makes it possible to connect the controller box to the network management cards. The communication interface has two edge connector slots (J2501) and (J2502) where the NMC edge connector (GF1) can be inserted.

The input port (J2500) connects to the port (J224) on the controller box (A102) with a 8P8C. Both the incoming and outgoing communication is according to the RS-232 standard using the UPS-Link protocol.
