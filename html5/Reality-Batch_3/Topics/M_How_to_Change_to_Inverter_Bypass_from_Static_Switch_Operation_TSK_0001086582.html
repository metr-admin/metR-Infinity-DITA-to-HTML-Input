How to Change to Inverter Output from Static Switch Output

Manual bypass switch (MBS) positions:
  • 0 = Static switch (SSW) output
  • 1 = Bypass output
  • 2 = Inverter output
Note: If active, turn off the audible buzzer by pressing the Mute icon in the upper right corner of the Display. The Mute icon is only visible if the audible buzzer is active.
  1. On the display check that the Inverter Icon does not have a warning icon and that no alarm LED is active.
  2. On the Display check that the power flow indicates static switch output.
  3. Open the door and rotate the manual bypass switch to the static switch output position (Position 2).
  4. Close the door.
  5. On the Display check that the power flow indicates inverter output.
The system will be in inverter output mode until the MBS position is changed.
Note: If the inverter output becomes unavailable during this time the system output is lost. Inverter output mode should only be used for a short period of time, for example during maintenance.
