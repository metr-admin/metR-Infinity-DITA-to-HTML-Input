How to Change to Bypass Output from Static Switch Output

Manual bypass switch (MBS) positions:
  • 0 = Static switch (SSW) output
  • 1 = Bypass output
  • 2 = Inverter output
Note: If active, turn off the audible buzzer by pressing the Mute icon in the upper right corner of the Display. The Mute icon is only visible if the audible buzzer is active.
  1. On the Display check that the Static switch icon does not have a warning icon and that no alarm LED is active.
  2. On the Display check that the power flow indicates static switch output.
  3. Open the door and rotate the manual bypass switch to the bypass output position (Position 1).
    The load is now supplied by the bypass.
  4. Close the door.
  5. On the Display check that the power flow indicates bypass output.
The system will be in bypass output mode until the MBS position is changed.
Note: If the bypass becomes unavailable during this time the system output is lost. Bypass output mode should only be used for a short period of time, for example during maintenance.
