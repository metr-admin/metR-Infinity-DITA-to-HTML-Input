Voltage Limitation Unit (VLU)

The voltage limitation unit (VLU) helps to limit the rectifier DC output voltage in case of a fast increase in the voltage variation on the rectifier AC mains input. The VLU directly communicates with the main controller and with the thyristor firing card.

For the VLU there are multiple conditions and steps to limit the DC voltage:

  1. VLU activation setting. Defines at what voltage level the VLU will activate.

  2. Control pulse blocking. Blocks the next rectifier thyristor firing pulse with direct communication to the thyristor firing card.

  3. Energy absorption. Lowers the DC voltage if it continues to rise.

As an option it is possible to have a switch that can temporarily deactivate the VLU function. This could be needed if the activation setting is very low, so that it might trigger if the rectifier charge mode is initial or boost charge.

The type and placement of the switch depends on customer specifications. For more details see the Drawings.
