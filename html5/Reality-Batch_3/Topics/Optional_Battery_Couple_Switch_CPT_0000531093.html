Optional Battery Couple Switch in a Redundant System

In a redundant system with two battery banks it is possible to have an optional battery couple switch. With an optional battery couple switch it possible to connect both batteries to one unit when isolating the other. This allows the full battery capacity of the redundant system to be connected to the load even if one unit is isolated. For information see How to Operate the Battery Couple Switch.
