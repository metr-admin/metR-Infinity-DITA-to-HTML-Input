Profibus-DP GSD File

The general station description (GSD) file extension used is developed by PROFIBUS & PROFINET International. The GSD file defines how devices communicates on a Profibus-DP bus.

When using the LUFP7 gateway on a Profibus-DP bus the primary device must use the GSD file: tele071f-gsd-file-for-profibus-v3.gsd

For additional information about the LUFP7 gateway and download of the GSD file for implementation see https://www.anybus.com/lufp.
