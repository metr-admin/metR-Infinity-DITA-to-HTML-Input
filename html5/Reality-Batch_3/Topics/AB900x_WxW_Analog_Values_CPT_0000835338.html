Analog Values

Each Modbus register is represented by two bytes. The first byte is the high byte (HB) and the second byte is the low byte (LB).

The decimal value is calculated from HB * 256 + LB. For example, if the first two bytes in the analog list (byte 18 and 19) reads 1 and 224, then the output voltage L1 is 1*256 + 224 = 480 V.
