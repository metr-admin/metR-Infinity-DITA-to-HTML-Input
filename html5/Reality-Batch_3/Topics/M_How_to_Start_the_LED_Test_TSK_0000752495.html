How to Start the LED Test

Note: The LED test is available for all user access levels, it does not require a login.
  1. Press the Double arrow icon to expand the Virtual LED bar.
  2. Press the LED test button at the bottom of the Virtual LED bar to start the test.
The test will illuminate all the virtual LEDs and flash the physical LED above the display for a few seconds. Check that all virtual LEDs are illuminated in their correct color and that the physical LED above the display flashes in all four colors (green, yellow, red and blue).
