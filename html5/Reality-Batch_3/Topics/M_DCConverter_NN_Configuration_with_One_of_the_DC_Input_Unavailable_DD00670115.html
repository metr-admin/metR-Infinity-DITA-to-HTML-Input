2N Configuration with One of the DC Input Unavailable

If one of the DC input becomes unavailable in a system with a 2N configuration, the other dc input will still supply one group of modules. One group modules is enough to supply the full load. The group modules can be N, N+1, N+x or N+N.

Part Status
DC Input 1 Available and within the tolerance range.
DC Input 2 Unavailable / Unknown
Converter DC Input 1 breaker Closed
Converter DC Input 2 breaker Closed / Open
DC/DC converter modules Half of the modules connected to the dc input that is still available are running and suppling the load. The other half is ready to operate when the unavailable dc input return.
