Interface Description

The human machine interface (HMI) is the main system interface. The HMI is installed on the front door and called the front panel. The display, keys and buttons are used to operate and configure the system. The mimic diagram gives a quick overview of the system status. The alarm panel shows any pending alarms.

From the display it is also possible to access:

  • The user stack which contains a menu with system options, modes and settings.

  • The alarm stack which contains more details on active alarms or events.

  • The event log which contains a list of previous alarms and events.

As an option analog instruments can be installed and programmable relay can be used.
