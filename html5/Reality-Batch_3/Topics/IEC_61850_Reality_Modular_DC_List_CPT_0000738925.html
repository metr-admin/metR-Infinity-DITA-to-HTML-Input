IEC 61850 Reality Modular DC Charger List

The default logical device (LD) for all IEC tags for Reality Modular DC Charger is “i61GMODACUPS_LD1”.

The IEC tag path format is LN$FC$DO$DA.

  • Logical node (LN)

  • Functional constraint (FC)

  • Data object (DO)

  • Data attribute (DA)

Note: The modbus register listed in the table is the number from the system controller, it is not the same as the NMC modbus register.
