How to Fix Forklift to a Carrier

  1. The rope/chain must have sufficient length for fixing.
  2. Park the vehicle on a level ground.
  3. Set the mast vertically. Lower the fork or attachment to the lowest position.
  4. Set all the operating devices to Neutral Position. Turn OFF the start switch.
  5. Apply the parking brake. Stop the tyres with blocks (D).
  6. If the vehicle has a mast, connect fixing rope/chain to the fork (A) and fix the vehicle using the reinforcing bar (B) of the lower frame. If without mast, fix with the drive axle wheel (C).
