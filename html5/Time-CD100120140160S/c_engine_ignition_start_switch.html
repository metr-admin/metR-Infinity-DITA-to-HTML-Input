C-Engine Ignition Start Switch

The key switch is a three position switch. Position (1) is ACCESSORY OFF. Position (2) is On/Ignition.

Position (3) is START, which engages the starter.

After the key has been turned to the START position, the key must be returned to the OFF position before it can be turned to the START position again.
