DEF/Ad-Blue Supply Module Filter Replacement

  1. Loosen the supply module plug located on the inside of frame step (LH).
  2. Remove the filter element from the supply module.
  3. Insert a new filter element.
  4. Fasten with the supply module plug.

    For more details, refer to the “engine service manual”
