Microsoft Windows Installation Fix

Some users encountered an error when using the X-Author for Contracts add-in in the Windows version of Word. Follow the steps in this section to correct the error and use the add-in normally.

Prerequisites

You will need to obtain the following files from your Apttus contact.

  • Apttus.XAuthor.AppManifest.xml – The add-in manifest file
  • WindowsOfficeAddInsLauncher.exe – The add-in utility required for completing Windows installation
  1. Save the manifest file Apttus.XAuthor.AppManifest.xml to a location on your computer. Make a note of the file path.
  2. Run the WindowsOfficeAddInsLauncher.exe file.
  3. Enter the complete file path that points to the manifest file on your system (for example, C:\OfficeManifest\Apttus.XAuthor.AppManifest.xml) and press Enter.
  4. Close the utility window.
  5. Open Microsoft Word and go to File > Options > Trust Center > Trust Center Settings > Trust Add-in Catalogs.
  6. Highlight the Office manifest file catalog entry and click Remove.
  7. Close all open Microsoft Word instances.
  8. Open Microsoft Word and go to File > Options > Trust Center > Trust Center Settings > Trust Add-in Catalogs.
  9. In the Category Url field, add the appropriate catalog path for your add-in, make sure Show in Menu is checked, and click OK.
  10. Close Microsoft Word.
  11. Open Microsoft Word and go to Insert > My Add-ins > X-Author Contracts to run the add-in.
