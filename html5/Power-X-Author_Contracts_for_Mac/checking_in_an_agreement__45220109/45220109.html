Checking in an Agreement Document

Whether you are saving a redlined version, clean version, or final version, the process for each is similar and each makes use of X-Author's Check-In feature.

Only Microsoft Word documents in DOCX format are supported for maintaining versions and checking in and checking out agreement documents. When you check in from X-Author, the document you are working on is converted back to the same document format as when it was first checked out.

  • Reconciliation
    X-Author provides a reconciliation feature which enables changes made to smart fields in an agreement document, outside of the Salesforce/X-Author environment, to be reconciled with the data held in the Agreement record in Salesforce.
