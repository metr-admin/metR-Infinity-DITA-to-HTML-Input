Start/Main Menu➔ Imaging Screen

Use this menu to set the cameras and the separation.

Camera Setup

For each camera (Main, Side, Side 2), set the gain. Adjust the gain percentages so that the camera images are not too dark and not too bright. (The brightness of the camera images can be adjusted.)

Separation

The ISWatch systems needs to separate camera images into bottle images.
  • By default, camera images are automatically separated into images of the bottle (Automatic Separation activated).
  • When halve bottle images are shown on the screen the automatic separation does not work correctly, the separation should be done manually.
Follow the steps described below. (See the image below.)
  1. Deselect Automatic Separation.
  2. Press Show Graphs to get an intensity curves over one machine cycle.
  3. Set the Limit for each camera, the limit is shown in the graph with a red line.
At the point where the intensity graph is higher than the red line (Limit value), the system will separate a bottle image.
Note:
Do not set the limit too low, or too many bottles will be identified as stuck ware. Do not set the limit too high, or too many bottles will not be detected correctly.

Start/Main Menu ➔Synchronization Screen

Use this menu to synchronize the system with the forming machine.
  • Press Start Synchronization in order to start the process.
  • Make sure at least 80% of the bottles are on the conveyor belt during synchronization.
  • When all the bottle images are displayed in the correct sections, synchronization is completed.

    In the image below, the bottle intensity is going from low to high, and the synchronization is correct.

    Note:

    If the synchronization is not correct, check the conveyor order in the Job Settings menu.

Start/Main Menu ➔Section Menu ➔Rejects Rejector Timing Screen

To set the rejector timing:
  1. Press Distance to Reject (Rejector), and enter the distance between the main camera viewpoint on the conveyor and the rejector. Normally this value is only set during the installation of the system and does not need to be changed.
  2. Press Pulse Time, and enter the pulse time in milliseconds (the time the reject valve is activated).
  3. Press Blow Off, and select one of the sections to reject.
  4. If the reject pulse is too early, increase the Distance to Reject value. If the pulse is too late, decrease the value.
  5. Repeat these steps until the rejector functions correctly.
Note:
In case the ISWatch system uses the forming machine rejector, and the machine reject pulse is connected to the ISWatch system, the Distance to Reject value can be calculated automatically by the system.
  1. Press Pulse Time, and enter the pulse time (the time the reject valve is activated).
  2. Press Start Procedure.
  3. Press Synchronization and select one section.
  4. Start rejecting bottles of this section with the forming machine reject (and ONLY one section). The ISWatch measures the distance during 5 rejects.
  5. Press Blow Off, and select one of the cavities/sections to reject.
  6. If the reject pulse is too early, increase the Distance to Reject value. If the pulse is too late, decrease the value.
  7. Repeat steps 5 and 6 until the reject system functions correctly.

Start/Main Menu ➔Machine Screen

Use the Machine menu to restart the computer, or shut down the ISWatch program.

  • Use Reset Learning to reset all internal software algorithms. Use this in case the system behaves abnormally.
  • Press Events to view system events, such as when an operator is logged on, etc.

Start/Main Menu➔ User Interface Screen

With the User Interface screen, you can customize the user interface to your needs.

Show equal images: This option displays the bottle images in the overview with equal intensities.

Clean Touchscreen: This feature disables the touchscreen for a short period in order to clean it.

Timeframe - Graphs: Change the time period for the graphs in the Section Screens.

Timeframe - Overview Statistics: Change time period for statistics shown in upper right of screens.

Alarm Indicators: Enable (Use selected) the Alarm Indicators to display them on the Dashboard and Overview screens as dots located below the cavity and filled regions in the bottles.

Choose between Alarm and Reject to show the indicators.
  • If the Alarm is selected, whenever an alarm is generated (for glass distribution etc.) or an inclusion is detected, an internal counter will be increased.
  • If the counter is equal or higher than the Number (in the screen below, 1) in the Timeframe set (60s in the screen below), the indicator will be shown.
  • For the Source Reject it works the same.
In the setup shown below, if one alarm is generated during the last 60 seconds, the alarm indicator will be shown.

Red Rectangle: Enable (Use selected) the Red Rectangle to display them on the Dashboard and Overview screen as a rectangle around the cavity

Choose between Alarm and Reject to show the indicators.
  • If the Alarm is selected, whenever an alarm is generated (for glass distribution etc.) or an inclusion is detected, an internal counter will be increased.
  • If the counter is equal or higher than the Number (in the screen below, 1) in the Timeframe set (60s in the screen below), the indicator will be shown.
  • For the Source Reject it works the same.
  • Worst Cavities is the number of maximum rectangles shown.

Alarm Lamp: Enabling (Use selected) the Alarm Lamp will turn the 24VDC Lamp output on IO-module on when the Red Rectangle is shown on the screen.

Disable With Rejector: Enabling this will disable the alarm Lamp when the Rejects are turned off.
