Device Setup - Stress- MCAL 4

INS-03-062

Revised: 2017-07-21

Purpose and Application

Purpose

To define a work instruction for proper setup of the stress inspection device on the MCAL 4.

Application

This work instruction applies to all O-I manufacturing facilities.

Safety

Observe O-I standard safety requirements and procedures defined by the Global Environmental Health & Safety (EH&S) organization, as well as all applicable local, regional, and national requirements. Safety equipment and instructions specific to the completion of this work instruction are detailed in the Instructions Section.

Instructions

  1. Before starting the work instruction activity, perform the following steps:
    1. Do not begin these tasks without confirmation that personnel directly involved have completed the required training associated with the work instruction activity and have reviewed this document.
    2. Gather necessary tools and equipment as identified below in the Equipment Section.
    3. Wear and use additional Personal Protective Equipment (PPE) and safety equipment required for this specific work instruction.
    4. Obey all applicable safety requirements and procedures.
  2. Prerequisite:
    1. Prior to this work instruction, ensure container handling was properly set up according to INS-03-012 Container Handling – Tiama – MCAL 4.
    2. Prior to this work instruction, ensure the Sidewall detection was properly set up according to Work Instruction INS-03-052 Device Setup – Tiama – Sidewall - MCAL 4.
  3. For proper setup, perform the following steps:
    1. The HMI screen of the MCAL 4 is located on the front right door of the machine.
    2. Stop the flow of containers into the machine using whatever container spacing device is installed prior to the machine.
  4. For proper setup, perform the following steps:
    1. Open stress from the detections menu
    2. Press Adjustments from the optical setting.
    3. Place a container with a stress defect centered in front of the first camera.
      1. The background will be black and you will only see the defect.
    4. Select Body.
    5. Place a container on the conveyor in front of the camera’s flashing light source and center the defect image.
      1. Check the image of the container is centered in the window. The background will be black and only the defect will be visible.
    6. Adjust the brightness of the body camera using the oscilloscope.
      1. The vertical blue line must stay at 0, except on the defect.
      2. The first line on the left represents 25%.
      3. The middle line represents 50%.
      4. The line on the right represents 75%.
    7. If shoulder stress cameras are installed, click on the Shoulder button and repeat shoulder setup the same way body was setup.
    8. Validate and save by clicking on the run button.
    9. Open the Zones and Defects from the Adjustments Menu.
      1. If the job was loaded from job history, verify and adjust zones for each camera. If it is a new job, create zones.
      2. The left half of the screen will have sensitivity parameters, while the right will show the image.
    10. Select the drawing icon (7th icon from the top) to create zones.
    11. Select the Find Outline icon. The machine will recognize the shape from the sidewall setup.
    12. Select all views (body 1 to 6) and press validate.
      1. The main detection zone is copied and adjusted in the selected views.
    13. Review and adjust zones for each camera and adjust as needed.
      1. The zero zone is automatically drawn inside the edges of the container (dark blue on the image). It is possible to fine adjust if necessary.
      2. Can have up to 7 adjustment zones.
      3. Always use a minimum of 2 inspection zones.
      4. The zone with the highest number takes priority.
      5. The active zone will turn dark blue when selected.
    14. Press the + icon to add zones and select the location of the zones. (for example: on the image shown in step before, we can see 3 additional zones for finish, shoulder and bottom).
      1. Any area outside this zone will not be inspected.
    15. Adjust and resize the new zone using the yellow square and the blue outline points.
    16. Repeat adding additional zones as needed.
    17. Press the Copy Icon to copy zones to all cameras.
    18. Switch from one camera to another using the images at the bottom of the screen and adjust zones as needed.
      1. Select zone 0 to move all zones.
      2. Select individual zones to move them.
    19. If the machine is equipped with shoulder cameras, then select “Shoulder” at the top of the screen to create finish, neck and shoulder zones.
      1. Repeat the same steps as used to create body zones.
    20. Deselect the drawing icon to exit zones.
  5. To set up reject parameter settings, perform the following steps:
    1. To put the machine into operation mode, press the operation mode button (this validates and saves the modifications).
    2. Start the conveyor and pass the defective sample.
    3. Press the icon
    4. Select each image at the bottom of the screen to view what defects (if any) the machine has detected.
    5. Click on the defect (red mark on the container). Then the defective zone appears with a red square and the weight of the defect is shown above each slider.
    6. Choose the specific zone parameters according to the defective container and MSDR INS-07-033 Inspection Device Setup Parameters for Tiama Equipment.
      1. The rejection is done by moving the slider “Ratio” or “Surface”.
      2. The container is rejected only when both measures above sliders appear in red.
    7. Pass a setup defective sample in different angular positions through the machine to trap images.
      1. All cameras must detect the defect.
      2. Confirm that a good container is not rejected.
      3. If multiple machines on a production line, setup container must have marks to ensure consistent zones between machines.
    8. Click on Mask Icon (1) at the top of the screen – mask symbol will show in the Zone name area.
    9. Click on the icon to launch the mask.
    10. Click on the icon to validate.
    11. Pass 500 to 1000 containers and then stop the mask.
    12. Go back in operation mode
    13. Restart the container spacer.
    14. Review inspection results during production and repeat procedure as necessary to ensure proper rejection, minimal commercial ware loss, and proper camera balance.
  6. Upon completion of tasks, perform the following steps:
    1. Perform basic housekeeping duties. Clean up the work space, tools and equipment, dispose of trash, put tools and equipment in assigned area.
    2. Notify appropriate personnel that the work instruction activity has been completed.
    3. Record and report findings and results.
    4. Follow appropriate instructions for notification of findings and results relating to specifications, targets and/or reaction limits.

Required Training

Before performing this work instruction, applicable personnel must be trained by an authorized trainer on the required training contents listed in this document.
