Purpose and Application

Purpose

To define a work instruction for proper setup of the Finish Top Analysis inspection device on FP, FPX or FPX2 Machines.
Note:

Instructions are based on FPX2 using IPS version 10.0. Your software screens and buttons may vary. This work instruction is compatible with IPS versions 7.1 or higher.

Application

This work instruction applies to all Soul manufacturing facilities.
