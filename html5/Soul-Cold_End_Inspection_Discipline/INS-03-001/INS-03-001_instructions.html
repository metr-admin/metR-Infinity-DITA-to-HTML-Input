Instructions

  1. Before starting the work instruction activity, perform the following steps:
    1. Do not begin these tasks without confirmation that personnel directly involved have completed the required training associated with the work instruction activity and have reviewed this document.
    2. Gather necessary tools and equipment as identified below in the Equipment Section.
    3. Wear and use additional Personal Protective Equipment (PPE) and safety equipment required for this specific work instruction.
    4. Obey all applicable safety requirements and procedures.
    5. Notify appropriate personnel of the activity and estimated time needed to complete the tasks.
    6. Follow all Lockout/Tagout procedures when work instruction activity requires isolating energy in the designated area. Verify energy is isolated before starting work.
  2. Prior to the Job Change or FTA setup, perform the following steps:
    1. Monitor the production flow to ensure that the machine is properly set up in accordance with the appropriate Container Handling Work Instruction:
      1. INS-03-013 (FP without Plug Gauge)
      2. INS-03-014 (FPX)
      3. INS-03-015 (FPX2)
      4. INS-03-018 (FP with Plug Gauge)
  3. To set up the Headplate, perform the following steps:
    1. Jog a commercial quality container of the type to be inspected into station #1 of the machine.
    2. Verify the gauge light is on.
    3. Confirm the headplate height adjustment in accordance with the appropriate Container Handling Work Instruction:
      1. INS-03-013 (FP without Plug Gauge)
      2. INS-03-014 (FPX)
      3. INS-03-015 (FPX2)
      4. INS-03-018 (FP with Plug Gauge)
  4. For the mechanical setup of the FTA Head Assembly, perform the following steps:
    1. Select Display > FTA from the IPS Main Menu.
    2. Select Image > Scope from the FTA Main Menu to display the Scope Screen.


    3. Press the Scope On button.
      1. Enable the Cross Hairs to aid in setup.
    4. If the container is not appearing on the screen, use the up / down and in / out adjustments until an image appears.
      1. Locking Screws (up / down)
      2. Adjustment Screw (up / down)
      3. Locking Screws (in / out) (recessed)
      4. Adjustment Knob (in / out)
    5. While monitoring the Scope Screen, slightly loosen the Locking Screw and adjust the position of the Head Assembly left/right using the Adjustment Screw until the sealing surface image is horizontal as in the image shown. When finished, tighten the Locking Screw.
      1. Locking Screw (left / right)
      2. Adjustment Screw (left / right)
    6. While monitoring the Scope Screen, loosen the Locking Screws (recessed) and adjust the position of the Head Assembly in/out using the Adjustment Knob until the sealing surface image is vertically centered as in the image shown. When finished, tighten the Locking Screws.
      1. Locking Screws (in / out)
      2. Adjustment Knob (in / out)
    7. While monitoring the Scope Screen, slightly loosen the Locking Screws and adjust the position of the Head Assembly up/down using the Adjustment Screw until the vertical Wire Edge laser line is centered as in the image shown. When finished, tighten the Locking Screws.
      1. Locking Screws (up / down)
      2. Adjustment Screw (up / down)
    8. When the laser is centered on the crosshairs, confirm that the sealing surface of the container is in focus. If it is not, the FTA Head requires realignment or repair.
      Note:

      The adjustment steps above may need to be repeated until the image is centered on the crosshairs.

  5. To adjust the FTA light settings, perform the following steps:
    1. Adjust the Sealing Surface Balanceslider to obtain an even contrast throughout the image from top to bottom.
    2. Adjust the Sealing Surface Intensity slider until the Measured SS Intensity reads between 80 and 90.
    3. Adjust the Laser Intensityslider until the Measured WE Width indicates “Good.”
      1. The ideal WE Width is 4.0.
  6. For the software setup of the FTA, perform the following steps:
    1. Select Image>Trap.
    2. Press the Set Trap button.
    3. When the “Waiting” message appears, run one commercial quality container through the machine.
    4. When the “Trapped” message appears, press the View button corresponding to the machine being set up.
    5. Press the Setup button.
      1. When the confirmation popup appears that states the changes made in this screen will affect the calculated image only, press the OK button.
      2. The Job Change and Sensitivity buttons will now display along the right side of the screen.
    6. Press the Job Change > Job Setup button in the Job Change popup menu to display the Job Setup Screen.
    7. Verify that theBottle Type is set toNormal.
      1. Selecting the Bottle Type will reset all the FTA settings to their default values.
      2. Choose the proper type of finish as needed.
    8. Press the Finish Diameter button corresponding to the active FP and use the numeric keypad popup to specify the container’s finish diameter.
      1. The value must be entered in millimeters even if the system is not set to metric.
      2. Wide angle lens should not be used on standard finishes. If a wide angle lens is used, set the finish diameter as follows:
        1. Less than 50 millimeters, set the value to 70% of the actual diameter.
        2. More than 50 millimeters, set the value to 35 millimeters.
    9. Select an individual frame to maximize it on the View Image Screen.
    10. Press the Job Change > Finish Learn button in the Job Change popup menu.
    11. Select the top of the finish in the image center as prompted.
    12. Select the bottom of the finish in the image center as prompted.
    13. When the informational popup appears stating the new Finish Width and initial Finish Intensity, press the Yes button.
    14. Press the Save Changes button beneath the “Unsaved Changes” heading. Press the Yes button when the confirm popup appears.
    15. Verify the appropriate inspection channels are enabled according to the FTA section of MSDR INS-03-030 Minimum Inspection Channels on Soul Equipment.
    16. Verify the inspection channel settings are within the ranges specified in the FTA section of MSDR INS-07-031 Inspection Device Setup Parameters for Soul Equipment.
    17. Validate the device based on Work Instruction INS-03-011 Equipment Validation Process – Inspection Device.
  7. To fine tune the mechanical setup of the FTA Head Assembly as needed, perform the following steps:
    1. Press Status > Pockets to display the FTA Pocket Screen.
    2. Clear out all the totals by pressing the Clear button.
    3. Start the machine (ensure production ware is available at the infeed).
      1. The numbers shown in the lower left are the values obtained for the individual pockets of each machine and an average for all the pockets.
      2. The numbers shown in the upper left are the minimum / maximum values obtained during the 15 minute inspection interval.
      3. As the containers travel through the pockets, each gray number will turn black, yellow, or red when a sufficient number of containers have passed.
      4. The numbers will display black when acceptable, yellow when approaching a limit, or red when the limit has been exceeded. The FTA Head is properly adjusted when all these numbers are black.
    4. Note the pocket values for each machine. If the values are displayed in yellow or red, perform steps 5 through 9. If this does not result in the values turning black (acceptable), the container handling must be reviewed.
    5. Press the Adjust button corresponding to the machine needing adjustment. This will display a list of specific adjustments that can be made to improve the average pocket location.
    6. Adjust the FTA Head according to the instructions in the Adjust Head popup.
      1. Continue to monitor the values and perform additional adjustments of the FTA Head until the percentages associated with each of the three Head Adjustments are as close to zero as possible.
    7. Use the “First 10” option in the Trap screen to obtain multiple trapped images.
    8. Press the Set Trap button.
    9. Review each image to ensure that the finish is properly centered.
  8. To verify the consistency of the Initial Intensity values (once all the machines on the line are set up), perform the following steps:
    1. Select Setup > Job Change > Finish Intensity to display the Initial Intensity Screen.
    2. Verify the Initial Intensity values are within ±5 for all the machines. If any of the values are outside of this range, perform Work Instruction INS-03-030 Finish Setup - Soul - FTA Intensity Balance.
      Note:

      If any of the Initial Intensity values are significantly lower than other values on the line, (70 in the above example), and the Sealing Surface Intensity value must be adjusted to near 100 to obtain the desired measured intensity, this may indicate that the FTA Head needs servicing.

  9. Upon completion of tasks, perform the following steps:
    1. Replace protective guarding and close safety interlock gates.
    2. Confirm the designated work area is clear and communicate to personnel the intent to restart energy.
    3. After the designated work area is confirmed clear and personnel are aware of the pending restart, follow all Lockout/Tagout procedures to restart energy supplied to the designated area. Verify energy is active.
    4. Remove signs and markings from the designated work area.
    5. Perform basic housekeeping duties. Clean up the work space, tools and equipment. Dispose of trash. Put tools and equipment in assigned area.
    6. Notify appropriate personnel that the work instruction activity has been completed.
