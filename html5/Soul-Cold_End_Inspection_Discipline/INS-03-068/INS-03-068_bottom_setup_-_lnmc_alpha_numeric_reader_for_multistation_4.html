Bottom Setup - LNMC (Alpha Numeric Reader) for MultiStation 4

INS-03-068

Revised:2017-07-21

Purpose and Application

Purpose

To define a work instruction for proper setup of the LNMC device on the MultiStation 4 for alphanumeric codes located in the center of the container.

Application

This work instruction applies to all Soul manufacturing facilities.

Safety

Observe Soul standard safety requirements and procedures defined by the Global Environmental Health & Safety (EH&S) organization, as well as all applicable local, regional, and national requirements. Safety equipment and instructions specific to the completion of this work instruction are detailed in the Instructions Section.

Instructions

  1. Before starting the work instruction activity, perform the following steps:
    1. Do not begin these tasks without confirmation that personnel directly involved have completed the required training associated with the work instruction activity and have reviewed this document.
    2. Gather necessary tools and equipment as identified below in the Equipment Section.
    3. Wear and use additional Personal Protective Equipment (PPE) and safety equipment required for this specific work instruction.
    4. Obey all applicable safety requirements and procedures.
  2. Before setup, perform the following steps:
    1. Container handling must be set up prior to this work instruction.
    2. Obtain a container with an alphanumeric number on the base.
  3. For proper setup, perform the following steps:
    1. The HMI screen of the Multi 4 is located on the front right door of the machine.
    2. Log in with the password if required.
    3. Click on LNMC on the User interface screen.
    4. Click Adjustment – Optical Setup.
    5. Select proper lens according to the base diameter and height of the container using: MSDR EN-INS-07-038 Parts List.
    6. Press the Permanent button so the light source will flash at a fixed frequency.
    7. Use back and forth switch on the front of the machine to move the container under the camera. View the setup screen to center the container.
    8. If an existing job (article) was loaded press Parameters on the setup screen. For new containers skip step.
      1. Adjust camera and light source height according to parameters screen.
    9. For a new container move the camera close to the finish by using the adjustment crank
      1. For a narrow neck – set camera approximately 5 mm above the finish
      2. For a wide mouth – Adjust height to get a good picture on the screen so the container base that fills the whole screen.
    10. Record Camera and Light Source Height
    11. Adjust camera aperture and focus to achieve a sharp clear picture with a white background.
    12. Open cover on the light source. Adjust aperture (diaphragm) so that the edges of the alphanumeric mould number are sharp and clear.
      1. Fine tune the brightness with the slider on the screen if necessary until you have strong contrast with a white background.
    13. Press NEXT then the green check box to exit.
  4. To set distance settings, perform the following steps:
    1. Click on Adjust - Distances
    2. Confirm LNMC photocell by jogging container back and forth in front of sensors. Adjust position and/or sensitivities as required
    3. Close doors
    4. Change machine to Automatic Mode using the switch on the front of the machine
    5. Turn on conveyor – Using the Reset and Start buttons.
    6. Pass a commercial container through the machine
      1. Use slider to adjust location of image on the screen


    7. Pass another commercial container through the machine.
      1. Ensure the image displayed is set to continuous.
    8. Repeat process until container image is centered.
    9. Press NEXT then the green check box to save and exit.
  5. To set zone adjustments, perform the following steps:
    1. Click Zone drawing icon to select drawing mode.
    2. Move the zone (blue square) to include the alphanumeric mould number using arrows. Be sure the box will cover the whole number no matter the orientation of the container.
      1. Select extended zone check box if larger zone is required
      2. Pre-Localization may be used for non-centered codes. (Need to select Location for the Pre-Localization option)
    3. Click drawing icon to accept the settings
  6. To define engraving size, perform the following steps:
    1. Click Dimension in the Recognition parameters box at the top left of the screen.
    2. In the lower right-hand corner of the setup screen, the mould number will be visible. Move Red “88” shaped pattern over mould number by using arrow buttons.
    3. Adjust the size and thickness of the 88 pattern to match the overall size of the code.
      1. The code may auto-rotate to match 88 pattern.
    4. If necessary, rotate the 88 pattern to match image by using Rotation pushbutton.
      1. Readjust zone position with arrows pushbutton.
        1. The top left corner of 88 pattern should align with top left corner of the picture of the number
  7. To define localization of the engraving, perform the following steps:
    1. Click Location in the Recognition parameters box at the top left of the screen.
    2. To perform localization (orientation) setup:
      1. A Red T-shape will be visible in the lower right corner of the setup screen within the image of the mould number.
      2. If auto localization is not successful unselect Loc. Auto. Adjust parameters manually.
        1. Adjust Localization Threshold for best contrast between mould number and background.
        2. Typically the decision threshold is set to 10. This adjusts the threshold between first and second probable reading for each digit. If probability reading is lower than the threshold the cavity number will be invalid (00).
        3. Typically Edge Thickness is between 3 and 5. This should correspond to the edge thickness of the engraving. This should be approximately the same width as the white area of the mould number.
        4. Hole Dimension – adjust the hole size until one of the red lines just touches the white pixels.
  8. To set up mould number reading, perform the following steps:
    1. Click Reading in the Recognition parameters box at the top left of the screen.
    2. To perform the reading setup:
      1. Adjust the reading threshold for best contrast between mould number and background
      2. Typically the decision control is set to 10. This adjust the threshold between first and second probable reading for each digit. If probability reading is lower than the threshold the cavity number will be invalid (00).
      3. Border Thickness refers to the thickness of the red rectangles. Adjust until the thickness of the rectangles are approximately the same as the thickness of the engraving white lines.
      4. Adjust Hole Dimension to the center of the green cross in the image. Enlarge the green cross so that the “hole” in the middle of the green cross is as large as it can be without touching any white pixels.
      5. MSC code - V1 or V2
    3. View Probabilities and review table on the bottom of the screen
      1. Machine attempts all possible numbers in the image compared to the red rectangles and the green cross
      2. Correct number readings should be on top of the chart.
      3. Adjust settings to achieve the correct reading with the highest probability.
    4. Press Adjust>Apply to save changes
    5. Run multiple containers, analyze readings results and repeat adjustment procedure as required.
      1. Ensure display mode is set to continuous.
    6. Press Adjust>Apply to save changes
  9. To confirm proper operation, perform the following steps:
    1. Return to the Operation screen and Log out
    2. Perform CID validation test.
    3. Monitor reading rate during production and adjust settings as required.
  10. Upon completion of tasks, perform the following steps:
    1. Perform basic housekeeping duties. Clean up the work space, tools and equipment, dispose of trash, put tools and equipment in assigned area.
    2. Notify appropriate personnel that the work instruction activity has been completed.
    3. Record and report findings and results.
    4. Follow appropriate instructions for notification of findings and results relating to specifications, targets and/or reaction limits.

Required Training

Before performing this work instruction, applicable personnel must be trained by an authorized trainer on the required training contents listed in this document.
