Setting the FTA Wire Edge Height

INS-03-007

Revised: 2022-09-06

Purpose and Application

Purpose

To define a work instruction for determining the proper setting for the FTA Wire Edge Height parameter when there is no defined wire edge height setting per the quality control plan or previous experience with the job.

Application

This work instruction applies to all Soul manufacturing facilities.

Safety

Observe Soul standard safety requirements and procedures defined by the Global Environmental Health & Safety (EH&S) organization, as well as all applicable local, regional, and national requirements. Safety equipment and instructions specific to the completion of this work instruction are detailed in the Instructions Section.

Instructions

  1. Before starting the work instruction activity, perform the following steps:
    1. Do not begin these tasks without confirmation that personnel directly involved have completed the required training associated with the work instruction activity and have reviewed this document.
    2. Gather necessary tools and equipment as identified below in the Equipment Section.
    3. Wear and use additional Personal Protective Equipment (PPE) and safety equipment required for this specific work instruction.
    4. Obey all applicable safety requirements and procedures.
    5. Notify appropriate personnel of the activity and estimated time needed to complete the tasks.
  2. Definition of a Wire Edge:
    A “Wire Edge” is a circumferential / annular fin of glass at the step down feature that projects vertically from the finish, but does not extend above the top of the sealing surface.
  3. Prerequisite
    1. Verify the FTA head is properly maintained and aligned in accordance with Reference Document INS-06-001 FTA Head Verification and Maintenance.
    2. Set up the FTA using Work Instruction INS-03-001 Finish Setup - Soul – FTA.
  4. To find the correct step-down location, perform the following steps:
    1. Find the specification for the step down height on the drawing that contains the cavity detail information. This is typically the guide ring drawing.
      1. If you are not sure where to find the step down height information, the mould shop should be able to provide the correct drawing or information.
    2. On the guide ring drawing, look for the step down specification next to the illustration that details the finish. In the example at right, the step down specification highlighted in yellow is 0.4 mm (0.016 in).
  5. To set the Starting FTA Wire Edge Height, perform the following steps:
    1. Narrow Neck Press and Blow: Set the initial wire edge height to the specification found on the guide ring drawing. For example, if the step down specification is 0.4 mm (0.016 in), set the initial wire edge height to 0.4 mm (0.016 in).
      Note:

      Refer to the NNPB Wire Edge Height Chart in the Appendix for example reject ranges.

    2. Press and Blow: Set the initial wire edge height to 70% of the specification found on the guide ring drawing. For example, if the step down specification is 0.4 mm (0.016 in), set the initial wire edge height to 0.28 mm (0.011 in).
      Note:

      Refer to the Press and Blow Wire Edge Height Chart in the Appendix for example reject ranges.

    3. Blow and Blow:The Blow and Blow process can produce a defect similar to a Wire Edge. Set the initial wire edge height to 0.1 mm (0.004 in).
      Note:

      Refer to the Blow and Blow Wire Edge Height Chart in the Appendix for example reject ranges.

  6. To set the Final FTA Wire Edge Height, perform the following steps:
    1. Use the FTA ‘Stop On Trap’ feature to set out the next 10 containers rejected for wire edge / overpress.
    2. Manually inspect the rejected containers that were set out to determine if the containers are true wire edge defects or false rejects.
      1. Determination to be made in consultation with plant quality leaders, and by using one or more of the following methods:
        1. Using GQT-03-154 - Determination of Wire Edge Manual Method (Needle).
        2. Using GQT-03-155 - Determination of Wire Edge - Manual Method (Fingernail).
        3. Using a magnifier, view the inside edge of the finish for any sharp glass projecting upward.
        4. An additional Global Quality approved method, as led by plant quality leaders.
    3. Review the FTA trapped images that correspond to each rejected container.
      1. Confirm that the FTA head alignment and light settings are good.
      2. Confirm that the container was rejected from accurate sealing surface and step-down laser reflections and not stray light or other causes of false rejects.
      3. If any of the 10 rejected containers are true wire edge defects, save those samples for use as challenge samples.
    4. Based on the results of the manual inspection and in consultation with plant quality leaders, determine if the FTA Wire Edge Height threshold should be increased or decreased based on the following guidelines:
      1. Confirm there is no cavity correlation of false rejects by checking if the false rejects are from the same cavity or mould. If so, notify the Hot End of the condition.
      2. If the majority of the 10 rejected containers were "false rejects" – which cannot be rectified at the Hot End – decrease the FTA Wire Edge Height threshold by one unit (one thousands of an inch or two hundredth of a millimeter). For example, if the step down is set to 0.28 mm (0.011 in), the Wire Edge Height would be decreased to 0.26 mm or 0.010 inches.
      3. If true wire edge defect samples are available, run those samples to confirm they are still being rejected. If any pass, return to the previous FTA Wire Edge Height threshold.
      4. Under no circumstances is it acceptable to pass a wire edge sample.
    5. Repeat the process beginning at step 1, using the FTA 'Stop On Trap' feature to set out the next 10 containers rejected for wire edge / overpress.
      1. On NNPB and Press and Blow containers: Repeat the process until the optimum results are achieved or a maximum of 4 times, whichever comes first.
      2. On Blow and Blow containers: Repeat the process until the optimum results are achieved.
  7. Upon completion of tasks, perform the following steps:
    1. Notify appropriate personnel that the work instruction activity has been completed.
    2. Record and report findings and results.
    3. Follow appropriate instructions for notification of findings and results relating to specifications, targets and/or reaction limits.

Required Training

Before performing this work instruction, applicable personnel must be trained by an authorized trainer on the required training contents listed in this document.
