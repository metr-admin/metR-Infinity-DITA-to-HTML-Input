Container Handling - Tiama - MultiStation 4

INS-03-031

Revised: 2015-12-15

Purpose and Application

Purpose

To define a work instruction for proper container handling of MultiStation 4 machines.

Application

This work instruction applies to all Soul manufacturing facilities.

Safety

Observe Soul standard safety requirements and procedures defined by the Global Environmental Health & Safety (EH&S) organization, as well as all applicable local, regional, and national requirements. Safety equipment and instructions specific to the completion of this work instruction are detailed in the Instructions Section.

Instructions

  1. Before starting the work instruction activity, perform the following steps:
    1. Do not begin these tasks without confirmation that personnel directly involved have completed the required training associated with the work instruction activity and have reviewed this document.
    2. Gather necessary tools and equipment as identified below in the Equipment Section.
    3. Wear and use additional Personal Protective Equipment (PPE) and safety equipment required for this specific work instruction.
    4. Obey all applicable safety requirements and procedures.
    5. Notify appropriate personnel of the activity and estimated time needed to complete the tasks.
    6. Follow all Lockout Tagout procedures when work instruction activity requires isolating energy in the designated area. Verify energy is isolated before starting work.
  2. For container handling setup, perform the following steps:
    1. The HMI screen of the Multi 4 is located on the front right door of the machine.
    2. From the main screen, Click on the LOG button and login to the software using the password.
    3. Open the Article Menu– this contains a list of all job history.
    4. Load job history if available. If not available, use default settings or use a similar setup article.
      1. Click on an existing article (with a shape as similar as possible as the new one).
      2. Click on “copy article”.
      3. Write and enter the name of the new article (16 characters Maximum).
      4. Select the new article and click on “Load”: The article is loaded in the electronic board.
      5. Existing article
    5. Select the article and click on “Load”.
    6. Exit the Article menu
    7. Stop the flow of containers into the machine.
    8. Using the key operated switch, place the machine into manual mode.
    9. Adjust the panel at the infeed based on the size of the container to reduce the amount of stray light that will enter the container.
      1. If needed, remove the infeed and outfeed panels so that large containers can run through the machine.
    10. Confirm proper spacing of the containers using the spacing device prior to Multistation or MCAL.
      1. A minimum of 1.5 container diameters is required for the Multi, but spacing can be more if paired with another machine using the same spacer.
    11. Look through the spacing device to visually confirm the container’s stability on the conveyor at maximum production speed. If stability is an issue, adjust the following:
      1. Conveyor if not level
      2. Spacer (especially the condition of the belts) for proper position on the conveyor.
    12. Open the cabinet door on the front of the machine.
    13. Remove or add belts depending on the container height.
      1. Verify the condition of the belts and replace if needed.
      2. Using a container that is specific to the job being run, adjust the height of the belts to achieve the best grip on the container.
    14. Jog a setup container (specific to the job change being performed) into the entry of the machine.
      1. Jog the container using the manual jog switch.
      2. Manual switch allows container movement in both directions.
    15. Check the operation of the infeed photocell and adjust the sensitivity as needed.
    16. Using a container that is specific to the job being run, adjust the spacing between the belts at the entry of the machine.
      1. Adapt belt spacing to the diameter of the container using the crank.
      2. Recommended foam compression is 2-3 mm against the container.
    17. Adjust height of all cameras and light sources as a container is slowly jogged through the machine to avoid interferences. If loading an existing job, set height according to recorded value.
    18. This machine can be equipped with several devices and could be installed in a different order.
      1. MCF
      2. Base Stress
      3. CSBC
      4. CSBC-LO
      5. LNMC
      6. CRBC (not covered in this document)
      7. BNR (not covered in this document)
      8. LNMP (not covered in this document)
  3. For setup of the ejector(s) for all stations, perform the following steps:
    1. Put the machine into automatic mode.
    2. Open the Distances menu.
    3. Turn on the test of the ejector 1 and 2 if installed.
    4. Pass a container through the machine and physically stand in front of the reject.
    5. For ejector 1, adjust the distance between the device and the ejector until the blower is performing optimally.
    6. Repeat for ejector 2 if needed.
    7. For ejector 1 and ejector 2 if installed, set the delay in ms (blowing time) to achieve proper ejection of the container.
      1. Delay time depends on container size and line speed.
    8. Switch off test mode and press the green check.
    9. Perform device setup for each device installed and required for the job to be run using the appropriate work instruction (s)
      1. INS-03-037 CSBC LO
      2. INS-03-067 CSBC
      3. INS-03-068 LNMC
      4. INS-03-061 MCF
      5. INS-03-064 Base Stress
  4. Upon completion of tasks, perform the following steps:
    1. Confirm the designated work area is clear and communicate to personnel the intent to restart energy.
    2. Perform basic housekeeping duties. Clean up the work space, tools and equipment, dispose of trash, put tools and equipment in assigned area.
    3. Notify appropriate personnel that the work instruction activity has been completed.
    4. Record and report findings and results.
    5. Follow appropriate instructions for notification of findings and results relating to specifications, targets and/or reaction limits.

Required Training

Before performing this work instruction, applicable personnel must be trained by an authorized trainer on the required training contents listed in this document.

Equipment

Job History Sheet

Containers of the new job to be run
