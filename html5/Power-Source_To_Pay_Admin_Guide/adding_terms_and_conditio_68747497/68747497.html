Adding Terms and Conditions Page on Supplier Portal

Apttus STP allows you to add your own Terms and Conditions on the Supplier Portal. These Terms and Conditions will be visible to the supplier once they log in to the supplier portal. Once a supplier accepts these Terms and Conditions, he will never be prompted to accept them again. You can also enter the content to be displayed on the Welcome page of public RFI.

Parent topic: Strategic Sourcing User Roles

To add terms and conditions page

  1. Go to Setup > App Setup > Develop and select Static Resources.
  2. Click New.
  3. Enter Terms_Conditions as text in the Name field.
  4. Enter a short description that describes the Terms and Conditions in the Description field.
  5. For File field, click Choose File and select the file that contains the terms and conditions. It accepts any of the following formats: .zip and .jar files, images, stylesheets, JavaScript, and other files.
  6. From the Cache Control picklist menu, select Public.
  7. Click Save.

To add content of RFI Welcome Page

  1. Go to Setup > App Setup > Develop and select Static Resources.
  2. Click New.
  3. Enter RFI_Welcome_Page as text in the Name field.
  4. Enter a short description that describes the Terms and Conditions in the Description field.
  5. For File field, click Choose File and select the file that contains the terms and conditions. It accepts any of the following formats: .zip and .jar files, images, stylesheets, JavaScript, and other files.
  6. From the Cache Control picklist menu, select Public.
  7. Click Save.
