Sourcing Event Type

Sourcing Event Type is a pre-defined sequence of steps for a sourcing event. Sourcing Event Type enables you to select the Phases that you want to include your sourcing event. The Chevrons are generated dynamically on the basis of the phases that you select. You can have multiple sourcing event types. For a sourcing event, only one sourcing event type can be selected. Preparation and Invitation are two mandatory phases for any sourcing event. By creating a sourcing event type, you can select any optional Phase such as RFI, RFQ, Evaluation, or Award. You can create sourcing event types based on your business requirements. You must enter a default Description and Detailed Description of the sourcing event.

Note

  • Preparation and Invitation are selected by default.
  • Award is auto-selected if RFQ is selected.
  • Either RFI or RFQ must be selected.
  • For concurrent sourcing event types, you must select Yes radio button in Do you want to run RFI and RFQ concurrently? field. This will let the suppliers respond to both RFI and RFQ phases, simultaneously.
  • For providing access to non-portal suppliers to RFI Surveys, you must select Do you want to collect responses from suppliers with no portal access? check box. For now, this feature is available for RFI Surveys only.
Tip:

Use Case

Tier 1 Mobiles has different sourcing events for different products. For a sourcing event of disinfecting wipes, they require a wide variety of products, and always have new suppliers coming in. Hence, they use a Full Sourcing event type. For bleach products, they use Only RFQ sourcing event type as they have fixed suppliers. Likewise, a lot of sourcing event types can be created depending on business requirements.

To create a sourcing event type

  1. Click the Sourcing Event Type tab.

    --or--

    Click and select Sourcing Event Type.

  2. Click New.
  3. Enter the Name of the sourcing event type.
  4. Enter the Description of the sourcing event type.
  5. Under Select Phase section, select the Phases that you want to include in your sourcing event.
  6. Select Yes radio button beside Do you want to run RFI and RFQ phases concurrently? if you want to run RFI and RFQ phases concurrently.

    or

    Select No radio button beside Do you want to run RFI and RFQ phases concurrently? if you want to include all the phases in your sourcing event.

  7. Select the Do you want to collect RFI responses from suppliers with no portal access? check box if you want to allow non-portal suppliers to submit their RFI responses through an unauthenticated URL.
  8. Select the Do you want to collect RFQ responses from suppliers with no portal access? check box if you want to allow non-portal suppliers to submit their RFQ responses through an unauthenticated URL.
  9. Select the Progression Mode Type applicable to this sourcing event type. Available options are:

    Option

    Description

    Manual Mode

    Select this event mode type if you want to manually manage your sourcing event. This includes starting and ending each phase by button-clicks.

    Auto Mode

    Select this event mode type if you want to manage your sourcing event automatically. This includes starting and ending each phase, based on the Milestones defined.
  10. Enter Default Description and Default Detailed Description. You can enter the text that will be common to all your sourcing events, to save your time.
    Note:

    Note

    The fields like Default Description, Default Detailed Description, Email Templates and Sourcing Category are auto-populated on Sourcing Event page from Sourcing Event Type.

  11. Under Other Configurations section, provide the requisite information for the following fields.

    Field Name

    Description

    Use Global Email Templates

    Select this check box to use the global email templates that are configured in the Email Templates. For more information on configuring email templates, refer to Configuring Email Templates. Note: If you select this check box, the email templates will be automatically configured below.

    No Email to Suppliers for Any Phase

    Select this check box if you do not want to send any emails to the suppliers for any phase.

    Sourcing Event - Invite Email Template

    Select the template that you want to use to send an email to invite the supplier.

    SE - Invite Thank You Email Template

    Select the template that you want to use to send an email to thank the supplier for participation in the sourcing event.

    SE - Published Email Template

    Select the template that you want to use to send an email to notify the supplier about publishing the event.

    SE - RFI Open For Bidding Email Template

    Select the template that you want to use to send an email to notify the supplier about starting of RFI phase.

    SE - RFI Thank You Email Template

    Select the template that you want to use to send an email to notify the supplier about ending of RFI phase.

    SE - RFQ Open For Bidding Email Template

    Select the template that you want to use to send an email to notify the supplier about starting of RFQ phase.

    SE - RFQ Thank You Email Template

    Select the template that you want to use to send an email to notify the supplier about ending of RFQ phase.

    SE - Award Not Selected Email Template

    Select the template that you want to use to send an email to notify the supplier about loosing the sourcing event.

    SE - Award Winner Email Template

    Select the template that you want to use to send an email to notify the supplier about winning the sourcing event.

    Default Survey Invite Email Template

    Select the default template that you want to use to send an email to invite the supplier.

    Default Scorecard Type

    Select the default scorecard.

    Sourcing Category

    Select the sourcing category. Select General for now.
  12. Click Save.
  13. To edit the sourcing event type, click Edit.

    or

    Click Back to List.

Warning:

You cannot edit a sourcing event type if it is in use.
