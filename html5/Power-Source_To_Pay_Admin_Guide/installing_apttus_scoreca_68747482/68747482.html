Installing Apttus Scorecard

You must install Apttus Scorecard to implement Apttus Supplier Relationship Management. Salesforce provides a simple wizard-led installation process for these packages.

Note:

Caution

Apttus recommends downloading and upgrading Apttus packages into a sandbox before provisioning into a production environment. Please contact Apttus Support before installing if you need assistance.

Parent topic: Source To Pay February 2017 Administrator Guide

Installing the Supplier Scorecard

  • You must have Apttus-provided login credentials.
  • You must have installed Apttus Contract Management (7.2 or higher). To install Apttus Contract Management, refer to Installing Apttus Contract Management.
  1. Go to the Install Center tab on Apttus Community Portal.
  2. Navigate to Apttus Scorecard product and click Get It Now.
  3. Click Log in to the AppExchange and use your Salesforce credentials to log in.
  4. Click the button for environment (production or sandbox) where you want to install the package.
  5. Select the I have read and agree to the terms and conditions check box and click Confirm and Install!. You may have to log in to Salesforce again with your credentials.
  6. On the Upgrade page, type the Apttus provided password.
  7. Verify the Package Installation Details and click Upgrade.
  8. Review the package API access and click Next.
  9. Select the security level. If you know the required settings, select Select Security Settings. Otherwise, we recommend you select Grant Access to all Users and refine the security settings later.
    • Grant access to admins only
    • Grant access to all users
    • Select security settings
  10. Note:
    Click Next.
  11. Note:
    Click Install.
