Managing Master Scorecards

Master Scorecard is a statistical report that keeps track of quantitative and qualitative performance of all your suppliers. You can set up universal scorecards that can be used through out all your suppliers to measure all types of metrics using a single scorecard. The scorecards consists of the supplier information and score metrics. Score Metrics are the categories and parameters by which you want to evaluate your suppliers. These scorecards are then sent to the scorers. Apttus allows you to select the period of validity for a scorecard. You can set the deadline for evaluation of scorecards as well. You can also assign specific categories to specific scorers for evaluation. You can use the existing Scorecard Types or create a new one.

Master Scorecards enable you to evaluate the performance of your suppliers and thus decide their compliance based on the results you have received. Master Scorecards help you analyze the performance of suppliers in a user-friendly way.

You can evaluate your suppliers after they have onboarded, after receiving supplier responses of the surveys that you have sent. Scorecards can also be sent during the on-going sourcing event, where you can evaluate the supplier on basis of the RFI and RFQ.

Apttus allows you to create two types of scorecards:

  • One-Time Master Scorecard - You can send this scorecards to your scorers for all your suppliers for once. These are then saved in our system. You can not reuse the same scorecard.
  • Recurring Master Scorecard - You can send this scorecards to your scorers for all your suppliers based on the frequency that you decide. For example, daily, weekly, monthly, yearly.

Refer to the usecase below to understand which scorecard suits your business requirements.

Tip:

Use Case

There are hundreds of suppliers registering for Tier 1 Mobiles. Sam, Procurement Manager, ensures that relevant evaluation is done for each supplier. The new suppliers that register, are to be evaluated once, for an overall evaluation. Whereas, the existing suppliers must be evaluated monthly to ensure that their quality and performance is up to the mark.

Sam creates a One-Time Master Scorecard for the new suppliers and a Recurring Master Scorecard for existing suppliers. This usecase is just for an example. Your business requirements might vary.

To manage a master scorecard, you must perform the following steps:

  • Create a Master Scorecard
  • Publish the Master Scorecard
  • View the Status of Master Scorecard
Parent topic: Source To Pay February 2017 Administrator Guide

To create a master scorecard

  1. Click the Master Scorecards tab.
  2. Click New.
  3. Select the Type of Scorecard. The available options are.

    Option

    Description

    One-Time

    Select this option if you want to assign this scorecard for one-time use to evaluate the supplier. For example, you want to evaluate Global Microphones supplier for once.

    Recurring

    Select this option if you want to assign this scorecard for recurring use to evaluate the supplier. For example, you want to evaluate Global Microphones supplier every month.
  4. Under Type section, complete the following fields.

    Field Name

    Description

    Name

    Enter a descriptive name of your Scorecard.

    Description

    Enter a short description that describes your scorecard.

    Period From

    Select a Starting Date from when the Scorecard will be active.

    Period To

    Select an End Date till when the Scorecard will be active.

    Deadline

    Select a date as a deadline for the Stakeholders to respond within.

    Schedule

    (The fields in the section below are visible only if you select Recurring as Type).

    Frequency

    The frequency by which you want the supplier evaluated.

    For example, in recurring scorecards, select Monthly if you want to evaluate your suppliers every month.

    Schedule

    Enter the schedule which must be followed to evaluate the scorecard.

    For example, Every 4 weeks on: Monday of the week.

    Scorecard Generation From

    Select a date to validate the generation of a scorecard.

    Scorecard Generation To

    Select a date to validate the ending date of generation of a scorecard.
  5. Under Scorecard Details section, click the Score Metric tab.
  6. In Scorecard Type text box, type a few relevant keywords of the scorecard type that you want to add to your master scorecard.
  7. Select the scorecard type that you want to add for your master scorecard.

    -or-

    To create a new scorecard type, click Create New. For more information on creating a new scorecard type, refer to Setting up Scorecard Types.

    After you select the scorecard, score metrics with Weight, Target, and Threshold are displayed below.

  8. To view the formula of a score metric, click View Formula.
    Note:

    Note

    View Formula link is available only if the Source of the survey is an internal or external survey.

    The Formula Builder window displays the formula added for that particular score metric.
  9. Click OK to go back to the Master Scorecard screen.
  10. To add scorers, click the Scorer tab.
  11. Click Add Scorers.
  12. Select the check boxes for the suppliers you want to add as scorers. Click Add.
  13. From the Score Categories picklist menu, select the category that you want to assign to the supplier for scoring for each supplier.
  14. To add suppliers, click the Supplier tab.
  15. Click Add Suppliers.
  16. Select the check box for the suppliers that you want to add.
  17. Select the relevant Contact Name from the Select Here picklist menu, for each supplier.
  18. Click Add.
  19. From the Score Categories picklist menu, select the category that you want to assign to the supplier for scoring.
  20. For a Supplier Name, select Reference Type.
  21. For Reference, click the
    icon to display the references associated the selected supplier. You can search for a reference in Search text box. Click Select for the respective reference that you want to add for your Supplier. Perform this step for all suppliers you have selected.
  22. Click Save.

You have created a master scorecard. You must publish it in order to send out the scorecards to scorers, for evaluation.

To publish a scorecard

  1. Click the Master Scorecards tab.
  2. Select the scorecard that you want to publish.
  3. Click Publish.
  4. Click Yes to publish the scorecard.

    -or-

    Click No to go back and edit the scorecard.

The Status of Scorecard is updated from Inactive to Active.

After you have published the master scorecard, a scorecard pertaining to each supplier is created and sent to each scorer. You can view the Status of the evaluation that is being done using the Master Scorecard.

To view the status of master scorecard

The Status tab is visible only after you publish the Master Scorecard. Status tab contains all the scorecards that are sent for scoring to scorers. The Score Response section displays the Scorer Name, Supplier Name, Score Categories assigned to a scorer, Status of the scorecard, and the link to respective scorecards.

  1. Click the Master Scorecards tab.
  2. Select the master scorecard that you want to view responses of.
  3. Click the Status tab. The scorecard of each supplier pertaining to each scorer is displayed under the Status tab.
  4. For a scorer, click View to view the results of an individual scorecard.
    -or-For a supplier, click View to view the results of Consolidated Scorecards. For more information on consolidated scorecards, refer to Managing Consolidated Scorecards.
