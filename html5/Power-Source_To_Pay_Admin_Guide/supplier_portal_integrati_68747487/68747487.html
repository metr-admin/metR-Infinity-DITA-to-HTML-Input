Supplier Portal Integration

Supplier Portal Integration enables the buyers to integrate the suppliers with the STP system. Basically, after the approval of new supplier requests, the system creates a new account for supplier; supplier gets access to the supplier portal to view the profile information. Create Supplier button is enabled for all the approved suppliers. The suppliers can then access Source-to-Pay modules such as Compliance Surveys and Sourcing Events.

To create a supplier

You must have an approved supplier request and appropriate number of licenses.

  1. Click the Prospective Suppliers tab.
  2. Select the approved supplier that you want to create an account for.
  3. Click Create Supplier.
    Note:

    Note

    You will get a notification about a new user created.

    The supplier gets a notification about his addition to the system as well.
