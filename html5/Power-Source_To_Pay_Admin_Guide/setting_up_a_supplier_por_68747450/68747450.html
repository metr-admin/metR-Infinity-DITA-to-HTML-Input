Setting Up a Supplier Portal

Supplier Portal is a community where you can engage with your employees, customers, and partners. You can access Surveys, Sourcing Events, and Chatter using the Supplier Portal. Your suppliers must have access to the supplier portal. You must set up a community first.

Perform the following steps to setup a supplier portal.

  1. Create a Unique DomainYou must create a unique domain as it is a unique address of your community. Your suppliers will use this domain to open the community. If you've already created a unique domain for your org, you can skip this step.
  2. Create a User ProfileYou must create a custom profile for the suppliers. For demo purpose, we have created a partner community profile. You can create a customer community profile as well. For more information on difference between Customer Community License and Partner Community License, refer to this link.
  3. Enable a CommunityBefore you create a community, you must enable a community in your org.
  4. Create a CommunityYou must create a community so your suppliers can access the sourcing events and surveys.
Note:

Caution

  • If you already have a Community enabled in your Salesforce org, skip Step 1 and Step 3. The pre-requisites of creating a community is to create a unique domain and enabling a community. If you have an existing community, you might have performed these steps earlier.
  • If you already have a domain name set up, skip step 1.

To create a unique domain

  1. Go to Setup > Domain Management > My Domain.
  2. Enter the desired domain name. Ensure that the domain name is unique.
  3. Click Check Availability.
  4. If the domain name is available, select I agree to Terms and Conditions check box and click Register Domain.
    Note:

    Note

    You will be notified once your domain name is published on the internet. You can still use the domain name for testing purposes.

To create a user profile

  1. Go to Manage Users > Profiles and click New Profile.
  2. Select Partner Community User as Existing Profile.
  3. Enter the appropriate Profile Name. You can enter any relevant name and use it accordingly in the system. We are using Test Partner Community User.
  4. Click Save.
  5. Assign Object Permissions to the profile. For more information on object permissions, refer to Profile Settings and Security.

To enable a community

  1. Go to Setup > Customize > Communities > Community Settings and select Enable Communities checkbox.
  2. Enter a unique Domain Name.
  3. Click Check Availability. Confirm that your chosen domain is available.
  4. Click Save.

To create a community

  1. Go to Setup > Customize > Communities > All Communities and click New Community.
  2. Select a Template and click Choose.
    Tip:

    Select Salesforce Tabs + Visualforce to give the Salesforce UI to your community.

  3. Enter appropriate Name and URL of the community.
  4. Click Create Community.
  5. Click Go to Community Management.
  6. Click the Administration tab and the Members tab.
  7. Under Available Profiles, select Test Partner Community User and click .

  9. Click the Tabs tab.
  10. Select Sourcing Events (Portal) and click .
  11. Click Save.
  12. Click the Overview tab and click Activate Community.
