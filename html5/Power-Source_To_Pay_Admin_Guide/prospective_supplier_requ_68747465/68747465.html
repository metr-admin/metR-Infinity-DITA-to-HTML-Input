Prospective Supplier Request

Prospective Supplier Request enables the salesforce internal users to create a new request for the supplier registration. The Prospective Supplier Request form collects the business information, tax related information, products or services, service locations, certifications and financial information. Additionally, the buyers can also fill the surveys that he has attached with the onboarding type. Onboarding Type defines the surveys that a supplier must respond to while onboarding. Onboarding Surveys allows the buyers to collect more information from the suppliers. The buyer must add the onboarding ID in the URL. Suppliers can self-register themselves as well.

Tip:

To obtain an onboarding Id from an onboarding type

You must have an existing community that you use for public RFI.

  1. Click the Onboarding Types tab.
  2. Select the onboarding type that you want to assign to the supplier.
  3. Copy the onboarding type ID, text between = and &.
    For example, in https://c.cs26.visual.force.com/apex/OnboardingType?id=a5r2800000000kW&sfdc.override=1 URL, a5r2800000000kW is the onboarding type ID.
    Warning:

    You must copy the ID of an Onboarding Type that you want to assign to a supplier.

To create a prospective supplier request

  1. Click the Prospective Supplier tab.
  2. Click New.
  3. Append the text &type=[Onboarding ID] as text at the end of URL and press the Enter key.
    Performing this step will append all the surveys attached in the Onboarding type.
  4. On the General Information tab, complete the following fields.

    Field Name

    Description

    Company Name

    Enter Name of your company. This field is mandatory.

    Contact Name

    Enter the Name of the contact person. This field is mandatory.

    Tax ID

    Enter the Tax ID of your company.

    D&B Number

    Phone Number

    Enter the Phone Number of your company. This field is mandatory.

    Fax Number

    Enter the Fax Number of your company.

    Email

    Enter the Email Address of your company. This field is mandatory.

    Website

    Enter the Web Address of your company.

    Address 1

    Enter the Address of your company.

    Address 2

    Enter the Address of your company.

    Country

    Select the Country.

    State

    Select the State.

    City

    Select the City.

    Zip Code

    Enter your Zip Code.

    Product and Service Location

    Industry

    Select the Industry of your company.

    Commodity

    Select the Commodity of your company.

    Region

    Select the Region of your company.

    Comments

    Enter Comments, if any.
  5. Click Upload to attach a document.
  6. Select the file that you want to attach, and click Open.
  7. To delete an existing file, click icon.
  8. Click Next and fill the additional surveys. The additional surveys refer to the surveys attached in onboarding type that you have added to this request earlier.
  9. After you are done responding to surveys, click Cancel to cancel the prospective supplier request.

    -or-

    Click Save Draft to save the supplier request.

    -or-

    Click Submit to submit the supplier request for approval.

The suppliers can self-register themselves as well. To create a supplier self-registration link, refer to Supplier Self-Registration.

After you have created a prospective supplier request, you must submit it for review and approval. For more information on reviewing and approving supplier requests, refer to Supplier Review and Approvals.
