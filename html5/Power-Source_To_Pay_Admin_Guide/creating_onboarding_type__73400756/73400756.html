Creating Onboarding Type

Onboarding Type describes the criteria on basis of which a supplier will be onboarded. You must associate the surveys to onboarding type. The surveys that you associate with the onboarding type will be marked as an entry point for a new suppliers in Supplier Onboarding process.

Tip:

Usecase

Sam must create a variety of onboarding types. An general onboarding type must be created where in all the major surveys must be added. You can gather general information using this general onboarding type. Sam creates the following onboarding types.

Onboarding Types

Surveys

General Onboarding

General Survey

Finance Survey

Legal Survey

Quality Survey

Now, if he wants to receive more information from the suppliers pertaining to their products, he can add them to respective supplier group, which contains the added surveys.

Parent topic: SRM Setup Console

To create an onboarding type

  1. On the SRM Setup Console tab, click Onboarding Type.

    -or-

  2. Click the Onboarding Types tab.
  3. Click New.
  4. Enter a descriptive Onboarding Type Name.
  5. Enter a relevant keyword in the Survey text box to search for the survey that you want to add in the supplier onboarding process and press Enter key to select the supplier you want to add.
  6. Click Save.
