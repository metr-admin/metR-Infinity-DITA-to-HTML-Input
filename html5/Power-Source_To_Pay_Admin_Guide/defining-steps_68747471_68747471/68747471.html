Defining Steps

Steps are used to designate the actions that occur when end-users use the wizard. Steps use common elements, like actions and rules, to define the flow of your wizard. Each step represents a screen in the wizard runtime. You can define the Weight and Score in a step as well.

Parent topic: Using the Wizard Component Library

To add a step

Ensure that you have created one or more input controls to use in the creation of a step.

  1. Click the Wizard Component Library tab.
  2. Click New Step.
  3. Under Step Settings tab, enter a descriptive Step Name.
  4. Complete the following fields.

    Field Name

    Description

    Step Name

    Enter a descriptive Step Name. This field is mandatory. You cannot drag-and-drop input values from the Inputs Library to the Inputs pane.

    Description

    Enter a Description that describes the input control.

    Keywords

    Enter the Keywords that would help find this input control.

    Category

    Enter the Category of this input control. The default value is Default. Defining a Category helps you organize your questions when creating input control.

    Subcategory

    Enter the SubCategory of this input control. The default value is Default. Defining a SubCategory helps you organize your questions when creating input control.

    Banner Statement

    Enter the statement to be displayed on surveys.

    Runtime Help Text

    Enter the Runtime Help Text that will be displayed to the users when they hover over the icon.

    Context Type

    Select the Context Type to which this step will be applicable to. The available options are.
    • All – Select this option if you want to apply this step to all the objects.
    • Object – Select this option if you want to apply this step to any specific object.

    Context Object

    Select the object that you want to apply this step to. This option will be enabled only if you have selected Object as the Context Type.

    Input Group Option

    Select the Input Group Option for the survey. The available options are.
    • Repeatable – Select this option if you want to display all the selected input controls in a repeated manner, for each item, in the survey.
    • Table Layout – Select this option if you want to display all the selected input controls in a repeated manner in a tabular form, for each item, in the survey.

    Inputs

    Drag all the inputs from Inputs Library to this section to include the input controls in your survey.

    Inputs Library

    A repository of all the input controls that you have added to the system.
  5. To add a new input, click New Input. Clicking this button will create a Step-specific Input which will not be saved in the Inputs Library or be reusable for future Wizards.
  6. Complete the fields as mentioned in step 4 of To add an input control.
  7. Click Save and Use in Step.
  8. You can use existing input controls as well. Drag the input controls, to the Inputs section, that you want to add in your survey from Inputs Library.
  9. On the Input Rules tab, set the rule for each step.
  10. In order to set the weight of a Step and score for each option, you must set the custom value for each input.
    1. On the Input Rules tab, click .
    2. From the Rule picklist menu, select Set Custom Value.
    3. To define the value of the weight of a question, select Weight from the Field picklist menu.
    4. In the Value text box, enter the value of weight for that particular question.
    5. Click Save.
    6. To define the value of the score of an option, click .
    7. From the Rule picklist menu, select Set Custom Value.
    8. Select Score from Field picklist menu.
    9. In the Value text box, enter the value of score of an option.
    10. Click Add Another Rule.
    11. Select the Rule for which you are entering scores.
    12. Select a relevant Operator.
    13. Enter the Value of the option for which you want to set that particular score.
    14. Click Save.
      Warning:

      You must define Weight and Score for each rule. Steps a through e are applicable for defining Weight of a Step. Steps f through n are applicable for defining score of an option. A new Input Rule must be defined for each option of a step.

  11. Click the Step Preview tab, to view the preview of the entire survey.
  12. Click Save.
