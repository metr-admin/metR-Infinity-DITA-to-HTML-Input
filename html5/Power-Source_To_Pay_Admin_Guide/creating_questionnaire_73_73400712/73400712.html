Creating Questionnaire

A Questionnaire consists of a series of questions for the purpose of gathering information from respondents. You can create different questionnaires based on categories and then club them in your RFI as well. The main purpose of utilizing a questionnaire is to create re-usable questions that can be utilized across multiple events. The buyer can create a questionnaire and add all the appropriate questions to that questionnaire. The questions can be of any topic, for example, general information of the company, finance, production, quote and many more. Basically, each and every question that you want to ask to the supplier goes in the questionnaire.

In our example, Sam can create a questionnaire with topics like General, Finance, Production, Legal, Safety, Quality and many more.

Note:

Note

The Questionnaire that you create here can be used as a part of RFI in sourcing event. If you want to create surveys, refer to Post-Installation Configuration.

Parent topic: SRM Setup Console

To create a questionnaire

  1. On the SRM Setup Console tab, click Questionnaire.

    -or-

    Click the Questionnaire tab.

  2. Click New.
  3. Enter the Questionnaire Name and Description.
  4. Click Save.
  5. Click Add Question. Add Question button will be visible once you click Save.
  6. Complete the following fields.

    Field Name

    Description

    Question

    Enter the statement of the question to be asked.

    Response Type

    Select the response type of the question. Available options are: Radio Button, Pick List, Text Box.

    Existing Topic

    Select the topic under which you want to add particular question to.

    New Topic

    Enter the name of the new topic. Note: This field will be visible if you select None in Existing Topic drop down menu.
  7. Enter the statement or an option of response in Response.
  8. Select the appropriate score of the response from Score drop down.
  9. Click Add to Response List.
  10. Click Add Question to add this question to the questionnaire.
  11. Click Save.
  12. You can enter a sequence number in the Sequence text box, to re-arrange the questions in a questionnaire.
  13. You can edit the Score of the responses from the View mode only.
  14. Click Save.
  15. Click Close to go back to the Questionnaire list page.
