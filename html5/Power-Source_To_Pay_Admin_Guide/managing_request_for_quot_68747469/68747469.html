Managing Request for Quotation

Request for Quotation (RFQ) is for collecting the pricing quotes from the suppliers for the products or services that the buyer wants to procure.

A request for quotation (RFQ) is used when discussions with suppliers are not required (mainly when the specifications of a product or service are already known) and when price is the main or only factor in selecting the successful supplier. An RFQ may also be used prior to issuing a full-blown RFP to determine general price ranges. In this scenario, products, services or suppliers may be selected from the RFQ results to bring in further research in order to write a more fully fleshed out RFP. RFQ can be closed and reopened as well. The Stakeholders are notified about the same. A notification is sent to the suppliers. You can add up to 2000 products in an RFQ Template.

RFQ Template enables you to add products in an RFQ. You can later re-use the template, add some more products, and save it as a new RFQ Template as well.

To create an RFQ

  1. Click the Sourcing Events tab.
  2. Select the appropriate sourcing event and clickEdit.
  3. Click the Preparation tab and click the RFQ tab.
  4. Click Create.

To add an existing RFQ template

  1. Click the Sourcing Events tab.
  2. Select the appropriate sourcing event and click Edit.
  3. Click the Preparation tab and click the RFQ tab.
  4. Click View.
  5. Under RFQ Catalog section, click Search & Add.
  6. Type in the keywords to search a template in the search box. -or-Click
    to display all existing RFQ templates.
  7. Select the check box beside the template that you want to add.
  8. Click Save to add the selected RFQ template(s).

To create a new RFQ template

  1. Click the Sourcing Events tab.
  2. Select the appropriate sourcing event and click Edit.
  3. Click the Preparation tab and click the RFQ tab.
  4. Click View.
  5. Under RFQ Catalog section, click Create New.
  6. On the RFQ Catalog window, select Group radio button to add a group.
  7. Enter Group Name and Description.
  8. Select an existing group from Parent Group picklist menu.
    Note:

    Note

    Perform this step only if you want to create a sub-group of a group.

  9. After you have added a group, on the RFQ Catalog window, select Item radio button to add an item to the existing group.
  10. Complete the following fields as described in the table below.

    Field Name

    Description

    Name

    Enter the Name of Item.

    SKU Item

    Select the SKU Item from the look-up. For more information on how to add SKU Item, refer to Adding SKU Item.

    Description

    Enter the description.

    Group

    Select the Group from the drop-down menu.

    Quantity

    Enter the Quantity.

    UOM

    Select a Unit of Measurement from the drop-down menu.

    Period

    Enter the frequency of delivering the products.

    Commodity

    Select the Commodity from look-up.

    Purchase Request

    Select the Purchase Request from look-up.

    Comments

    Enter your comments, if any.

    Response Required

    Select this check box if you want to ensure that supplier responds to this item.
  11. Click Save.
  12. To save your products as a new template, click Save as New RFQ Template.
  13. Enter the RFQ Template Name and Description for the new template.
  14. Click Upload to upload an attachment to the sourcing event.
    Note:

    Note

    For the Attachments in Apttus STP, you can upload any file. We allow files of the following extension types: (.PDF), (.RTF), (.RTX), (.TXT), (.JPG), (.JPEG), (.PNG), (.GIF), (.TIF), (.BMP), (.DOC), (.DOCX), (.DOCM), (.PPT), (.PPTX), (.PPTM), (.PPS), (.PPSX), (.XLS), (.XLSX), (.XLSM), (.ZIP), (.MP3), (.M4A), (.OGG), (.WAV), (.MP4), (.M4V), (.MOV), (.OGV), (.3GP), (.3G2). The application might become slow if you upload a file higher than 25 MB.

  15. Click Choose File and select the appropriate file.
  16. Enter the File Description.
  17. Select the appropriate permission from the Permissions picklist menu.

    Permission

    Description

    Public

    To make the attachment available to all the Suppliers.

    Private

    To make the attachment available to only the Buyer.

    Selective Public

    To make the attachment available to the selected Supplier(s).

    Note: At least one supplier must be selected for this option.
  18. Click Attach.
  19. To edit an attachment, click Edit.
  20. To delete an attachment, click Delete.
  21. To download the RFQ Catalog, click Download.
  22. Click Save.
