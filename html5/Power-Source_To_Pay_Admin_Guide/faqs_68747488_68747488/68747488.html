FAQs

This section contains frequently asked questions. It also provides the solution to frequently occurring errors. This FAQs section is for buyers.

Can I add suppliers, to the sourcing event, who do not have portal access?

Yes, you can add suppliers who don't have portal access to the sourcing event. The event is automatically accepted on behalf of these suppliers.

Can I accept the event invitation on behalf of a supplier?

Yes, you can accept the event invitation on behalf of the suppliers that have portal access. For the suppliers who do not have portal access, the event is automatically accepted by the system.

To accept the invitation on behalf of suppliers,

  1. Click the Sourcing Events tab.
  2. Select the sourcing event that you want to edit.
  3. Click the Invitation tab.
  4. For the supplier that you want to accept the event, click Accept.

Can I create RFI response on behalf of supplier?

Yes, you can create the RFI response on behalf of any supplier.

To create a RFI response in behalf of suppliers,

  1. Click the Sourcing Events tab.
  2. Select the sourcing event that you want to edit.
  3. Click the Responses tab.
  4. Click the RFI tab.
  5. Click Create Response button.
  6. Select the supplier for whom you are creating RFI response from the Supplier picklist.
  7. Fill in the responses as prompted by the supplier and click Save.

Can I create RFQ response on behalf of supplier?

Yes, you can create the RFQ response on behalf of any supplier.

To create a RFQ response in behalf of suppliers,

  1. Click the Sourcing Events tab.
  2. Select the sourcing event that you want to edit.
  3. Click the Responses tab.
  4. Click the RFQ tab.
  5. Click Create Response.
  6. Select the supplier for whom you are creating RFQ response from the Supplier picklist.
  7. Fill in the responses as prompted by the supplier and click Save.

Why can't I shortlist any supplier?

You can shortlist any supplier only after the RFI/RFQ deadline has passed.

For example, if the RFI deadline is October 16,2015 , you will be able to shortlist or reject the RFI/RFQ responses only after October 16.

Can I interact with any supplier before starting the RFI phase?

Yes, you can interact with any supplier as soon as you publish the event, even if any other phase hasn't started yet.

Can I attach a file for any item in RFQ Catalog?

Yes, you can attach a file for any particular item in RFQ Catalog.

To attach a file to any item in RFQ Catalog,

  1. Click the Sourcing Events tab.
  2. Select the sourcing event that you want to edit.
  3. Click the Preparation tab.
  4. Click the RFQ tab.
  5. Click View.
  6. Add any item and scroll down to that item.
  7. To attach a file to that item, click .
  8. Upload the appropriate file and enter the required details.
  9. Click Attach.
  10. Click Save.
