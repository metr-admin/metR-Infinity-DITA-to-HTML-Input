Note:

Caution

Apttus recommends downloading and upgrading Apttus packages in a Salesforce sandbox before installing them in your production environment. For information on installing and upgrading in a sandbox, please contact Apttus Support before you install any packages.

To install Apttus STP packages

  1. Go to the Install Center tab on the Apttus Community Portal.
  2. Click the Supplier Relationship Management tab.
  3. Select the version that you want to install from the Version drop down menu.
  4. Click Install Now.
  5. Select the environment in which you want to install the packages.
    • Click Install in Production to install the packages in your production org.
    • Click Install in Sandbox to install the packages in your sandbox org.
  6. Enter your Salesforce Credentials and click Log In.
  7. Enter the Password provided by Apttus.
  8. Select the Security Level. If you know the required settings, select the third option, otherwise it is recommended you select the second option and refine the security settings as required later.
    • Grant access to admins only
    • Grant access to all users
    • Select security settings
  9. Click Upgrade. You will receive an email when the installation of packages is successful.
