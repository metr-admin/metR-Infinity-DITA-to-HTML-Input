Activity Tracker

Activity Tracker enables you to track the activity history related to the sourcing event and all suppliers. You can track activity that was performed, the date on which that activity was performed, the user who performed that particular activity, the object and field name on which the activity was performed. Activity Tracker comes handy when you have more than one buyer handling the same sourcing event.

To access activity tracker

  1. Click the Sourcing Events tab.
  2. Select the appropriate sourcing event.
  3. Click the Activity Tracker tab. This displays all activities that have been performed by all buyers and suppliers.
  4. You can filter the activities by selecting appropriate options from the following filters. Select the appropriate date from From: field.
  5. Select the appropriate date from To field.
  6. Select the appropriate object from Object picklist.
  7. Select the appropriate supplier from the Supplier picklist.
  8. Select the type of participant from Participant Type picklist. Available options are All, Internal Participant (Buyers / Stakeholders), and External Participant (Suppliers).
  9. Click Go.
