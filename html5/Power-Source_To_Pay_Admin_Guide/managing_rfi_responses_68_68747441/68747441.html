Managing RFI Responses

RFI Responses are the responses sent by the suppliers to the buyers in response to the RFI sent by the buyer. RFI Responses allows the buyer to view the RFI responses submitted by the suppliers. RFI can contain all the questions that you would like to get information for, from suppliers. RFI includes questions related to general information, financial information, safety information, quality information, and much more.

The buyer can do the following tasks with RFI Responses.

  • Create RFI Response , to create the response on behalf of a supplier if the supplier has responded by any other means.
  • Compare RFI Response , to compare the responses provided by suppliers
  • Download RFI Response , to download the responses provided by suppliers
Parent topic: Managing the Responses

To create RFI response

  1. Click the Sourcing Events tab.
  2. Select the appropriate sourcing event and click Edit.
  3. Click Start RFI.
    Note:

    Note

    Skip this step if you have selected Auto Mode as your Event Mode Type in your sourcing event type. For Auto Mode, the phase progression is carried out automatically, based on the date and time mentioned in Milestones.

  4. Click the Responses tab and click the RFI tab.
  5. Click Create Response.
  6. Select the appropriate supplier from the Supplier drop-down menu.
  7. Complete the responses as mentioned by the supplier.
  8. To upload an attachment, click Upload.
  9. Click Choose File and select the file that you want to upload.
  10. Enter the File Name and Description.
  11. Click Attach.
  12. Click Save.
  13. To edit the response, click Edit.

    or

    Click Back to Sourcing Event.

To compare RFI response

  1. Click the Sourcing Events tab.
  2. Select the appropriate sourcing event and click Edit.
  3. Click the Responses tab and click the RFI tab.
  4. Click Compare.
  5. Click View to view the individual RFI responses.
  6. To edit any existing response, select the option that is relevant and click Save.
  7. Select one option from Set As drop down menu as per the following table. If there are more than two suppliers, scroll to the right to access more suppliers.

    Option

    Description

    New

    Selected by default.

    Reviewed

    Select Reviewed if you have reviewed the RFI response.

    Shortlisted

    Select Shortlisted if you have shortlisted this supplier.

    Declined

    Select Declined if you have declined this supplier.
    Note:

    Note

    Set As drop-down menu is visible only after the completion of RFI End Date (Milestones tab).

  8. Click Download to download the comparison of all RFIs.
    Note:

    Note

    The Buyer Response Status will change according to the selected option in the Set Status drop down menu.

  9. Click Complete RFI.
    Note:

    Note

    A Phase Progression pop-up is displayed for the confirmation of completing the phase. Click Yes to proceed to next phase. Click No to continue working in the same phase.

To download RFI response

  1. Click the Sourcing Events tab.
  2. Select the appropriate sourcing event and click Edit.
  3. Click the Responses tab and click the RFI tab.
  4. Click Download to download the responses of all the suppliers visible in RFI Responses section.
    Note:

    Note

    The RFI Responses.xls file will start downloading. To view the downloaded file, click RFI Responses.xls .

    In order to download a response of any individual supplier, click View for the supplier you want to download the response, and then click Download.
