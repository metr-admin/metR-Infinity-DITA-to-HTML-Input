Adding Scores to Scorecards

A Scorecard is a measure used to rate a supplier, company, customer, process, or vision. A Scorecard is also used for scoring the Key Performance Indicator (Score Metrics) for a given business.

A Scorecard is a module of Supplier Relationship Manager that enables organizations to effectively manage supplier relationships, including vendor onboarding, agreements, spend, purchase orders, vendor scoring, business reviews, issue tracking, and comprehensive relationship dashboards.

As a Supplier Manager user, you can track all supplier engagement activities including

  • Vendor Scoring
  • Contract process,
  • Supplier Performance,
  • Supplier Spend
  • Issue Tracking.
  • Activities related to suppliers
  • Spend Visibility
  • Calls, emails, and calendar events.

You can also access pre-configured reports and dashboards.

To evaluate performance, you can define various categories you will use to evaluate the performance. A Scorecard provides you the capability and flexibility to define how you wish to measure the metrics.

After you have configured the scorecards, you can publish them. After publishing the scorecards, the scorecards are sent to respective stakeholder for scoring. You must enter scores in the scorecard to complete the scorecard.

In a scorecard, the Information section contains all the basic information pertaining to that scorecard. The Information section includes the following fields:

Field

Description

Survey Response ID

Displays the system generated Survey Response ID.

Survey Name

Displays the name of the Survey entered at the time of creating a survey.

Description

Displays the Description entered at the time of creating a survey.

Survey From

Displays the date since which the survey is active.

Survey To

Displays the date till which the survey will be valid.

Deadline

Displays the deadline till which stakeholders must evaluate the scorecard.

Status

Displays the current status of the scorecard.

Supplier

Displays the name of the supplier to which this scorecard belongs to.

Submitted By

Displays the name of the user who evaluated this scorecard.

Submitted Date

Displays the date when the scorecard was submitted.

Last Modified Date

Displays the date when the scorecard was modified.

Last Modified by ID

Displays the name of the user who modified the scorecard.
Note:

Note

A Guest User can add scores to a scorecard only once. However, an Internal User can update the scorecard multiple times within the mentioned scoring time frame (deadline).

To add scores to scorecard

  1. Click the Master Scorecards tab and select the master scorecard that contains the scorecard you want to score.

    -or-

    Click the Link that you have received in your email.

  2. Click the Status tab.
  3. Click View for the individual scorecard that you want to evaluate.
  4. Click Evaluate.
  5. Enter the scores for every Score Metric under the Score column.
  6. Click Save to save your current work. The status will be updated to Pending.
  7. Click Submit Scores to submit your scorecard. The status will be updated to Completed after you complete the entire scorecard. A complete scorecard with ratings and radar chart is displayed as the below image.
