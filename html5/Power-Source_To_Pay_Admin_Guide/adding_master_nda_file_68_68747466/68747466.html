Adding Master NDA file

Master NDA is an agreement between the buyer and supplier. It is essential to add Master NDA file for all the suppliers. You can configure the Event NDA file from the sourcing event itself. The suppliers can view the Master NDA once they open the sourcing event.

The following steps must be performed in order to add a Master NDA file to the system.

  • Create a Sourcing Doucment
  • To attach a Master NDA file
Parent topic: Appendices

To create a sourcing document

  1. Click the Accounts tab.
  2. Select the supplier for whom you want to add the Master NDA file for.
  3. Under the Supplier Documents related list, click New Supplier Document.
    Tip:

    If you are not able to view Supporting Documents related list,

    1. Click Edit Layout on the right corner.

    2. Click Related Lists.

    3. Search for Supplier Documents and drag it to the Related Lists section.

    4. Click Save.

  4. Enter a descriptive Document Name.
  5. Select Master NDA as the Category.
    Tip:

    Tip

    If you are not able to view Master NDA in Category picklist, 1. Go to Setup > Build > Create > Objects and select Supplier Document. 2. Under Custom Fields & Relationships, select Category. 3. Under Picklist Values, click New. 4. Enter Master NDA as text. 5. Click Save.
  6. Enter the Valid From and Valid To dates.
  7. Select Active check box.
  8. Click Save.

To attach a Master NDA file

  1. Select the supplier document name that you just created.
    Tip:

    Tip

    If you are not able to view Supporting Documents related list,

    1. Click Edit Layout on the right corner.

    2. Click Related Lists.

    3. Search for Notes & Attachments and drag it to the Related Lists section.

    4. Click Save.

  2. Under the Notes & Attachments section, click Attach File.
  3. Click Choose File and select the file that you want to upload.
  4. Click Attach File.
  5. Once the file is attached, click Done.
