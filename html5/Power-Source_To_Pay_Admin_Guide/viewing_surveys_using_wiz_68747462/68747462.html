Viewing Surveys using Wizards

You can preview the surveys that you have created using the Wizards tab. This will give you an preview of how will your survey look like in the runtime.

Parent topic: Configurable Wizard

To view a survey using wizard tab

  1. Click the Wizards tab.
  2. Select the survey that you want to preview from the Wizard Design picklist menu.
  3. Click Go. Another tab containing the Wizard Runtime opens.
