Creating Surveys using Wizard Designs

Parent topic: Configurable Wizard

Creating Surveys

A Survey is a methodology of collecting information from the suppliers using forms. In STP, the buyers assign the surveys to suppliers. Suppliers then respond to those surveys. The approvers then evaluate suppliers based on responses to the surveys. The buyers or admins can create the surveys that are to be assigned to suppliers.

When you create surveys using Wizard Component Library, you can reuse the input controls and steps that you have created. If you create surveys using Wizard Designer, you can not re-use the steps in other surveys.

To create a survey using wizard designer

  1. Click the Wizard Designs tab.
  2. Click New Wizard Design.
  3. On the Wizard Settings tab, enter a descriptive Designer Name.
  4. Complete the following fields.

    Field Name

    Description

    Design Name

    Enter a descriptive Design Name. This field is mandatory. You cannot drag-and-drop steps from the Steps Library to the Steps pane.

    Description

    Enter a Description that describes the input control.

    Keywords

    Enter the Keywords that would help find this input control.

    Category

    Enter the Category of this input control. The default value is Default.

    Subcategory

    Enter the SubCategory of this input control. The default value is Default.

    Banner Statement

    Enter the statement to be displayed on surveys.

    Context Type

    Select the Context Type to which this step will be applicable to. The available options are.
    • None – Select this option if you want to apply this step to all the surveys.
    • All – Select this option if you want to apply this step to all the objects.
    • Object – Select this option if you want to apply this step to any specific object.

    Context Object

    Select the object that you want to apply this step to. This option will be enabled only if you have selected Object as the Context Type.

    Allow Review Edit

    Select this check box to edit the survey while reviewing.
  5. To add any existing step to the survey, drag the step that you want to add in the survey to Steps in Wizard section, from Steps Library.

    -or-

    To add a new step, click New Step. Clicking this button will create a Wizard-specific step which will not be saved in the Steps Library or be reusable for future Wizards.

  6. On Step Settings tab, complete the following fields.

    Field Name

    Description

    Step Name

    Enter a descriptive Step Name.

    Description

    Enter a Description that describes the input control.

    Keywords

    Enter the Keywords that would help find this input control.

    Category

    Enter the Category of this input control. The default value is Default.

    Subcategory

    Enter the SubCategory of this input control. The default value is Default.

    Banner Statement

    Enter the statement to be displayed on surveys.

    Runtime Help Text

    Enter the Runtime Help Text that will be displayed to the users when they hover over the icon.

    Context Type

    Select the Context Type to which this step will be applicable to. The available options are.
    • None – Select this option if you want to apply this step to all the surveys.
    • All – Select this option if you want to apply this step to all the objects.
    • Object – Select this option if you want to apply this step to any specific object.

    Context Object

    Select the object that you want to apply this step to. This option will be enabled only if you have selected Object as the Context Type.

    Input Group Option

    Select the Input Group Option for the survey. The available options are.
    • None – Select this option if you want to display all the selected input controls for a single time in the survey.
    • Repeatable – Select this option if you want to display all the selected input controls in a repeated manner, for each item, in the survey.
    • Table Layout – Select this option if you want to display all the selected input controls in a repeated manner in a tabular form, for each item, in the survey.

    Inputs

    Drag all the inputs from Inputs Library to this section to include the input controls in your survey.

    Inputs Library

    A repository of all the input controls that you have added to the system.
  7. To add a new input, click New Input.
  8. On Input Settings tab, complete the following fields.

    Field Name

    Description

    Control Name

    Enter a descriptive Control Name.

    Description

    Enter a Description that describes the input control.

    Keywords

    Enter the Keywords that would help find this input control.

    Category

    Enter the Category of this input control. The default value is Default.

    SubCategory

    Enter the SubCategory of this input control. The default value is Default.

    Question/Instruction

    Enter the Question or Instruction to be asked to the suppliers.

    Runtime Help Text

    Enter the Runtime Help Text that will be displayed to the users when they hover over the icon.

    Runtime Input

    Select the Runtime Input of this input control. The available options are.
    • Required – Select this check box if you want to set your question as mandatory for the suppliers to respond to.
    • Read Only – Select this check box if you want to set your question as read only.

    Field Class

    Select the Field Class of this input control. The available options are.
    • Wizard Input Field – Select this option if you want to enter the field type in the wizard itself.
    • Object Field – Select this option if you want to refer to the field type of an existing field.

    Input Control - Field Type

    Select the Field Type for this input control. The available options are.
    • Checkbox
    • Date Number
    • Picklist
    • Radio Button Picklist
    • Multi Picklist
    • Text
    • Text Area

    Input Control - Picklist Values

    Enter the values to be displayed to the supplier, separated in each line. This option is visible only if you select
  9. Click the Preview tab to view the preview of your input control.
  10. Click Save and Use in Step. This input control will not be saved in Inputs Library.
  11. Drag the inputs that you want to add in your step to the Inputs section, from Inputs Library.
  12. On the Input Rules tab, set the rule for each step.
  13. In order to set the weight of a Step and score for each option, you must set the custom value for each input.
    1. On the Input Rules tab, click .
    2. From the Rule picklist menu, select Set Custom Value.
    3. To define the value of the weight of a question, select Weight from the Field picklist menu.
    4. In the Value text box, enter the value of weight for that particular question.
    5. Click Save.
    6. To define the value of the score of an option, click .
    7. From the Rule picklist menu, select Set Custom Value.
    8. Select Score from Field picklist menu.
    9. In the Value text box, enter the value of score of an option.
    10. Click Add Another Rule.
    11. Select the Rule for which you are entering scores.
    12. Select a relevant Operator.
    13. Enter the Value of the option for which you want to set that particular score.
    14. Click Save.
      Warning:

      You must define Weight and Score for each rule. Steps a through e are applicable for defining Weight of a Step. Steps f through n are applicable for definingscore of an option. A new Input Rule must be defined for each option of a step.

To clone surveys using wizard designer

  1. Click the Wizard Design tab.
  2. Under the Actions column, click the Clone link for the wizard that you want to clone.
  3. In the Clone Wizard Design form, enter a unique design name for the new wizard in the Wizard Design Name field.
  4. Click outside the field and then click Clone. The new Wizard design is created and displayed in the Wizard Design list on a new line with the Draft status. All information is copied from the cloned Wizard.
