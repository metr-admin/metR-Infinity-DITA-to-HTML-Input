Adding the Milestones

Milestones are estimated timelines in which the sourcing event would start or end in new phases. You must define the milestones for each phase of sourcing event by entering the dates (or deadlines). Milestones are defined to maintain the schedule of the sourcing event. You must define the Milestones before publishing the event. The suppliers will be notified once the event is published.

If you have selected Auto Mode as Event Mode Type in the sourcing event type, the entire sourcing event is driven by the timelines that you set in Milestones. You cannot edit any date or time once you have completed the Preparation phase. By default, the date and time are populated in the Milestones for each phase with a time frame of 60 minutes on the same day. You can set the Milestones as per your business requirements. You cannot edit the Preparation Start Date field. You or the suppliers cannot submit the responses after the deadline after RFI or RFQ deadline.

Tip:

Use Case

Sam sets the deadlines for all phases of the sourcing event. The suppliers are not allowed to submit their responses after RFI or RFQ deadline. Also, Sam cannot create a response or submit it after the RFI or RFQ deadline. In a nutshell, the sourcing event is held under strict deadlines.

To add a milestone

  1. Click the Sourcing Events tab.
  2. Select the sourcing event you want to edit and click Edit.
  3. Click the Preparation tab and click the Milestones tab.
  4. Select the relevant date and enter Time.

    Field Name

    Description

    Open Date

    Auto-populated with the date the event is created.

    Preparation Start Date

    Auto-populated with the date and time the event is created.

    Preparation End Date

    Enter the date and time when you want to end the Preparation phase.

    Publish Date

    Enter the date and time when you want to publish the sourcing event.

    Invitation Acceptance Deadline

    Enter the date and time before which the suppliers must accept your invitation.

    RFI Start Date

    Enter the date and time when you want to start the RFI phase.

    RFI End Date

    Enter the date and time when you want to end the RFI phase.

    RFQ Start Date

    Enter the date and time when you want to start the RFQ phase.

    RFQ End Date

    Enter the date and time when you want to end the RFQ phase.

    Internal Survey Start Date

    Enter the date and time when you want to start the RFQ phase.

    Internal Survey End Date

    Enter the date and time when you want to end the RFQ phase.

    Award Start Date

    Enter the date and time when you want to start the RFQ phase.

    Award End Date

    Enter the date and time when you want to end the RFQ phase.

    Event End

    Enter the date and time when you want to end sourcing event.
  5. Click Save.
