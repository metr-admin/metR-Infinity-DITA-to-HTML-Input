Custom Settings for Activity Tracker

Activity Tracker enables you to track the history of your sourcing events. History related to Sourcing Event, Bid Response, and RFI Bid Responses can be tracked using the Activity Tracker. For more information on activity tracker, refer to Activity Tracker.

You must activate the history tracking in order to view the history. You must perform the following custom settings to activate the activity tracker.

Parent topic: Post-Installation Configuration

To activate activity tracker

  1. Go to Setup > Build > Create > Objects and select Sourcing Event. (Label: Sourcing Event, Installed Package: Apttus Supplier Relationship Management).
  2. Under the Custom Fields & Relationships section, click Set History Tracking.
  3. Select Event Status, Phase, and Phase Status check-boxes.
  4. Click Save.
  5. Go to Setup > Build > Create > Objects and select Bid Response. (Label: Bid Response, Installed Package: Apttus Supplier Relationship Management).
  6. Under the Custom Fields & Relationships section, click Set History Tracking.
  7. Select Supplier Response Status check-box.
  8. Click Save.
  9. Go to Setup > Build > Create > Objects and select RFI Bid Response. (Label: RFI Bid Response, Installed Package: Apttus Supplier Relationship Management).
  10. Under the Custom Fields & Relationships section, click Set History Tracking.
  11. Select Supplier Response Status check-box.
  12. Click Save.
