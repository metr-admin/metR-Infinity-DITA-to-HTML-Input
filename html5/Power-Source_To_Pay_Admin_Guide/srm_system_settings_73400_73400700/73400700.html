SRM System Settings

SRM System Settings consists of the basic settings required for the system. These include Sourcing Event Settings for templates, score scale, and community URL; Supplier Onboarding Settings for Default Supplier Profile, Default Onboarding Type, and Captcha Settings; Survey Manager Settings for templates.

Parent topic: SRM Setup Console

To configure SRM system settings

  1. Click the SRM Setup Console tab.
  2. Click SRM System Settings.
  3. Under Sourcing Event Settings section, select the relevant templates. If you have not created any custom template, then you can select the templates in the image below. Ensure that you select the same templates as shown in the image.
  4. Select the Display Terms and Condition check box if you want your suppliers to accept the terms and conditions of your organization when they log in to the supplier portal for the first time. You must add your terms and conditions in your org. For more information on uploading Terms and Conditions in Salesforce org, refer to Adding Terms and Conditions Page on Supplier Portal.
  5. In the Terms & Condition Static Resource text box, enter the name of your static resource that includes the file of Terms and Conditions that you want to set for your suppliers.
  6. In the Public RFI Welcome Message text box, enter the name of your static resource that includes the file of RFI Welcome Page. For more information on adding the content of RFI Welcome page, refer to Adding Terms and Conditions Page on Supplier Portal.
  7. Under Supplier Onboarding Settings section, complete the following fields.

    Field Name

    Description

    Captcha URL

    Enter the Captcha URL for the captcha program to be used. Enter https://www.google.com/recaptcha/api/ as text in the Captcha URL text box.

    Tip:

    Refer to https://developer.salesforce.com/page/Adding_CAPTCHA_to_Force.com_Sites page to configure your Captcha program.

    Captcha Public Key

    Enter the Captcha Public Key of the captcha program to be used.

    Captcha Private Key

    Enter the Captcha Private Key of the captcha program to be used.

    Default Supplier Profile

    Enter the Default Supplier Profile that you want to set for the new suppliers added in the system. Note: We are using Suppliers Profile for demo purpose only. You must enter the name of the profile that you want to assign to your new suppliers.

    Default Onboarding Type

    Enter the Default Onboarding Type that you want to set for the new suppliers that perform self-registration.

    Note: We are using Default Onboarding for new suppliers for demo purpose only. You must enter the name of onboarding type that you want to assign to your new suppliers.

    Supplier Onboarding Survey Email

    Select the template to be sent to the supplier for onboarding survey.
  8. Under Survey Manager Settings, select the templates as shown in the following image.
    Tip:

    Use Case

    Sam is a Survey Manager, who wants to send the survey invite emails to both portal and non-portal suppliers. He wants to maintain consistency in the email template for all the suppliers. To achieve this:

    1. Go to SRM Setup Console> SRM System Settings

    2. In the Survey Manager Settings section select the same email template from the picklist for both Survey Invite- Supplier Email Template and Survey Invite- Portal Email Template fields.

    This ensures that all the suppliers shall get emails in the uniform format.

  9. Select the Create Task On Survey if you want to create a task for your ursers when you assign a survey to them.
  10. Select the Enforce Survey Deadline check box if you want to restrict your suppliers to submit entries, post the deadline mentioned in the Survey Deadline.
  11. Click Save.
