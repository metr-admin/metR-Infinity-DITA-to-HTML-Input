Adding SKU Item

You can add an SKU Item as a Product. Later on, when you look up for an SKU item in RFQ Item menu, the description is pulled from the product and added to RFQ Catalog.

Parent topic: Appendices

To add an SKU Item

  1. Click the Products tab.orClick
  2. Under the Recent Products section, click New.
  3. Enter the Product Name.
  4. Complete the following fields.

    Field Name

    Description

    Product Name

    Enter name of the product.

    Product Family

    Select the Product Family from the drop-down menu.

    Product Code

    Enter the Product Code.

    Active

    Select the Active check-box to flag that the product is active.

    Product Description

    Enter the description of the product.
  5. Click Save.
