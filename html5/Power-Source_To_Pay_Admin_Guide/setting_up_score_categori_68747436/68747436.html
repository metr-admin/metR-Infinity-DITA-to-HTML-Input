Setting up Score Categories

The Score Categories allow you to organize performance scoring into categories which each contain their own individual metrics.

When you analyze the performance or try to score the performance, you have an option to define the categories and the metrics. It could apply to supplier, company, customer, vision, and even any process that you need to evaluate. For example, you may evaluate the Support System or Cost, provided you know how to arrange it and group the categories.

Suppose for a supplier, you need to arrange the Overll as your category. Under Overall, you can define the subcategory or other metrics that can be used to evaluate a supplier.

Note:

Note

There is no limit as to how many metrics you can add for a particular category.

For example, consider the following metrics that fall under the Overall category:

  • Overall Company score
  • Cost
  • Finance
  • Legal

Metrics provide you the ground on which to evaluate a given category (in this case, Overall category).

Parent topic: Scorecard Setup Console

To set up score category

  1. Click the Scorecard Setup Console tab.
  2. Click Add Score Category.
  3. Enter the Category Name and a short Description.
  4. To define the score structure, enter a Name and Description for each metric. Click to add a row. Click to delete a row. For example, provide Category Name as Overall, and the following metrics to evaluate the Overall category:
    • Company
    • Cost
    • Finance
    • Legal
    • Quality
    • Spend Analysis
  5. Click Save.

To edit Score Category

  1. Click the Scorecard Setup Console.
  2. Click List of Score Category.
  3. Choose a category to edit under the Category Name column.
  4. Click Edit.
  5. Make the appropriate changes.
  6. Click Save to confirm your changes.
