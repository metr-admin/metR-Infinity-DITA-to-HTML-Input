Setting Up DocuSign

Before you start the setup, you must have a DocuSign account. After creating a DocuSign account, perform the following tasks:

  • Create apps and keys in DocuSign account
  • Configure DocuSign parameters in Power Platform
  • Configure webhook in DocuSign account
