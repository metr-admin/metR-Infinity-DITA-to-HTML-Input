Configuring OAuth for an Application in Adobe Sign

After creating an API application in your Adobe Sign account, configure OAuth for that application.

  1. Log in to your Adobe Sign account as an administrator.
  2. Navigate to the Account tab.
  3. ClickAcrobat Sign API > API Applications.
  4. Select the relevant API application from the list.
  5. Click Configure OAuth for Application.
  6. Enter redirect URI: {Power Platform Instance Url}/Poweradmin/integrations/details/AdobeSign
  7. Check the Enabled check-box for all scopes. Keep the account option in the Modifier field.
  8. Click Save.
