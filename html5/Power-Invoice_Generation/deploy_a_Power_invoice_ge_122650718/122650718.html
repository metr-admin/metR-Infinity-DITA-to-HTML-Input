Deploy a Power Invoice Generation for Salesforce Billing Quick Start

How to deploy the Power Invoice Automation for Salesforce Quote to Cash Power Composer Starter Pack.

What Quick Starts are Available?

Quick Starts include all of the components needed for a document generation solution. The purpose of a Quick Start is to help you get up-and-running quickly with a pre-configured template and pre-determined data sources.

The QuickStart solutions for Power Invoice Generation for Salesforce Billing include:

  • Account Statement
  • Create Receipt
  • Create Credit Memo
  • Create Debit Memo
  • Generate Invoice

How Do You Deploy a QuickStart?

To deploy a Quick Start, follow these steps:

  1. When installing Power Invoice Generation, select Install for All Users.
  2. Select Power Invoice Generation from the App dropdown.
  3. Click the Power Quick Start tab.
  4. Select the Quick Start you want to use.
  5. Click Go to Page Layouts.
  6. Add the newly created Power Invoice Generation button to the Page layout and click Save.

You will be prompted to enable Remote Sites when you select the QuickStart. You must do this in order for Power Invoice Generation to be able to create the button. Also, you must configure Configure Composer as a Connected App before using the solution.
