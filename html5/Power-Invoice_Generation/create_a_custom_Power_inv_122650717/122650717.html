Create a Custom Power Invoice Generation for Salesforce Billing Solution

If you want to create a custom Power Invoice Automation for Salesforce Quote to Cash solution (or tweak the Starter Pack solution), you can do so from the Power Solutions tab using the Power Solution Manager.

Parent topic: Using Power Invoice Generation

Create Your Own Document Generation Solution

If you want to create a custom document generation solution (or customize one of the Quick Starts), you can do so from the Power Solutions tab using the Power Solution Manager. For an overview of this process, see Creating and Managing Composer Solutions.

Tip:

Power Invoice Generation for Salesforce Billing includes all of the features and functionality of Power Composer, with the exceptions listed below.

Create Documents in Batches

After deploying or creating your document generation process for Power Invoice Generation, you can run that process in batches using Power Batch. Read more about creating a Batch solution .

Automate with Power Trigger

You can also use Power Trigger to automate your Power Invoice Generation for Salesforce Billing solutions. Read more about automating with Power Trigger .

Limitations

Power Invoice Generation ​​​​​​​for Salesforce Billing has the following limitations:

  • Customers can only generate documents from the objects within the Salesforce Billing installed package (those with the blng prefix).
  • Word and HTML Email are the only supported template types.
  • Email and PDF are the only output file types.
  • Up to 1,000 invoices per month.
