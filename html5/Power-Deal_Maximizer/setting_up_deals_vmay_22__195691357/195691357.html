Setting Up Deals

You must complete the post-installation configuration first and then set up the colors, dimensions, and guidance rules. When initially configuring Deal Maximizer and Deal Manager modules you must complete the tasks this section describes, as there are dependencies between the objects. Once the objects are all in place and you are using Deal Maximizer, the order in which you modify or add new objects is not as important.
