Deal Maximizer Objects

The following table lists the objects inside the Power Deal Maximizer Setup package.

Objects

Purpose of the object

Deal

Represents a deal.

Deal Color Band

Represents the deal rating colors and their percentile ranges.

Deal Guidance Dimension

Represents a single dimension in a dimension set.

Deal Guidance Dimension Set

Represents a set of dimensions shared by rules.

Deal Guidance Dimension Set Member

Represents a single dimension in a dimension set.

Deal Guidance Rule

Represents a single rule in a deal guidance ruleset.

Deal Guidance Rule Entry

Represents a single entry in a deal guidance rule.

Deal Line Item

Represents a product or service line item associated with a deal.

Deal Summary Group

Represents a summary of deal line items.

Deal Term

Represents the term associated with a deal.

Search Filter (Deal)

Represents the Deal Maximizer search filter.
