Deal Manager System Properties

The following table provides a brief description of all the Deal Manager System Properties. Go to Setup > Custom Settings > click Manage next to Deal Manager System Properties.

Setting

Description

Deal Guidance Callback Class

The deal guidance callback class name. The class should implement the CustomClass.IDealGuidanceCallback interface.

Deal Optimizer Callback Class

The deal optimizer callback class name. The class should implement the CustomClass.IDealOptimizerCallback interface.

Display Dimensions In Guidance

Enable this setting to display rule dimensions in the pricing guidance detail.

Display Net Price Per Unit In Guidance

Enable this setting to display the net price per unit in the pricing guidance detail.

Hide Percentile Range In Guidance

Enable this setting to hide the percentile range in the pricing guidance detail. You can hide either percentile range or price range, not both.

Hide Price Range In Guidance

Enable this setting to hide the price range in the pricing guidance detail. You can hide either percentile range or price range, not both.
