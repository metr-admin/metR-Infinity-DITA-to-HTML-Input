Using Custom Deal Guidance Layout

You can use a custom Deal Guidance layout for deals on the Cart page. Custom pages are useful when you want to add custom-built content in layout for deals. To view the custom page, you need to provide the custom page URL in Config System Properties.

Parent topic: Setting Up Deals

To enable custom deal guidance page

  1. Go to Setup > App Setup > Develop > Custom Settings > Config System Properties > Manage.
  2. Click Edit next to System Properties.
  3. Enter the Deal Guidance custom page URL in the Custom Deal Guidance Page field. For example, enter https://na34.visual.force.com/apex/MyCustomDealGuidancePage for a custom page named MyCustomDealGuidancePage.

For a selected Quote/Proposal Line Item, you can 'Preview' the custom Deal guidance interface and activate the custom page if the preview is satisfactory.
