Configuration for Deal Guidance Rules

You must add the following values in the Business Objects picklist in the Deal Guidance Rule object to use them to configure Deal Maximizer rules.

  • Apttus_Config2__LineItem__c
  • Apttus_Config2__ProductConfiguration__c

You must also define field dependencies, if they are not already created, on the Guidance Type field to complete the configuration of the Deal Guidance Rule object.

Parent topic: Configuring Post-Installation Settings

To A the Business Objects picklist

  1. Go to Setup > App Setup > Objects > Deal Guidance Rule > Business Objects.
  2. Click New next to the Values section.
  3. Add the following two object references:
    • Apttus_Config2__LineItem__c
    • Apttus_Config2__ProductConfiguration__c
  4. Click Save.
    You are redirected to the Business Object details page.
  5. In the Values section, rename the Label field for the picklist values as described in the following table. Click the API Name field each time to populate the updated name automatically.

    Picklist Values

    Rename to

    Deal__c

    Apttus_DealMgr__Deal__c

    DealLineItem__c

    Apttus_DealMgr__DealLineItem__c

  6. Click Save.

The values are added to the Picklist Values for the Business Object and are now available when you create Deal Guidance Rules.

If you have upgraded the package, you must add the value Custom to the Measure picklist field. On the first installation, this field is available in the picklist.

To add values to the Measure picklist

  1. Go to Setup > App Setup > Objects > Deal Guidance Rule > Business Objects.
  2. Click New next to the Values section.
  3. Enter Custom.
  4. Click Save.
