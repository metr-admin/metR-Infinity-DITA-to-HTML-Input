Configuring Post-Installation Settings

After you have installed the Deal Maximizer Setup, Deal Maximizer, and CPQ packages, you must configure some package objects before you can begin using the Deal Maximizer solution.

You must configure picklist values for deal objects, set field dependencies, integrate deal guidance with the cart, and ensure that Deal Maximizer is available when you click Configure Products from a Quote/Proposal.

The first two are required to ensure that the drop-down lists are populated with the correct values when you create dimensions, dimension sets, and guidance rules. The last piece of configuration is done to make deal guidance available from the cart, which includes having an action button and deal guidance information available when you hover over a line item.
