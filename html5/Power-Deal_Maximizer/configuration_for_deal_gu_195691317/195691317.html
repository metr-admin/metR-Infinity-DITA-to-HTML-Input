Configuration for Deal Guidance Dimension Sets

You must add the following values in the Business Objects picklist in the Deal Guidance Dimension Sets object, to use them to configure Deal Maximizer rules.

  • Apttus_Config2__LineItem__c
  • Apttus_Config2__ProductConfiguration__c

You must also define field dependencies, if they are not already created, on the Guidance Type field to complete the configuration of the Deal Guidance Dimension Sets object.

Parent topic: Configuring Post-Installation Settings

To add values to the Business Objects picklist

  1. Go to Setup > App Setup > Objects > Deal Guidance Dimension Set > Business Objects.
  2. Click New next to the Values section.
  3. Add the following two object references:
    • Apttus_Config2__LineItem__c
    • Apttus_Config2__ProductConfiguration__c
  4. Click Save.

The values are added to the Picklist Values for the Business Object and are now available when you create Deal Dimension Sets.

To set Field Dependencies

  1. Go to Setup > App Setup > Objects > Deal Guidance Dimension Set > Guidance Type.
  2. In the Field Dependencies section click New.
  3. For Controlling Field select Guidance Type and for Dependent Field select Business Object and click Continue.
  4. For Deal Level and Line Level include the following:

    Deal Level

    • Opportunity
    • Apttus__APTS_Agreement__c
    • Apttus_DealMgr__Deal__c
    • Apttus_Config2__ProductConfiguration__c

    Line Level

    • Apttus__AgreementLineItem__c
    • Apttus_DealMgr__DealLineItem__c
    • Apttus_Config2__LineItem__c
  5. Click Save. When you create a Deal Guidance Dimension Set, the appropriate deal and line level values will be displayed.
