Configuration for Deal Guidance Dimension

You must add the following values in the Business Objects picklist in the Deal Guidance Dimension object to use them to configure Deal Maximizer rules.

  • Apttus_Config2__LineItem__c
  • Apttus_Config2__ProductConfiguration__c
Parent topic: Configuring Post-Installation Settings

To add values to Business Objects picklist

  1. Go to Setup > App Setup > Objects > Deal Guidance Dimension > Business Objects.
  2. Click New next to the Values section.
  3. Add the following two object references:
    Apttus_Config2__LineItem__c Apttus_Config2__ProductConfiguration__c
  4. Click Save.
    The values are added to the Picklist Values for the business object and are now available when you create Deal Dimensions.
