Power Connection Board (A101)

Profiles for UL and IEC are in this topic

All power lines pass through the power connection board. For each power block one power connection board is required.

Main Functions of the Power Connection Board

  • Input, output and DC bus connections to the power modules

  • RFI filters on power module input and output

  • DC bus supply via a passive rectifier circuit for system start-up

  • PSU start control

  • Small resistive battery load for battery midpoint stabilization

  • Power module input voltage sense (always 400 V)

  • Power module input voltage sense (always 208/220 V)

  • Bypass voltage sense (configurable for different bypass voltages)

  • Power block output current sense

  • Controller number selection

  • Connection of input backfeed contactor K101

  • Connection of temperature switches for the power modules

The board ensures:

  • distribution of the control signals and power wiring of the rectifier part of the UPS system

  • conversion of the rectifier input voltage to a lower value for measurements on the rectifier controller

  • connection of the rectifier backfeed contactor

  • start-up of the power supplies (placed in the power module) of the UPS system

  • distribution of the control signals and power wiring of the inverter part of the UPS system

  • conversion of the bypass voltage to a lower value for measurements on the inverter controller

  • conversion of the output current to a lower value for measurements on the inverter controller.

Input Backfeed Contactor (K101)

The input voltage sense must be performed prior to the K101 contactor as the contactor only closes if the voltage is within the tolerances of the PFC rectifier. This input voltage is also used as supply of the internal DC bus during system start-up.

Configuration of the Power Connection Board

Depending on the system, different configurations are necessary:

  • System output voltage adaptation by removing corresponding cut-out bridges

  • Enabling second PSU through jumper settings (two power modules)

  • Jumper on thermoswitch in case of odd number of power modules

  • Controller number selection by jumper settings
