Update the Firmware of the Controller Box (A102)

Reality EMEA

Note: The update of the firmware on the controller box can be done via a RS232 serial connection.
  1. Connect your laptop to the service port (J223) located on the front side of the controller box.
    Note: Use the service port cable 940â€“0024D for the proper connection of all required signals.
  2. Launch the UPS Tuner and select Controller Box. After start-up, your laptop will automatically log in and the main menu appears on the terminal screen.
    Note: Navigate through the menu tree by typing the number of the submenu you wish to enter.
    >> Welcome to Service Port Main Menu <<1: Configure Factory Settings (Locked)2: Configure System Parameters3: System Information Inquiry4: Upgrade Controller Firmware5: System Reboot/Shutdown0: End Session   Choose [1-5, 0]:
  3. Type 4 (Upgrade Controller Firmware).
  4. Select the Send file function for the terminal application to transfer the new firmware hex file to the SRAM.
  5. Type 1 (Erase FLASH & program new code) to program the code into the flash memory. 
    Download complete - HEX data is validSelection:1. Erase FLASH  & program new code9. Abort & Exit   Choose [1, 9]:
  6. Type y to confirm the update and copy the new firmware from the SRAM to the FLASH memory.
    Note: If you wish to cancel the update and keep the old firmware, press the ESC key.
  7. Restart the system.
