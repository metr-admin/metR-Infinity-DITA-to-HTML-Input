Human Machine Interface (A810)

Reality EMEA

The HMI includes:

  • Display unit

  • Main operating status indications

  • Operating buttons

  • Detailed operating status indications

  • Alarm indications

  • Analog outputs for controlling the analog instruments

  • Service port

The HMI controller board is part of the UPS front panel assembly and is responsible for all the interaction with the system. The board provides the following functions:

  • Galvanically isolated ABUS interface (CAN-bus) to the main controller. The supply of the SELV 4 part of the board is also realized via this interface

  • RS232 service port for the connection of a laptop and the UPS Tuner

  • RS232 display port for communication with the LCD module

  • 24 red-green LEDS and 4 yellow LEDs for the mimic diagram

  • 15 red LEDs for alarm indication

  • A microcontroller STR710FZ2 with 512 kB external SRAM is integrated to perform all the functions of the front panel. The microcontroller STR710FZ2 handles the system settings, so all settings are stored in the HMI board and not in any of the other controllers.
