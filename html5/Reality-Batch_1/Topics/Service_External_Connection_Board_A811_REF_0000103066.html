External Connection Board (A811)

Reality EMEA

The external connection board contains terminals and connectors for external connections to and from the UPS. Terminals are mounted for connection to:

  • various signalling switches

  • ON/OFF buttons

  • emergency stop (may be supplied from an external source)

  • a temperature sensor

Notice
  • The input contacts on the external connection board are designed for internal use only. The wires connected to these terminals must not be lead out of the UPS cabinet. No external supply voltage is allowed.
  • Do not interchange wires of different functions (battery temperature sensors, EPO, input contacts and output relays) as this will damage the board.

Input Contacts and Output Relays

Summary of signals:

  • 1 common alarm output relays

  • 2 free programmable output relays

  • 1 EPO input contact

  • 2 free programmable inputs contacts

  • 1 battery temperature input and/or ambient temperature (free programmable)

Assignment

All of the input contacts and output relays can be assigned to functions and alarms by configuration (no fixed assignments). The Emergency Power OFF (EPO) is an exception as it is always assigned as shown in the connection diagram for the external connection board. There are two connection options:

Input Contacts

Name Description Location
EPO Emergency Power Off J1602 on 0P2553A
Option 1 IN Block boost/Initial charge J1600 on 0P2553A
Option 2 IN Force to boost charge J1601 on 0P2553A

Technical Data for Input Contacts

Internal DC supply voltage 24 V limited (10 mA per input)
Threshold resistance 0 → 1 Less than 520 Ohm → 1
Threshold resistance 1 → 0 More than 1050 Ohm → 0

Output Relays

Name Description Location
Common alarm Common alarm J1607 on 0P2553A
Option 1 OUT Battery operation J1608 on 0P2553A
Option 2 OUT Static bypass operation J1609 on 0P2553A

The output relays react to an alarm occurrence with a 5 seconds delay.

Temperature Sensors

  • 1 input

  • Configurable as ambient or battery temperature sensor

  • Temperature sensor name configurable

  • Alarm level configurable

  • Battery temperature compensation function selectable
