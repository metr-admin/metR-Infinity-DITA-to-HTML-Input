View the Event Log for the HMI

Reality EMEA

  1. Connect your laptop to the service port (J405) located on the rear side of the front panel (A810).
    Note: Use the service port cable 940â€“0024D for the proper connection of all required signals.
  2. Launch the UPS Tuner, click Next , select Display Module (HMI) and click Next.
  3. Type 1 followed by the Enter key to open System Information.
  4. Type 3 followed by the Enter key to open the Eventlog. The first 20 events will now be displayed. To view all events, type 2 followed by the Enter key.
    When the logs have finished loading into the terminal window, the event logs can be filtered.
  5. In the bottom of the UPS Tuner window, click Event Log Viewer.
    The event logs can now be filtered by module, date, event type or event description for an easy overview of a single alarm.
