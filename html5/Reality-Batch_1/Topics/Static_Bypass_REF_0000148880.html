Static Bypass

Reality NAM

The static bypass consists of the following main parts:

  • Thyristors (Q401)

  • Static bypass controller (A401)

  • Fan with fan monitoring (E401)

The static bypass provides the following functions:

  • Built-in PSU supplied from bypass mains input

  • Dual speed fan supply for the static bypass fans (the fan speed is controlled from the interface board via an input contact)

  • Galvanically isolated thyristor drivers for the static bypass

  • Snubber RC networks for the static bypass thyristors

  • Output voltage measurement. If the UPS main controller or the communication interface becomes inoperable, the board monitors the system output phase-neutral voltage and independently enables the static bypass if the output is lost

  • Communication interface to the UPS controller

  • Two signal outputs to indicate the state of the controller
