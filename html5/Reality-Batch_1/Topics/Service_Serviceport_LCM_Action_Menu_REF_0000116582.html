Service Port LCM Action Menu

Reality EMEA

The Life Cycle Monitoring LCM – Actions page lists all the possible actions to be performed by the FSR.

Available Actions

Action # Description
001–006 Acknowledge “...” Reminder

Definitively acknowledge a pending alert. This alert may already have been temporarily acknowledged via the display. After performing this action, the pop-up will not be repeated on the display.

Note: This will not automatically restart the counter to show the alert again in the future (see below).
012–016 Restart “...” Reminder

Restart a counter with the actual date as the start date. The alert will be shown again after the defined period. This action must be performed by the FSR after the corresponding preventive maintenance has been carried out.
022–026 Set “...” duration before first reminder

Change the default values set for the individual counters.
032–036 Set “...” Counter Start Date

Set a specific start date for a counter which is different from the date automatically set under Start Counters Now and Acknowledge “...” Reminders.

This action can be used to set the correct start dates for each of the counters after a HMI replacement.
101 Enable / Disable LCM Reminders

Enable or disable the display of LCM pop-up alerts. This setting only disables the pop-up messages from appearing on the display. All LCM functions are still running.
102–103 Set time window start / Set time window end

Configure the time window when the LCM alerts are displayed. In case a counter elapses, the alert will occur on the HMI the next time this window is reached.
104 Set reminder repeat number

Configure the number of repeats of a LCM pop-up after it has been temporarily acknowledged. After the set repeat number, the pop-up will not be displayed again.
105 Set reminder repeat period

Configure the time period between repeats of an LCM pop-up after it has been temporarily acknowledged.
106 Set web address

Configure the web address displayed on the LCM alert pop-up.
