Overview of the UPS System

The Components in the UPS Principle Diagram

Component Explanation
X001 Rectifier mains terminal This terminal provides for the connection of the rectifier mains (L1, L2, L3, PE).
T001 Option: Rectifier mains transfomer Matches the rectifier mains voltage to the correct value for the rectifier, and provides galvanic isolation.
Q001 Rectifier mains input switch This switch ensures isolation of the rectifier mains input (no trip function).
K101 Input backfeed contactor See Power Connection Board (A101)
A101 Power connection board See Power Connection Board (A101)
A102 Controller box See Controller Box (A102)
T101 — T102 Power modules See Power Module (T101/T102)
X201 Battery terminal This terminal ensures the connection of the battery positive pole (B+) and the negative pole (B-).
Q201 Battery switch
Q401 Static bypass switch (Thyristors) See Static Bypass Switch Module
A401 Static bypass switch controller See Static Bypass Switch Controller Board (A401)
T401 Option: Inverter output transformer This transformer galvanically isolates the load from the battery and adapts the output voltage to the required voltage of the inverter power module. No transformer is used, if no galvanic isolation is required, and if the required output voltage is in the range from 380–415 V for 3–phase system and 220–240 V for 1–phase systems.
X501 Bypass mains input terminal This terminal provides for the connection of the bypass mains: 1, 2, or 3 phases, neutral and ground, dependent on the configuration and type. For 3–phase systems one common set of terminals for rectifier and bypass mains is possible.
Q501 Bypass mains input switch This switch isolates the bypass mains input.
T501 Option: Bypass mains input transformer This component provides for the galvanic isolation between the bypass mains input and the load. It can also be installed in the case of a different bypass mains input voltage and output voltage. A stabilizer can be installed instead of a transformer.
Q528 Bypass mains input switch Provides for the isolation of the bypass static switch input.
Q601 Manual bypass switch (MBS) This switch enables service or maintenance work. Two positions are available:

  • Auto: in this position the load is supplied from the system (inverter or static bypass)

  • Bypass: in this position the load is supplied directly from the bypass mains. The static switch is isolated from the bypass mains. For a total isolation of the system part, Q001 and Q201 must also be open.
X601 System output terminal (Load terminal) This terminal provides for the connection of the load.
A801 Interface board See Interface Board (A801)
A802,A803 Current sense boards These boards contain the burden resistor for the appropriate current transformer.
A805,A806 Voltage sense boards These boards convert the acquired voltage via the interface board to a standard value.
A810 Human Machine Interface (HMI) See Human Machine Interface (A810)
A811 External connection board See External Connection Board (A811)
A812 Option: Parallel connection board This board is installed if two units are connected in parallel. The board’s functions are:

  • Communication between the individual units

  • Load sharing between the individual units. Each unit takes over the same proportional part of the load.
E101E102E401E701 Fan system Consists of:

  • fans with internal hall sensor for monitoring of the rotation speed

  • fan monitor for monitoring the rotation speed of each fan. If the rotation speed falls below the threshold value an alarm is generated

See Power Module Ventilation (E101/E701) and Static Bypass Switch Ventilation (E401)
A103A104 Capacitor boards See Capacitor Board (A103 / A104)
