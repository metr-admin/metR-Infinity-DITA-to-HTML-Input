Fan Modules

This topic contains profiles for IEC and UL

The setup of the system cooling system depends on different factors. Generally, one power module (20 kVA 12.5 kVA) is equipped with tree fans (E101).

If the system is configured with input and output transformers, additional cooling fans are mounted for transformer cooling (E701).

Only for 75-100 kVA UPS: If the system is configured with input and bypass transformers, additional cooling fans are mounted for transformer cooling (E701).

The static bypass switch module is also equipped with three fans (E401).

Only for 25-50 kVA UPS: To increase the air circulation in the cabinet, one additional fan is mounted in the static bypass switch module.

All fan modules are equipped with similar speed monitoring boards.
