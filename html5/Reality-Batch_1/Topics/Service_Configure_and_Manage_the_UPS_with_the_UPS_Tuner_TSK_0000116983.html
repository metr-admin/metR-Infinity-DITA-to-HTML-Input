Configure and Manage the UPS via the UPS Tuner

Reality EMEA

  1. Connect your laptop to the service port (J405) located on the rear side of the front panel (A810).
    Note: Use the service port cable 940â€“0024D for the proper connection of all required signals.
  2. Launch the UPS Tuner application and click Next in the window that appears.
    Note: If the connection fails, you can reconnect, by clicking Communication > Connect in the top menu bar.
  3. Select the module you can want to configure: if you want to configure the HMI, select Display Module (HMI), and if you want to configure the controller box, select Controller Box, then click Next.
  4. Follow the instructions on your laptop.
    Note: If the connection fails, check the communication ports via the menu Communications > Set communication parameters and adjust if needed. All other parameters are configured automatically to be compatible with the UPS for both the HMI and the controller box.
