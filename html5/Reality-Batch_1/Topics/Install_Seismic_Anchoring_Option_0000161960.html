Install the Seismic Anchoring (Option)

Reality NAM UPS with transformer 25–50 kVA

Note: Anchoring bolts are not supplied. Use appropriate hardware for the floor type.
  1. Drill anchoring holes in the floor according to your system.
  2. Install the rear seismic anchoring bracket(s) on the floor.
  3. Push the cabinet(s) into position against the rear seismic anchoring brackets.
  4. Save the front seismic anchoring brackets – they will be installed in Final Installation Steps.
