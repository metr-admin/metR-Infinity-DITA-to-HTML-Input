System Calibration

You must check all measurements and calibrate as needed. Take extra care to calibrate the transformer and the output current:

  • Transformer output voltage calibration: The characteristics of the output transformer must be compensated to guarantee the 1% output accuracy independent of the load.

  • Output current calibration: For a correct indication of the load percentage and for the correct detection of an overload situation and the transformer compensation.

Transformer Output Voltage Calibration

The transformer is feedforward controlled. This means that the output on the secondary side of the transformer is calculated based on the actual measured current and compensated based on a stored characteristic of the output transformer. The second regulator is the feedback controlled outer control loop. This control loop is very slow and compensates slow changes (temperature, ageing etc) to guarantee the 1% output voltage tolerance. The transformer output voltage can be configured as follows.

  1. Set the UPS system in ‘standby state’.

  2. Disable the outer control loop via the service port menu System calibration > System test function > Disable output transformer compensation.
    Note: Type two characters in: To activate function two type in “02”.
  3. After disabling the outer control loop, the system can be set on. Configure the settings for the transformer output voltage as per the table. If there is only resistive load available the 140% must interpolated or the battery must be applied to help the input. Be sure that the output current is already calibrated, so that the load percentage is displayed correctly:
    Setting Description
    329 TransformerOut Volt at 20% load10
    330 TransformerOut Volt at 60% load
    331 TransformerOut Volt at 80% load
    332 TransformerOut Volt at 100% load
    333 TransformerOut Volt at 120% load
    334 TransformerOut Volt at 140% load11
    It is independent if the load is resistive or has also an inductive part. The relevant value to compensate the transformer is the current and therefor the displayed load. It is not correct to apply only 16% with resistive load to have the same compensation as with 20% and PF 0.8 load.

  4. After calibration, restart the system to enable the outer control loop and activate the new settings.

Outer Control Loop Calibration

Note: Schneider Electric does not recommend calibrating the outer control loop. Instead It is preferred to do the transformer calibration very precisely.
The following settings are used for the outer control loop. For one phase system the Ph-N voltage x 10 must be entered. For three phase systems the Ph-Ph voltage x 10 must be entered.
Setting Description
398 Calibration OutVoltage L12 / L1-N12
399 Calibration OutVoltage L23 / L1-N
400 Calibration OutVoltage L31 / L1-N
One phase systems have standard values of 2200, 2300 and 2400 with 1:1 transformer. Three phase systems have standard values of 3800, 4000 and 4200 with 1:1 transformer. With all different transformers values from 1000 to 7000 (100V to 700V) are possible. The outer control loop is activated 5 seconds after turning on the system to avoid regulation during startup ramp of output voltage. Too low voltages can be set higher if the value in the outer control loop is set a bit higher.
