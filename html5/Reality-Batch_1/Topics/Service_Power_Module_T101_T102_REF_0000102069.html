Power Module (T101/T102)

Reality EMEA

One power module consists of:

  • 3 rectifiers (PFC and step-up)

  • 3 inverters

  • 1 battery charger (step down)

  • 1 RFI board

  • 1 PSU board

  • 1 current limiter and filter board

Three individual rectifiers and three individual inverters form one power module with a 20 kVA rating. Each PFC rectifier and inverter is connected in one common single phase board. For a 3-phase output configuration each line (phase) has its own board. For a 1-phase output configuration the three boards are in parallel and the 120° phase shift is disabled.

3-Phase Output System

One power module consists of three individual inverters each of which deliver 6.7 kVA with a phase shift of 120°. In a 3-phase configuration with one power module (20 kVA System), the individual output line delivers 6.7 kVA.

1-Phase Output System

In a 1-phase output system, the three individual phases are connected and the 120° phase shift for each phase is disabled. The system rating is 20 kVA.
