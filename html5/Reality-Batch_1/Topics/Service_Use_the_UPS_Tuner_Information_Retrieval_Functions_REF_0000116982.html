Use the UPS Tuner Information Retrieval Functions

Reality EMEA

To access and retrieve information from the UPS, the menu tabs in the bottom of the UPS Tuner window can be used:

Capture

The Capture function generates an extract of the information shown in the terminal window. Click on Start Capture, and all actions performed in the terminal window will be recorded. When all the needed information has been shown in the terminal window, click on Stop Capture to stop recording. A file containing all the information you have had shown in the terminal window during capture will be generated and saved locally on your laptop. If an existing file is selected for saving the capture information, the new information will be added at the end of the file.

Event Log Viewer

The filter function in the Event Log Viewer can be used to filter events in the event log by module, date, event type or event description for an easy overview of a single alarm. In the Event Log Viewer you can choose to either view current logs from the terminal window, from a saved file or from the newest saved file. Logs are gathered from the Service Inquiry menu listed in the main menu. Click on Display to see the filtered logs.

UPS Report

Using the function Create Report under UPS Report generates a report containing all relevant information on the system. The service-relevant information generated in the report is easily understandable, but some information (raw data form) may be suitable and understandable for PE and R&D only as it requires qualified interpretation. The created report is stored locally on your laptop. Clicking on View Report allows you to see saved reports in a dedicated window.
Note: For any break-fix activities the Tuner report must be attached in Intouch.

UPS Raw Data Viewer

In the UPS Raw Data Viewer you choose which module to extract the raw data from, e.g. the intelligence module.

Click on Translate to get an adjusted and analyzed version of the data with translated label for each raw data line. Once translated the information can be saved in a .csv file. It is possible to save several raw data extracts in one file by selecting the same file when saving each raw data extract. In this case information is added at the bottom of the file each time. The raw data extract can be very useful for the technical support.

RAW Data Viewer supports two modes:

  • Offline mode (Tuner connection is not required here): When the user clicks on Translate, Tuner opens the raw data translator window displaying some default values. In this mode you can enter or modify the raw data manually.

  • Online mode (Tuner must be connected to the UPS): Tuner automatically captures the raw data from the controller once you click on Display and translates the raw data when you click on Translate. The raw data cannot be modified in this mode.

Settings

The Settings function can be used to view, configure, export and import the UPS settings. See Configure UPS Settings with Tuner.
