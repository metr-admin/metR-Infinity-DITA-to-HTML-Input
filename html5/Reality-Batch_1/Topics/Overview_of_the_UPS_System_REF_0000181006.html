The Components in the UPS Principle Diagram

Component Explanation
Input Rectifier mains terminal This terminal provides for the connection of the rectifier mains (L1, L2, L3, N, G).
T001 Option: Rectifier mains transfomer Matches the rectifier mains voltage to the correct value for the rectifier, and provides galvanic isolation.
UIB Rectifier mains input breaker This breaker ensures isolation of the rectifier mains input (no trip function).
K101 Input backfeed contactor See Power Connection Board (A101)
A101 Power connection board See Power Connection Board (A101)
A102 Controller box See Controller Box (A102)
T101 -T108 Power modules See Power Block
B+, B- Battery terminal This terminal ensures the connection of the battery positive pole (B+) and the negative pole (B-).
BB Battery switch
Q401 Static bypass switch (Thyristors) See Static Bypass
A401 Static bypass switch controller See Static Bypass Switch Controller Board (A401)
Bypass Bypass mains input terminal This terminal provides for the connection of the bypass mains: L1, L2, L3, N, G.
SSIB Bypass mains input breaker This switch isolates the bypass mains input.
T003 Option: Bypass mains input transformer This component provides for the galvanic isolation between the bypass mains input and the load. It can also be installed in the case of a different bypass mains input voltage and output voltage. A stabilizer can be installed instead of a transformer.
MBB Maintenance bypass breaker (MBS) The load is supplied directly from the bypass terminals.
Output System output terminal (Load terminal) This terminal provides for the connection of the load: L1, L2, L3, N, G.
A801 Interface board See Interface Board (A801)
A802,A803 Current sense boards These boards contain the burden resistor for the appropriate current transformer.
A805,A806 Voltage sense boards These boards convert the acquired voltage via the interface board to a standard value.
A810 Human Machine Interface (HMI) See Human Machine Interface (A810)
A811 External connection board See External Connection Board (A811)
A812 Option: Parallel connection board This board is installed if two units are connected in parallel. The boardâ€™s functions are:

  • Communication between the individual units

  • Load sharing between the individual units. Each unit takes over the same proportional part of the load.
Fan system Consists of:

  • fans with internal hall sensor for monitoring of the rotation speed

  • fan monitor for monitoring the rotation speed of each fan. If the rotation speed falls below the threshold value an alarm is generated
A103 Capacitor board (Used in 37.5 kVA UPS only)See Capacitor Board (A103)
