Shut Down/Reboot the UPS System after Firmware Update

Reality EMEA

Note: If shutdown via the display has been disabled, all commands to shut down or reboot the UPS system have been disabled. The only way to turn off the UPS is by using the system enable switch on the front of the UPS by removing all sources.
  1. Press down the ON and OFF buttons simultaneously to shut down the UPS.
  2. Wait 20 seconds for the display to power down.
  3. Press the ON button to reboot the UPS.
    Note: To perform the UPS shutdown, the load must be disconnected form the UPS or the load must be supplied by the external bypass.
