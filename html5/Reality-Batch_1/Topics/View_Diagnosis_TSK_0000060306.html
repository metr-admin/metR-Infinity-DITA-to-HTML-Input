View Diagnosis

  1. From the Main Menu, select System Status > Diagnosis using the UP/DOWN keys and the ENTER key.
  2. Use the UP/DOWN keys to browse through the diagnosis screens. Here you can see information about the digital inputs, internal measurements, and internal status.
