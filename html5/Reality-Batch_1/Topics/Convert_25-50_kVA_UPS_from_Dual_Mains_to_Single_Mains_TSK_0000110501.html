Convert the 25-50 kVA UPS from Dual Mains to Single Mains (Option)

  1. Remove the two plates in the bottom of the UPS cabinet.
  2. Remove the three input busbars from the unit input breaker UIB. Save the three busbar screws.
  3. Remove the three busbar screws from the static switch input breaker SSIB. Save the three busbar screws.
  4. Install the provided busbar assembly for single mains between the unit input breaker UIB and the static switch input breaker SSIB and fasten with the six busbar screws removed in earlier steps.
