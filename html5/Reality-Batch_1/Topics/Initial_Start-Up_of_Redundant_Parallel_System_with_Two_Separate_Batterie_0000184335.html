Initial Start-Up of Redundant Parallel System with Two Separate Batteries and One Common Bypass Source

Reality EMEA 10–80 kVA

Check the following before connecting mains, battery, and the load to the UPS system:

  • Check that all power sections are deenergized, see Measure for Voltages and Currents.

  • Check that all cabinets are correctly grounded.

  • Check that all connectors are correctly fixed.

  • Check that all printed circuit boards are correctly mounted and the plugs are firmly plugged in.

  • Check all interconnections and junctions according to the connection diagram.

  • Check that the external fuses have the correct size.

  • If bags with moisture absorbing material are in the cubicle, remove these. Check that the interior of the UPS system is dry. If there is any condensed water, let the UPS system stand with the front doors open in a dry and warm area for at least 24 hours.

  • Check that all available switches in the system are in the correct position:
    Switch name Switch location Correct switch position
    Q301 Auxiliary cabinet Open
    Q692 Auxiliary cabinet Closed
    Q694 Auxiliary cabinet Open
    Q001 UPS 1 and UPS 2 Open
    Q211 Battery breaker box Closed
    Q201 UPS 1 and UPS 2 Open
    Q601 UPS 1 and UPS 2 Position BYPASS
    Q501 UPS 1 and UPS 2 Open
    Q528 UPS 1 and UPS 2 Open
  1. Connect the input source to both UPS units and check the phase sequence and the voltage.
  2. Connect the bypass source and check the phase sequence and the voltage.
  3. Close switch Q301 in the auxiliary cabinet.
  4. Close switch Q501 on UPS 1.
    The safe bus is now energized via the closed bypass mains input switch Q501, the manual bypass switch Q601 of UPS1 and the safe bus switch Q692 (if available).
  5. On UPS 1: Close switch Q001. UPS 1 will start up. Check any pending alarms on the display and the system specific alarm indication LEDs below the mimic diagram (see Troubleshooting).
    To confirm one pop-up messages, press the ENTER key. To confirm all pending pop-up messages, press the ESC key. When all alarms have been confirmed, the display shows the following screen:
    Manual Bypass
Standby
    Note: If the system is programmed for AUTOSTART it will start automatically after 60s if the rectifier mains input switch Q001 is closed and the input voltage is within tolerance.
  6. On UPS 1: Press the UP key twice to start the light indicator and display test. All LEDs should turn on.
  7. On UPS 1: Press the ON button to turn on UPS 1 (if not already automatically started with the AUTOSTART function.
    The display will show the following screen:
    Table 1. Overview Screen 
    Manual Bypass Normal OperationLoad 058%  Cap. 093%405Uin  xxxUout  xxHz
  8. On UPS 1: From the Main Menu, select System Status using the UP/DOWN keys and the ENTER key. Check the input and output voltages.
  9. On UPS 1: Close switch Q201 or Q211 in the battery breaker box.
  10. On UPS 1: Use the DOWN key to go to the battery measurements page and check the battery voltage and current. Press the ESC key until you return to the overview screen.
  11. On UPS 1: Open switch Q001. Check that the UPS transfers to battery operation.
    Table 2. Overview Screen 
    Manual Bypass Battery OperationLoad 058%  Cap. 093%405Uin  xxxUout  xxHz
  12. On UPS 1: Close switch Q001. Check that the UPS transfers to normal operation.
    Table 3. Overview Screen 
    Manual Bypass Normal OperationLoad 058%  Cap. 093%405Uin  xxxUout  xxHz
  13. On UPS 1: Close switch Q528. Check that the synchronous LED turns on on the mimic display.
    Note: The bypass source must be available and within tolerances.
  14. On UPS 1: From the Main Menu, select System Control > Bypass Control > YES, UPS into Bypass using the UP/DOWN keys and the ENTER key. After the UPS has transferred to static bypass operation, press the ESC key until you return to the overview screen.
    If the UPS is not able to transfer to static bypass operation, see Troubleshooting.
  15. On UPS 1: From the Main Menu, select System Control > Bypass Control > YES, UPS out of Byp using the UP/DOWN keys and the ENTER key. After the UPS has transferred to normal operation, press the ESC key until you return to the overview screen.
    If the UPS is not able to transfer to normal operation, see Troubleshooting.
  16. All functionality of UPS 1 has been checked now. Press down the ON and OFF button to turn off UPS 1.
    Note: The safe bus is still supplied via the manual bypass switch Q601 at position BYPASS.
  17. On UPS 2: Close switch Q501. Repeat step 5–15 on UPS 2.
  18. Close safe bus switch Q694 (if available).
  19. On UPS 1: Press the ON button.
  20. On UPS 1 and UPS 2: From the Main Menu, select System Control > Bypass Control > YES, UPS into Bypass using the UP/DOWN keys and the ENTER key.
  21. On UPS 1 and UPS 2: Turn switch Q601 to position AUTO.
  22. On UPS 1 and UPS 2: From the Main Menu, select System Control > Bypass Control > YES, UPS out of Byp using the UP/DOWN keys and the ENTER key. After the UPS units have transferred to normal operation, press the ESC key until you return to the overview screen.
  23. Check that UPS 1 and UPS 2 are operating correctly in normal operation and that there are no pending alarms present.
    Note: The safe bus is energized. The load is now supplied from all UPS units in parallel/redundant operation with load sharing.
  24. Reset any pop-up messages.
  25. From the Main Menu, select Event Log > Clear Log > YES, Clear Log using the UP/DOWN keys and the ENTER key.
