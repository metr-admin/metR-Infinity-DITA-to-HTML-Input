Static Bypass Switch Module

Reality EMEA

The static bypass switch module consists of the following main parts:

  • Thyristors (Q401)

  • Static bypass switch controller (A401)

  • Fan with fan monitoring (E401)

The static bypass switch module provides the following functions:

  • Built-in PSU supplied from bypass mains input

  • Dual speed fan supply for the static bypass switch fans (the fan speed is controlled from the interface board via an input contact)

  • Galvanically isolated thyristor drivers for the static bypass switch

  • Snubber RC networks for the static bypass switch thyristors

  • Output voltage measurement. If the UPS main controller or the communication interface becomes inoperable, the board monitors the system output phase-neutral voltage and independently enables the static bypass switch if the output is lost

  • Communication interface to the UPS controller

  • Two signal outputs to indicate the state of the controller

Configuration

Different configurations are possible: 1- or 3-phase output.

The same static bypass switch module is used for 1-phase output and for 3-phase output power. In 1-phase configurations all three thyristors are turned on and the input and output terminals are linked together.

The static bypass switch controller board must be configured for the actual output voltage and output Phase configuration. This is done by cutting resistors. Also some Resistors are marked to be cut for "PXP" have to be removed. The installed board can be used as reference here.
