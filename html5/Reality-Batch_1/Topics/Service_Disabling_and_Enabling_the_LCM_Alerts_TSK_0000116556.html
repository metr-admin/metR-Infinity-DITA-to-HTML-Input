Disable / Enable the LCM Alerts

Reality EMEA

Note: This setting only disables the pop-up messages from appearing on the display. All LCM functions are still running.
  1. From the main menu, select Settings > System.
  2. Press the UP/DOWN key, select LCM Alerts and press the ENTER key.
  3. Set the LCM Alerts setting to either On or Off. Select On to enable the LCM alerts and Off to disable the LCM alerts.
