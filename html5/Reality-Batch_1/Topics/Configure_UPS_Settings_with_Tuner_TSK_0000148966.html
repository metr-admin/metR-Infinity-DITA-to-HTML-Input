Configure UPS Settings with Tuner

Reality EMEA

  1. In Tuner, click on the Settings tab and click on Edit Settings.

    In the column Systems Settings you see the present settings of the UPS.

    In the column User Settings you can configure the UPS settings. Changed values in the User Settings column are red.


    • Configure the setting parameters in the User Settings column as needed and click on the Send button to send them to the UPS.
    • Click on the Export button to export the settings as a binary (BIN) file.
    • Click on the Import button and browse to a stored settings file on your laptop to import these settings to the User Settings column.
    • Click on the Reset button to restore all the settings in the User Settings column to match the settings in the Systems Settings column.
