View the Alarms

  1. From the Main Menu, select Event Logging > View Alarms using the UP/DOWN keys and the ENTER key. Wait a few seconds while the alarm list is being prepared.
  2. Use the UP/DOWN keys to browse through the alarms present on the UPS system.
