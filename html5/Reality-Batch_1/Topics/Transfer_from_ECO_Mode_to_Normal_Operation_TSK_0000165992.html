Transfer from ECO Mode to Normal Operation

Reality EMEA 10–80 kVA

Note: This menu is only available if ECO Mode is enabled.
  1. From the Main Menu, select System Control > Bypass Control using the UP/DOWN keys and the ENTER key.
  2. Select YES, UPS out of ECO Mode to transfer to normal operation. 
    Confirm action:UPS out of ECO Mode NO, Abort►YES, UPS out of ECO Mode
    Note: For parallel UPS units, it is only necessary to perform the transfer into normal operation on one UPS – the other parallel UPSs will automatically transfer to normal operation.
