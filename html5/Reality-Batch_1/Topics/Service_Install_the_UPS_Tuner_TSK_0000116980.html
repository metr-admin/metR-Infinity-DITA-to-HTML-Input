Note: You must have administrator rights to install and use the software.
Note: The date and time on the laptop must be correct. If a former release of the UPS Tuner is already installed, you must remove the older version when asked before installing the new version of the UPS Tuner.
  1. Open the UPS Tuner installation file and follow the instructions on your laptop.
    Note: The default installation directory is: C:\Program Files\Schneider IT Services\UPS Tuner Reality PXC or C:\Program Files (x86)\Schneider IT Services\UPS Tuner Reality PXC (for 64 bit Windows version) on your computer for a first time installation. If a previous installation directory exists on your laptop, this directory will be used.
  2. When the installation is finished, launch the UPS Tuner software.
    Note: You will be asked for a license code on a first start-up. If this is the case, follow steps 3 and 4.
  3. To obtain the license, copy the environment code from the license requested window by clicking Copy and send this code to your licensing server or your local Schneider Electric contact. See Service View for more information.
  4. When you receive your license code, paste it into the Your license field and click Valid. This license is valid for a year.
    The UPS Tuner is now ready for use.
