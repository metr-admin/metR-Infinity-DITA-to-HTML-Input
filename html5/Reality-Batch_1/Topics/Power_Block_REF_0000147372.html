One power block consists of:

  • 25 kVA: Two power modules, or

  • 12.5 kVA: One power module with a capacitor board and an empty power module box1

  • Controller box

  • Power connection board

  • Fan with fan monitor

Power Block Structure

One power block consists of one or two power modules (maximum 25 kVA).

  • Up to four power blocks can be paralleled (maximum 100 kVA)

  • Each power block has its own controller (maximum four controllers)

    • Data collection, communication, load sharing and parallel behavior is controlled by the master controller

  • Inverter self-test is run at every start-up

  • Odd number of power modules is allowed

1 The 12.5 kVA power block can be added to a 25 kVA power block to make a 37.5 kVA UPS)
