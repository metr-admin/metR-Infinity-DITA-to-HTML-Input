Viewing Data Sync Run History

You can view the run history for all of your syncs, including error messages and warnings that were generated during a sync.

The Run History page displays a list of all the recent syncs executed between this Dynamics 365 org and Power. Refer to the following table for a description of the information provided in each Run History column.

Column Name

Description

Name

The system sync job name. Click the link to view data sync run history details.

Sync Status

Status of the sync. A sync can have one of the following statuses:

  • Success
  • InProgress
  • Incomplete
  • PartialSuccess
  • Error
  • Interrupted
  • Unprocessed

Sync Start Time

The date and time that the sync job started

Sync End Time

The date and time that the sync job ended

Real Time Sync

Yes / No. All syncs for Power Quote-to-Cash for Dynamics 365 are in real time ('Yes'). Otherwise, they are scheduled ('No').

Duration

The duration of the sync job. Time is displayed in minutes or seconds.
Note:

Sync start and end times are displayed in the logged-in user's time zone.

To view data sync run history details

  1. Open the connector.
  2. Click the Run History tab. The Run History page is displayed, sorted by the most recent data sync job.
  3. Find a Sync Group in the Run History table and click the Sync Group Name. The Sync Status page is displayed.
  4. Use the search field to look up sync status for one or more sync group objects. Refer to the following table for a description of the Sync Status fields.

    Name

    Description

    Sync Group Id

    A link to the Sync Group associated with the sync job

    Sync Start Time

    The date and time that the sync job started

    Sync End Time

    The date and time that the sync job ended

    Last Synced Time

    The date and time that the sync group was last synced

    Succeeded

    Indicates if the sync succeeded or failed (Yes/No)

    Inserted

    The number of records inserted (new)

    Updated

    The number of records updated (modified)

    Deleted

    The number of records deleted

    Failed

    The number of records that failed to sync

    Unprocessed

    The number of records that were not processed

    Failure Reason

    If the sync failed, the reason for the sync failure

    Sync Object Status (Table)

    A table that displays a list of objects in the sync group

    Unprocessed Details

    A table that displays a list of errors for each sync group object that failed
    Note:

    Sync start and end times are displayed in the logged-in user's time zone.

    Note:

    No information is displayed for records that were successfully synced. Records that failed to sync are displayed in the Sync Error Status list.

  5. If required, use the search field to look up sync error status for a specific sync group object.

    For a table of common errors during the sync process and troubleshooting suggestions, refer to Troubleshooting Error Messages in the Appendices.
