Mapping Objects

The first thing you need to do when setting up data sync is to configure the objects that you want to sync between Dynamics 365 and Power. In this task, you will map Dynamics 365 objects to corresponding objects in Power.

To map objects between Dynamics 365 and Power

  1. From the left panel under Groups, select Map Objects. The Map Objects page is displayed.
  2. From the ribbon at the top of the page, click New. The New Map Object page is displayed.
  3. Enter the Mapping Name. This can be any name that identifies the object being mapped (for example, "clm_agreement mapping").
  4. Enter the Dynamics Object (API Name). This is the API name of the object stored in Dynamics (for example, "apttus_apts_agreement").
  5. Enter the Power Object (API) Name. This is the API name of the object stored in Power (for example, "Apttus__APTS_Agreement_c").
  6. Click Save or Save & Close.
    Note:

    When you map objects for the first time, the Map Objects page is blank.
