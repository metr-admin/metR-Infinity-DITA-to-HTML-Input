Mapping Fields

After you have mapped at least one object, you can map fields between the corresponding mapped Dynamics 365 and Power objects.

Note:
  • A maximum of 150 fields can be synced per object.
  • You cannot use any of the following tasks to sync picklist fields between Dynamics 365 and Power. Refer to the topic Mapping Picklist Field Values for complete steps.
  • Before initiating an object sync, you must map an External Id field and map a ModifiedOn field.
