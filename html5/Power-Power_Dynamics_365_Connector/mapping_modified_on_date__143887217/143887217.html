Mapping Modified On Date

You must create a mapping for the Modified Date/On field prior to initiating an object sync.

Note:

There is no need to create a "ModifiedOn" custom field as it already exists in both Dynamics 365 and Power.

Parent topic: Importing and Exporting Field Mapping

To map the ModifiedOn field for object sync

  1. From the left panel under Groups, select Map Fields.
    The Map Fields page is displayed.
  2. From the ribbon at the top of the page, click New.
    The New Map Field page is displayed.
  3. Enter "ModifiedOn" in the Name field.
  4. Enter "ModifiedOn" in the Display Name field.
  5. Enter "DateTime" in the Type field.
  6. Enter "modifiedon" in the Dynamics Field Name field.
    Note:
    To avoid errors, copy-and-paste the field name from Dynamics 365.
  7. Enter "LastModifiedDate" in the Power Field Name field.
    Note:
    To avoid errors, copy-and-paste the field name from Power.
  8. Search and select the Object Mapping Name. This is the mapping name of the field's object (for example "Opportunity").
  9. Click Save or Save & Close.
